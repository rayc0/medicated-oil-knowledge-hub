# 印尼药油传统完整指南：白树油、Telon 婴儿油与 Jamu 草药体系

> **摘要** — 印尼是全球药油文化最深厚的国家之一，其传统比荷兰殖民档案能记录的更古老。理解印尼药油，必须先分清四个本地类别：**白树油 Minyak Kayu Putih（卡疗油 / 白千层精油）**作为家家户户的万用油；**Minyak Telon** 三油配方专供新生儿与婴幼儿；**Minyak Gosok / Minyak Urut** 是成人按摩驱风油；以及 **Minyak Angin**（"风油"）用于晕车晕船与感冒鼻塞。本文梳理植物学源头、活性成份、Cap Lang 鹰标 / Cap Kapak 斧标 / Konicare 等代表品牌，并解释印尼药油与 Jamu（佳木）草药传统的共生关系，以及华人买家、孕婴家庭与穆斯林消费者最关心的清真认证与安全问题。

## 一、印尼药油的四个本地类别——千万别混为一谈

西方旅行指南往往把印尼药油笼统称为"巴尔姆 / balm"，但印尼本地市场对它们的分类极为严格：

| 印尼名称 | 中文俗称 | 实质 | 典型使用者 |
|---------|---------|------|-----------|
| Minyak Kayu Putih | 白树油 / 卡疗油 / 白千层油 | 接近纯品的白千层精油（含 1,8-桉叶素 50–65%） | 全家通用，从婴儿到老人 |
| Minyak Telon | "Telon 三合一"婴儿油 | 白树油 + 茴香油 + 椰子油的三油配方 | 新生儿至 2 岁婴幼儿 |
| Minyak Gosok / Urut | 按摩油 / 跌打油 | 多成份红肿温热油（樟脑、薄荷、桂皮、丁香） | 成人，运动后或感冒受寒 |
| Minyak Angin | 风油 | 嗅吸 + 涂抹两用的清凉油 | 婴儿以上全年龄 |

"Minyak"在印尼语和马来语中只是"油"的意思，关键在修饰词。一个雅加达祖母给孙子腹绞痛涂的 *minyak kayu putih*，与一位肌肉劳损的祖父睡前涂的 *minyak gosok*，从药理上完全不是同一类东西，尽管它们在便利店货架上瓶身长得差不多。**这是华人买家在印尼旅游或托代购时最常踩的坑**——把婴儿用的白树油当成成人按摩油，或反过来把含高浓度樟脑的成人按摩油给婴儿用。

## 二、白树油（Minyak Kayu Putih）——印尼版的"万金油基底"

### 2.1 植物学与命名

*Kayu putih* 在印尼语 / 马来语中字面意思就是"白色的木"，得名于白千层（*Melaleuca cajuputi*，及商用近缘种 *M. leucadendra*）那层层剥落的纸状白色树皮。英文里"cajuput / cajeput"就是马来原文的直接音译。

这种树原产于印尼群岛，主产区集中在 **马鲁古群岛（Maluku，旧称香料群岛 / Moluccas）、苏拉威西岛（Sulawesi）与巴布亚（Papua）**。其中布鲁岛（Buru Island）上由印尼林业部经营的国营白千层油厂（PMKP）已经持续生产超过一个世纪，今天苏拉威西仍是产量最大的产区。

**与澳洲茶树油的区别**：白千层油（cajuput）、互生叶白千层 / 茶树油（*M. alternifolia*）和绿花白千层（*M. quinquenervia* / niaouli）虽然同属白千层属，但化学组成完全不同。白树油主打 1,8-桉叶素，茶树油主打松油烯-4-醇，**两者不可互换**。

### 2.2 化学成份

药典级印尼白树油的主要成份：

- **1,8-桉叶素（Eucalyptol / Cineole）50–65%** — 主要活性，与桉树油共有的拟刺激、祛痰、温和发红剂作用
- α-蒎烯、β-蒎烯 — 微量驱风作用
- α-松油醇、柠檬烯 — 抗菌与香气调和
- 痕量松油烯-4-醇

可以把白树油理解为"印尼版的桉树油，但气味更甜更柔，对儿童更友好"。详细的桉叶素药理可参考[桉油药理完整指南](/zh-hans/ingredients/eucalyptus-oil-pharmacology)。

### 2.3 印尼人怎么用

- **沐浴后涂抹**：印尼妈妈给婴儿洗澡后例行涂抹背部、胸口与脚底，"防风寒"（*masuk angin*）
- **腹绞痛 / 胀气**：在脐周顺时针打圈按摩
- **感冒鼻塞**：胸口与背部薄涂，或滴几滴在枕头上嗅吸
- **蚊虫叮咬**：直接点涂患处止痒
- **肌肉酸痛**：与按摩同步使用（成人）

**重要差别**：与中国的清凉油 / 风油精不同，印尼白树油常常 **接近 100% 纯白千层精油**（部分产品会用椰子油稀释）。这意味着它的桉叶素浓度比清凉油里的桉叶素含量高出十几倍，使用时**用量要小**。

## 三、Minyak Telon——新生儿出院第一天就开始涂的油

### 3.1 配方

Telon 在爪哇语中意为"三"，指三种油的固定配比：

- **白树油（Minyak Kayu Putih）约 30–62%** — 主要驱风成份
- **茴香油（Minyak Adas / Fennel Oil）约 13–16%** — 助消化、缓胀气
- **椰子油（Minyak Kelapa）约 22–55%** — 基底油，稀释 + 滋润

各品牌微调比例与额外香料（如薰衣草、洋甘菊），但核心三油不变。Konicare、Cap Lang Telon Plus、Caplang GPU、My Baby 是几大主流品牌。

### 3.2 为什么是这三种油

印尼传统育儿观念认为新生儿"体寒易受风"（*masuk angin*）。三油协同作用：

- 白树油的 1,8-桉叶素提供温和的"温感"（实际上是 TRPM8 / TRPV1 受体激动产生的感觉错觉，**不会真的提高体温**）
- 茴香油的反式茴香脑（trans-anethole）是**唯一被多项儿科 RCT 证实可缓解婴儿肠绞痛的精油成份**
- 椰子油作为大分子载油，降低活性物的透皮速度，形成保护性油膜

### 3.3 用法与禁忌

**标准用法**：婴儿沐浴后 5–10 分钟内，将 Telon 油倒在大人手掌中搓热（不可直接倒在婴儿身上），薄涂前胸、背部、肚脐周围、脚底与脚踝。

**安全红线**：

1. **不可涂鼻孔下方与人中**——含桉叶素的油用于 2 岁以下婴幼儿面部，有诱发喉痉挛与呼吸抑制的报告。这条规则在印尼也常被忽略，但儿科文献立场明确。
2. **避开眼睛、生殖器与开放伤口**
3. **G6PD 缺乏症（蚕豆症）患儿慎用**——Telon 中常见的薰衣草版本含芳樟醇，部分配方还会添加微量樟脑。详见 [G6PD 缺乏症与药油安全指南](/zh-hans/safety/g6pd-deficiency)
4. **不要"为了更暖"加大用量**——透皮吸收剂量随面积线性上升

## 四、按摩油 Minyak Gosok / Minyak Urut——成人专用的红肿温热油

### 4.1 与白树油的本质区别

白树油单一成份（白千层精油），按摩油则是**多成份红肿温热复方**，典型成份谱：

- 樟脑（Camphor）3–10%
- 薄荷醇（Menthol）3–8%
- 水杨酸甲酯（Methyl Salicylate）少量至 10%
- 丁香油、桂皮油、姜油、肉豆蔻油 — 提供温感与活血印象
- 椰子油或矿物油基底

代表品牌：**Cap Tawon（蜜蜂牌）、Cap Kapak（斧标）GPU、Tresnojoyo、FX Balsem**。其中 Cap Kapak GPU 在马来西亚与新加坡华人圈知名度最高，常被误认为是"印尼版的虎标"。

### 4.2 使用场景：Kerokan 刮痧

按摩油在印尼最具文化辨识度的用法是 **kerokan**——用一枚硬币（*koin*）蘸油，在背部沿膀胱经反复刮擦，直到皮肤出现暗红色条状瘀痕。这是华人移民带来的刮痧文化在印尼本土化的形态，几乎每个爪哇家庭都会做。

**与中国刮痧的区别**：中国刮痧多用刮痧板与万花油 / 红花油，印尼 kerokan 则使用硬币（旧荷兰盾或现代 Rupiah 硬币）配合按摩油，刮拭力度通常更重，留下的痧痕更深。

### 4.3 成人用法注意

- 单次涂抹面积不超过手掌大小
- 含水杨酸甲酯的产品**禁用于 12 岁以下儿童**与**正在服用华法林等抗凝药的患者**——参考[药油与抗凝药相互作用安全指南](/zh-hans/safety/medicated-oils-and-anticoagulant-medication-safety)
- 涂后避免直接日晒（部分柑橘类精油有光毒性）

## 五、Minyak Angin——印尼版的"嗅吸式风油精"

Minyak Angin 字面意思就是"风油"，专门用于 **晕车、晕船、头晕、感冒鼻塞、突发恶心**。与新加坡的虎标万金油 Aromatherapy 系列、中国的风油精同属一类，但印尼版本通常更稀（樟脑 + 薄荷 + 桉叶素的酒精溶液），强调"开盖嗅吸即起效"，而非长时间涂抹。

代表产品：**Cap Lang Minyak Angin Aromatherapy、FreshCare、Plossa（嗅吸 + 涂抹 + 刮痧三合一棒）**。Plossa 是近年大热的便携产品，瓶身设计成口红状，一端可作为 kerokan 的刮痧头使用。

## 六、Jamu 草药传统——印尼药油背后的文化骨架

要理解为什么印尼药油如此普及，必须知道 **Jamu（佳木）草药体系**。Jamu 是印尼自 8 世纪起延续至今的传统草药医学，2023 年被 UNESCO 列入人类非物质文化遗产名录。

Jamu 与中医、印度阿育吠陀并列为亚洲三大本土医学传统，但 Jamu **更去仪式化、更家庭化**：

- 没有中医的"经络脏腑"理论框架
- 没有阿育吠陀严格的 Vata / Pitta / Kapha 体质分型
- 核心是"对症使用 + 经验配方 + 妇女主导的家庭传承"

历史上 Jamu 几乎完全由女性掌握。荷兰殖民时期（17–20 世纪），荷兰妇女与印尼妇女共同经营 Jamu 业务，留下了大量殖民地药典记录。其中最重要的早期文献是德国-荷兰博物学家 **Rumphius** 在马鲁古群岛安汶（Ambon）写成的 *Herbarium Amboinense*（《安汶植物志》，1741 年），其中详细记载了白千层的提油工艺。

**药油是 Jamu 体系中"外用方"的代表**——内服用煎剂 / 粉剂 / 蜜丸（Jamu Gendong 流动小贩至今仍背着药篮在巴厘岛的市场叫卖），外用则用 minyak（油）+ pilis（额头敷药）+ param（身体涂膏）。

## 八、华人买家与海外代购的实操要点

### 8.1 真伪辨别

印尼药油的假冒主要发生在马来西亚与新加坡的小杂货店。三个核心识别点：

1. **BPOM 注册号**——印尼药监局 BPOM 的注册号格式为 *POM TR / TI XXXXXXXXXX*（草药类）或 *POM HT XXXXXXXXX*。瓶身或包装盒上必须印有
2. **MUI 清真认证（Halal）**——Cap Lang、Konicare、My Baby、Cap Kapak 都有印尼乌理玛理事会 MUI 的清真标识，假货往往遗漏或印刷模糊
3. **批号与生产日期**——压印在瓶底的钢印批号应清晰，喷印模糊或日期与瓶身不符多为翻新假货

### 8.2 华人体质适用性

印尼药油的桉叶素 + 茴香油配方与中国传统的樟脑 + 薄荷醇配方在感官上有明显差异——**印尼油更"甜"、更"暖"，华人初次使用可能觉得"不够冲"**。但实际功效不弱，特别是 Minyak Telon 在婴儿胀气与肠绞痛上的循证基础比中国的小儿驱风油更扎实（fennel anethole 的 RCT 数据）。

### 8.3 与中国 / 香港药油的搭配建议

许多印尼华人家庭会同时备着：

- **印尼白树油**（婴儿与日常）
- **香港黄道益活络油**（成人跌打）
- **新加坡虎标白色万金油**（头痛蚊咬）

三者互补——白树油负责婴儿与全家通用，活络油负责跌打劳损，虎标负责蚊虫叮咬与晕车。详细对比参考[东南亚华人药油传统综述](/zh-hans/regions/southeast-asia-chinese)。

## 九、安全须知（不可忽略）

1. **白千层精油原液不可口服**——印尼民间偶有"喝几滴白树油治胃痛"的偏方，但 1,8-桉叶素口服剂量超过 5 mL 可引起严重中枢神经抑制
2. **婴幼儿用量极少**——Minyak Telon 单次 0.5–1 mL（约 8–15 滴）足够全身使用
3. **G6PD 缺乏症儿童**慎用所有含薰衣草、樟脑、薄荷脑成份的婴儿油——这是华南、东南亚华人家庭高发的代谢病
4. **孕妇前三个月**避免大面积使用所有按摩油（含水杨酸甲酯、丁香油、肉桂油的产品风险更高）——参考[孕妇与药油安全指南](/zh-hans/safety/pregnancy-guide)
5. **过敏体质**——白千层精油在桃金娘科过敏者中可引起接触性皮炎，初次使用建议手腕内侧斑贴测试 24 小时

## 十、结语

印尼药油不是单一商品，而是一整套贯穿婴儿 → 成人 → 老人、覆盖日常保健 → 跌打损伤 → 文化仪式（kerokan）的家庭健康体系。它根植于 Jamu 千年草药传统，在荷兰殖民与华人移民的双重影响下完成了现代化，今天通过 Cap Lang、Konicare、Cap Kapak 等品牌走向全球。

对中国与香港的消费者而言，最值得收藏的入门组合是 **一瓶 60 mL 的 Cap Lang Minyak Kayu Putih + 一瓶 Konicare Minyak Telon Plus**——前者是全家通用的"印尼万金油基底"，后者是世界少见的、循证基础扎实的婴儿专用油。
