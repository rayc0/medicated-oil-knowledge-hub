## 一、品牌历史：从跌打医馆到全球华人药柜

### 1.1 黄道益医师与香港跌打文化

1960 年代初，香港工业起飞，劳动人口庞大，工厂、建筑工地工伤频繁，传统跌打医馆在港九两岸遍布街巷。**黄道益**（Wong To Yick）是当时深受街坊信赖的本地跌打医师，擅以外用草药油处理筋骨劳损、扭挫伤痛。

1962 年，黄道益参考自身临床经验，将多味惯用药材的浸膏与高比例水杨酸甲酯调配成外用药油，以家族名「黄道益」为品牌正式推出市场，注册为**黄道益活络油**（Wood Lock Medicated Oil），「木」字取其疏通经络、舒筋活血之意。

### 1.2 家族传承与规模化

| 年份 | 里程碑 |
|---|---|
| 1962 | 黄道益活络油在香港正式面市 |
| 1970s | 随香港移民社群进入东南亚、北美华人社区 |
| 1980s | 开始在香港正式商业化生产，走出跌打馆 |
| 1990s | 进入中国大陆市场；取得香港卫生署认可注册 |
| 2000s | 获中国 NMPA（原 SFDA）注册批文，正式进入大陆正规渠道 |

| 现至今 | 黄道益家族第二代继续掌舵，未被大型药企收购，保持家族独立经营 |

与虎标万金油（已上市跨国公司）、双飞人（广药集团）不同，黄道益至今仍为**家族式私营企业**，主要生产线维持香港本地及大陆合规工厂。

### 1.3 产品线概览

黄道益旗下主力产品（本文聚焦原版活络油）：

- **黄道益活络油（Wood Lock Medicated Oil）**——经典外用跌打药油，红色/棕色液态瓶装
- **黄道益活络油膏**——半固态膏剂，适合定点敷用
- **黄道益喷雾**——近年推出，适合运动后快速涂敷

> **注**：确切成份比例以各地注册批文为准。香港卫生署、中国 NMPA 注册成份表须印于外包装，建议购买时核对。

### 2.2 水杨酸甲酯（冬青油）——主力止痛成份

**水杨酸甲酯**（Methyl Salicylate，CAS 119-36-8）是黄道益活络油最具特色的成份，也是其浓郁气味的来源。

- **药理机制**：经皮肤吸收后，水解为水杨酸（Salicylic acid），抑制花生四烯酸级联反应中的环氧合酶（COX-1/COX-2），降低局部前列腺素 E₂（PGE₂）浓度，达到**局部抗炎镇痛**效果。[^1]
- **透皮吸收（Transdermal absorption）**：外用时经皮吸收率与皮肤完整度、用量、封包面积密切相关。多项药代动力学研究显示，大面积涂抹或封包使用可导致血液水杨酸浓度升高，出现系统性毒性（见安全警示）。[^2]
- **USP-NF 外用制剂上限**：USP-NF 规定外用水杨酸甲酯制剂浓度一般不超过 30%，部分跌打专用制剂经注册批准可使用更高浓度，但须在处方或注册条件下使用。67% 属高浓度，应严格按用量使用。[^3]
- **中国药典 2020 年版**：收录水杨酸甲酯（冬青油）为外用镇痛辅料，规定外用品须注明浓度及禁忌。

### 2.3 樟脑（Camphor）——双效成份

**樟脑**（Camphor，CAS 76-22-2）在黄道益中约占 11%，兼具**局部镇痛**与**反刺激（Counterirritant）**作用：

- 作用机制：激活皮肤感觉神经末梢的 TRPV1 及 TRPM8 通道，产生先凉后热感觉，干扰痛觉传导（Gate Control Theory）。[^4]
- FDA 规定（21 CFR §310.545）：外用樟脑浓度上限 **11%**，超过此浓度视为非处方外用违规。黄道益约 11% 处于上限边缘，合规但无余量。
- **G6PD 特别警示**：樟脑为已知 G6PD 诱发溶血物质，G6PD 缺乏患者（蚕豆症）严禁接触（详见安全警示章节）。

### 2.4 薄荷醇（Menthol）——感觉调节成份

**薄荷醇**（L-Menthol，CAS 2216-51-5）约占 3.5%，主要作用：

- 激活 TRPM8 冷感受器，产生清凉感，辅助缓解局部不适感。
- 增强皮肤通透性（Penetration Enhancer），协助水杨酸甲酯及樟脑的透皮吸收。[^5]
- 有轻微局部麻醉效果（抑制 Nav1.8 钠离子通道）。

### 2.5 中药复合浸膏

黄道益配方保留传统跌打草药成份，包含：

- **川芎**（Ligusticum chuanxiong）：活血行气
- **当归**（Angelica sinensis）：补血活血
- **木香**（Aucklandia lappa）：行气止痛
- 其他可能成份（以官方注册批文为准）

此类中药浸膏成份在外用时的药理数据相对有限，现有研究多为体外或动物模型，人体临床对照数据仍不充分，但为历代跌打民间验方，有长期使用经验记录。

## 三、跌打作用机制：从涂抹到止痛的完整路径

### 3.1 透皮吸收动力学

外用药油涂抹皮肤后，活性成份须穿越皮肤屏障到达靶组织：

```
皮肤角质层（Stratum Corneum）
    ↓ 水杨酸甲酯 + 薄荷醇（辅助通透）
表皮层（Epidermis）→ 真皮层（Dermis）
    ↓
皮下组织、肌肉、关节囊
    ↓
局部抗炎 + 镇痛
```

透皮速率取决于：分子量、脂溶性（logP）、皮肤温度、皮肤完整性。水杨酸甲酯 logP = 2.55，脂溶性适中，透皮效率较好。

### 3.2 局部抗炎路径

```
水杨酸甲酯 → 皮肤内水解 → 水杨酸
    ↓
抑制 COX-1/COX-2
    ↓
↓ 前列腺素 E₂（PGE₂）、前列腺素 I₂（PGI₂）
    ↓
↓ 局部血管扩张、↓ 痛觉敏感化
```

效果类似口服非甾体抗炎药（NSAIDs）的局部版本，但系统暴露量远低于口服（视用量而定）。[^1]

### 3.3 反刺激镇痛（Counterirritant Analgesia）

樟脑 + 薄荷醇激活皮肤 Aδ 及 C 类感觉纤维，产生人为热/凉刺激，通过脊髓「门控理论」（Gate Control Theory）抑制深层痛觉信号上传。此为中西医均认可的外用止痛机制。[^4]

## 四、正确使用方法：跌打扭伤分阶段方案

### 4.1 急性期（受伤后 0—48 小时）

| 项目 | 建议 |
|---|---|
| 原则 | PRICE 原则：保护（Protect）、休息（Rest）、冰敷（Ice）、加压（Compress）、抬高（Elevate） |
| **活络油使用**| ⚠️ **急性期 48 小时内不建议使用**，尤其避免热敷或大力按摩 |
| 原因 | 急性期局部血管扩张、炎症反应正在进行；水杨酸甲酯的促血流作用可能加剧局部肿胀出血 |

> 部分用户在急性期轻涂使用以缓解疼痛感，但需注意力度极轻，避免加重软组织损伤。如有骨折疑虑，须先影像学排查。

### 4.2 恢复期（48 小时后至愈合）

**使用步骤：**

1. **清洁皮肤**：确认涂抹部位皮肤完整无破损、无伤口、无皮疹。
2. **适量取油**：每次取约 1—2 mL（约硬币大小），无需大量涂抹。
3. **轻柔按摩**：顺肌肉走向轻轻揉擦 2—3 分钟，至局部有温热感。
4. **包覆注意**：涂后**不要立即用保鲜膜、绷带封包**，避免加速经皮吸收导致浓度过高。
5. **使用频率**：每日 2—3 次，单次不超过 5 mL；连续使用不超过 7 天，若症状无改善须就医。
6. **洗手**：使用后彻底洗手，避免触碰眼睛、黏膜。

### 4.3 常见使用场景

| 使用场景 | 适用性 | 备注 |
|---|---|---|
| 踝关节扭伤（恢复期） | ✅ 适用 | 须排除骨折 |
| 肌肉劳损、腰背酸痛 | ✅ 适用 | 常见场景 |
| 运动前热身辅助 | ✅ 部分 | 少量使用，勿封包 |
| 关节炎慢性疼痛 | 🟡 辅助 | 只可镇痛，不可替代治疗 |
| 新鲜伤口、皮肤破损 | ❌ 禁用 | 严重过敏风险 |
| 大面积涂抹 | ❌ 禁用 | 全身水杨酸中毒风险 |

## 七、🚨 强制安全警示

**以下为 P0 强制披露内容，请在使用前仔细阅读。**

### ⚠️ 2岁以下婴幼儿——绝对禁用

- 水杨酸甲酯、樟脑、薄荷醇对 2 岁以下婴幼儿均有严重风险。
- **樟脑**：婴幼儿皮肤薄、经皮吸收率高，小剂量即可导致神经毒性（抽搐、呼吸抑制），严重可致死。美国 AAP（美国儿科学会）明确反对任何形式的婴幼儿外用樟脑制品。[^6]
- **水杨酸甲酯**：误服 5 mL 纯品对幼儿可致命；大面积涂抹可导致水杨酸盐中毒（代谢性酸中毒、耳鸣、意识障碍）。[^2]
- **薄荷醇**：可引发新生儿呼吸抑制，不可用于鼻部或口腔周围。
- **结论**：黄道益活络油 **不得以任何方式用于 2 岁以下儿童**，包括接触性使用。

### ⚠️ 孕妇——慎用，建议避免

- 水杨酸甲酯大量透皮吸收后，水杨酸可进入胎盘循环，理论上有胎儿出血风险及干扰前列腺素介导分娩过程。
- 目前缺乏孕妇外用高浓度水杨酸甲酯制品的充分安全数据。
- 建议孕妇**避免使用**，如必要使用须经妇产科医师评估。

### ⚠️ G6PD 缺乏（蚕豆症）患者——禁用

**这是本文最重要的安全警示之一，华南、东南亚地区发生率较高（约 4—10%），务必重视：**

- **樟脑**为已确认的 G6PD 诱发溶血剂（G6PD deficiency hemolysis trigger），已有大量文献记录。[^7]
- **水杨酸甲酯**（水解后产生水杨酸）亦有诱发 G6PD 溶血的案例报告。
- G6PD 缺乏患者接触樟脑制品后，红细胞可发生氧化性溶血，轻者贫血、黄疸，重者肾衰竭危及生命。
- **结论**：G6PD 确诊患者及家庭成员**严格禁用**黄道益活络油，其他含樟脑药油同样禁用。

### ⚠️ 皮肤破损——禁用

- 皮肤有伤口、溃疡、严重皮疹、湿疹破溃时，禁止涂抹活络油。
- 破损皮肤失去角质层屏障，经皮吸收率大幅升高，可导致全身水杨酸中毒及强烈刺激反应。

### ⚠️ 水杨酸过敏、阿司匹林过敏——禁用

- 对阿司匹林（乙酰水杨酸）或其他水杨酸类药物过敏者，须避免使用含水杨酸甲酯的制品。
- 过敏反应包括：局部荨麻疹、接触性皮炎，严重者可出现全身过敏反应（Anaphylaxis）。

### ⚠️ 其他注意事项

| 情形 | 建议 |
|---|---|
| 大面积封包使用 | 禁止，防止全身水杨酸中毒 |
| 用于眼睛/黏膜 | 严禁，立即以清水冲洗 |
| 与口服水杨酸类药物同用 | 须医师评估，避免叠加毒性 |
| 肾功能不全患者 | 慎用，水杨酸主要经肾排泄 |
| 连续使用超过 7 天 | 如症状无改善，停用并就医 |

## 八、常见问题（FAQ）

**Q1：黄道益活络油和虎标万金油有什么区别？**

A：两者均为跌打/肌肉酸痛类外用药油，但成份侧重不同。黄道益以**水杨酸甲酯（~67%）**为主，油性液态，渗透感更强，主攻跌打扭伤深层止痛；虎标万金油以**樟脑+薄荷脑**为主，呈膏状，止痒、提神功能较突出。跌打扭伤效果方面，黄道益因水杨酸甲酯浓度高，止痛渗透效果普遍评价更强，但二者均非处方药，严重伤患仍须就医。

**Q2：活络油可以天天用吗？**

A：短期（3—7 天内）每日 2—3 次使用是常见用法。但长期连续大量使用须注意水杨酸甲酯经皮累积吸收，建议使用超过 7 天症状无改善即停用并就医，不建议作为长期慢性疼痛的唯一管理手段。

**Q3：活络油可以内服吗？**

A：**绝对禁止内服。** 水杨酸甲酯纯品口服毒性极高，5 mL 可对幼儿致命，成人大量摄入亦可致严重水杨酸中毒。外用药油仅供体外使用。

**Q4：买到的活络油气味比以前淡，是假货吗？**

A：不一定。气味浓淡受储存条件（温度、光照、瓶盖密封）影响。长时间开瓶或高温储存会加速挥发，导致气味减弱但浓度实际可能升高（溶剂挥发）。建议比对上述 5 个防伪信号综合判断，并从正规渠道购买。

**Q5：活络油涂后皮肤发红正常吗？**

A：轻微发红属正常的反刺激（Counterirritant）反应，是水杨酸甲酯/樟脑扩张局部毛细血管的生理反应。但若出现大面积红疹、水疱、严重瘙痒，应立即停用并以清水冲洗，考虑过敏反应，必要时就医。

**Q6：蚕豆症患者家里有孩子，可以让其他家庭成员用吗？**

A：可以，蚕豆症是遗传性酶缺乏疾病，不会通过接触传染。但须注意：
- G6PD 缺乏儿童不得接触药油（避免皮肤接触或吸入挥发气体）
- 使用者应在通风处使用，用后密封瓶盖
- 家中有 G6PD 患者，建议改用不含樟脑/水杨酸甲酯的替代产品

**Q7：活络油可以和其他外用药膏一起用吗？**

A：不建议混用。不同外用制品叠加可能改变透皮吸收率或产生不可预测的化学反应。如同时使用处方类外用药（如激素类、消炎类），须先咨询医师或药师。

**Q8：过期的活络油还能用吗？**

A：不建议使用。过期后活性成份浓度可能因氧化分解而改变（升高或降低），防腐效果下降，亦可能产生刺激性分解产物。请按包装标示保质期使用并妥善丢弃过期产品。
