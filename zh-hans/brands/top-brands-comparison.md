## 为什么华人家庭都备一支药油？

药油（活络油 / 风油 / 万金油 / 跌打油 / 药膏）是华人世界 1000+ 年的家庭常备品。从清末民初开始，数十个经典品牌历经几代人传承，覆盖跌打、感冒、晕车、头痛、关节酸痛、蚊虫叮咬等日常场景。

但全网没有一个地方做到：
- **客观对比**：每个品牌的官方页都吹自己最好
- **成份透明**：实际药用浓度、辅料、安全剂量极少公开
- **跨地区法规**：内地 / 香港 / 台湾 / FDA / EMA 对同一成份的判断可能完全不同
- **安全警示**：婴幼儿 / 孕妇 / 蚕豆症（G6PD 缺乏）患者的禁忌

本文根据《中华人民共和国药典》（2020 年版）、USP-NF、国家药品监督管理局（NMPA）、香港中医药管理委员会、台湾卫福部食药署、新加坡 HSA、马来西亚 NPRA 公开资料，对 8 大经典华人药油做横向对比。

## 成份背后的科学

### 1. 樟脑（Camphor）

- **作用**：激活 TRPV1 / TRPM8 受体，产生 cooling-then-warming 感觉，轻度局部麻醉
- **安全剂量**：成人外用 ≤ 11%（FDA OTC monograph 上限）
- **🚨 婴幼儿警示**：FDA 明令禁止 2 岁以下使用樟脑制品。误食 1 茶匙（约 1g）可致幼儿抽搐
- **孕妇**：避免大面积使用，可经皮吸收
- **参考**：FDA 21 CFR 341.14；American Academy of Pediatrics

### 2. 薄荷脑 / 薄荷醇（Menthol）

- **作用**：激活 TRPM8 冷感受体，分散痛感（gate control theory），轻度局部麻醉
- **安全剂量**：外用 ≤ 16%（FDA OTC）
- **🚨 婴幼儿警示**：< 2 岁禁用于鼻部 / 胸口（曾有 laryngospasm 案例）
- **参考**：Eccles R., J Pharm Pharmacol 1994

### 3. 水杨酸甲酯 / 冬青油（Methyl Salicylate）

- **作用**：经皮吸收后转化为 salicylate，具抗炎止痛效果
- **安全剂量**：外用 ≤ 30%（FDA），但**小面积使用**
- **🚨 高风险**：1 茶匙纯冬青油 ≈ 21 颗 325mg 阿司匹林。曾有运动员因大面积涂抹后热敷致死（2007 NYC 案例）
- **🚨 蚕豆症（G6PD 缺乏）**：避免使用 — 水杨酸类可诱发溶血
- **🚨 儿童**：< 12 岁不建议
- **参考**：Chan TY., Hum Exp Toxicol 1996

### 4. 桉叶油 / 尤加利油（Eucalyptus Oil）

- **作用**：1,8-cineole 主成份，轻度抗炎、祛痰、抗菌
- **安全剂量**：外用稀释 ≤ 10%
- **🚨 婴幼儿警示**：2 岁以下禁用于面部 / 鼻部
- **参考**：Australian TGA monograph

### 5. 松节油（Turpentine Oil）

- **作用**：α-pinene 为主，反刺激（counter-irritant），促进局部血流
- **🚨 纯松节油**口服极毒，外用亦需稀释至 ≤ 5%
- **参考**：USP-NF Turpentine Oil monograph

## 真假辨识：5 个共通信号

1. **批次号 + 条形码**：所有正货都有，可上品牌官网或当地药监部门查询
2. **气味层次**：正货的精油有复杂层次感；假货多为单一刺鼻香精味
3. **油液澄清度**：正货多为澄清淡黄 / 琥珀色；假货可能浑浊或颜色过深
4. **包装印刷**：正货字体锐利，封口完整；假货常见模糊、错字、封口松
5. **价格**：明显低于市场均价 50% 大概率是假货

各品牌的具体真伪辨识细节，将在独立的品牌页陆续上线。

## 安全警示（必读）

⚠️ **以下人群使用任何药油前必须咨询医师**：

- **2 岁以下婴幼儿** — 樟脑 / 薄荷脑神经毒性风险
- **孕妇** — 多种精油可经皮吸收，部分有催产 / 致畸风险
- **G6PD 缺乏（蚕豆症）患者** — 樟脑、薄荷醇、水杨酸类均需避免
- **皮肤破损 / 大面积烧伤** — 经皮吸收量大幅升高
- **与热敷 / 电热毯并用** — 可显著增加水杨酸吸收，曾有致死案例
- **眼周、黏膜、生殖器附近** — 任何含樟脑 / 辣椒素 / 薄荷脑制品都禁用
