# 清凉油（清凉万金油）介绍

## 什么是清凉油

清凉油，学名"清凉万金油"，是中国大陆家庭中最为普及的外用消炎止痛药物之一。其绿色铁皮小圆罐的包装已成为几代中国人共同的童年记忆。不论是夏天被蚊子叮咬，还是出门感觉头晕脑涨，很多家庭的第一反应就是拿出清凉油来抹一抹。

清凉油的历史可以追溯到20世纪初。最初是在东南亚华人万金油配方基础上，结合中国本土生产条件进行改良，逐步成为中国大陆本地化的药油产品。1949年后，清凉油由国营药厂批量生产，以极低的成本广泛进入城乡家庭，成为计划经济年代最具代表性的"国民药品"之一。

## 核心成分

清凉油的配方在不同品牌及生产厂商之间存在轻微差异，但核心成分基本一致：

- **樟脑（Camphor）**：通常占比8%–15%，提供温热与穿透感，是清凉油发挥镇痛效果的主要成分之一。
- **薄荷脑（Menthol）**：通常占比10%–15%，产生显著的清凉感，并有轻度局部止痛及止痒效果。
- **桉叶油（Eucalyptus Oil）**：含有桉叶素（1,8-cineole），具有一定抗菌与消炎作用，同时为清凉油带来清新的草药气味，有助于缓解鼻塞。

辅料通常包括凡士林（白凡士林）作为基质，使产品保持软膏状态并易于涂抹。

部分配方还添加有微量松节油（turpentine oil）或薄荷油，进一步增强挥发性气味的强度。

## 主要用途

清凉油在中国家庭中的使用范围极广：

**蚊虫叮咬与皮肤瘙痒**
这是清凉油最普遍的用途。薄荷脑的清凉刺激感可迅速压制瘙痒感，涂抹后数秒内即有效果。适用于蚊子、蚂蚁、蜂类等轻微叮咬，不适用于严重过敏反应。

**头痛与头晕**
将少量清凉油涂于太阳穴（两侧）、印堂（眉间），轻揉按摩，挥发的薄荷脑与桉叶油气味有助于缓解紧张性头痛及轻微偏头痛。

**鼻塞与伤风感冒**
涂于人中穴或鼻翼两侧，深呼吸吸入挥发气体，有助通鼻、提神，是感冒初期常见的辅助缓解手段。

**晕车晕船**
乘车前将清凉油涂于肚脐或太阳穴，有一定预防晕动症的效果，民间使用历史悠久。

**轻微肌肉疲劳**
涂抹于颈部、肩背等疲劳部位，短暂缓解肌肉紧张感，但对深层肌肉酸痛的效果较有限。

## 清凉油 vs 虎标万金油：有什么不同？

两者同源于华人传统药油文化，但在以下几个维度存在明显区别：

| 对比维度 | 清凉油 | 虎标万金油（白色） |
|---|---|---|
| 产地 | 中国大陆（多个国营/民营药厂） | 新加坡（恒生制药/虎标集团） |
| 配方重点 | 清凉为主（薄荷脑、桉叶油） | 清凉+轻微温热（薄荷脑+樟脑） |
| 气味风格 | 清新、偏桉叶草药味 | 较辛辣、复合草药香 |
| 包装 | 绿色/白色铁皮圆罐，极简设计 | 白色/红色圆罐，老虎商标，设计较精致 |
| 价格 | 极低（人民币几元至十余元） | 较高（HKD/SGD十余至数十元） |
| 国际知名度 | 以中国大陆为主，海外华人社区有流通 | 全球100+国家正式销售 |
| 药用浓度 | 通常标注8–15%樟脑、10–15%薄荷脑 | 标准白色配方含25%樟脑、10%薄荷脑（红色版） |

整体而言，清凉油的清凉感较突出，偏向于日常止痒与提神；虎标万金油的配方更为复杂，兼顾清凉与温热双重效果，对肌肉疼痛的针对性更强。

## 购买与价格

清凉油是中国大陆药店与超市的常备品，几乎所有连锁药房（如大参林、老百姓大药房、益丰药业）及各大商超均有售。常见品牌包括"风油精"（虽配方略有不同，但功效相近）、"白凤凰清凉油"、"顺气清凉油"等。

价格极为亲民，一罐标准规格（约3–10克）售价通常在人民币2至15元之间，是国内最具性价比的家用急救药品之一。

## 注意事项

- **2岁以下婴儿禁用**：樟脑对婴幼儿有神经毒性，不可使用于婴儿皮肤或衣物。
- **避开眼睛、口腔及黏膜**：清凉油刺激性较强，误入眼睛需立即用清水冲洗。
- **孕妇慎用**：部分成分可能对孕妇产生影响，建议咨询医生后使用。
- **皮肤破损处禁用**：有伤口、湿疹、皮炎等皮肤问题时不应涂抹。
- **不可内服**：误食须立即就医。
