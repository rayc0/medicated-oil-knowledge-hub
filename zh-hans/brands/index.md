# 药油品牌大全

药油品牌繁多，产地横跨香港、新加坡、泰国、中国内地与日本，各有历史传承与配方特色。本页汇整主要品牌，按产地地区分类，帮助你快速了解各品牌定位，找到最适合自己需求的选择。

---

## 香港传统品牌

香港药油历史悠久，以出口闻名海外，配方融合中西成分，注重多功能性。

- **[白花油完整品牌指南](white-flower-embrocation-complete-guide.md)** — 香港最具代表性的药油，创立逾百年，以薄荷脑、水杨酸甲酯与桉油为核心，主打提神止痛通鼻三合一。
- **[正红花油完整指南](red-flower-oil-complete-guide.md)** — 专为跌打瘀伤设计，以红花提取成分为主要活性成分，活血散瘀效果突出。
- **[保心安油完整指南](po-sum-on-complete-guide.md)** — 香港老铺出品，以丁香与薄荷为主，提神通气效果突出，广受长者信赖。
- **[黄道益活络油完整指南](wong-to-yick-complete-guide.md)** — 香港本地著名跌打油，由黄道益创办，中医活络配方，专注筋骨关节。
- **[万华油完整指南](wan-hua-oil-complete-guide.md)** — 香港本地生产，广泛用于肌肉酸痛与跌打损伤的多用途药油。
- **[万应止痛油](wan-ying-zhi-tong-gao/)** — 多功能外用止痛膏油，香港家庭常备。
- **[双狮行军散完整指南](double-prince-complete-guide.md)** — 香港老字号，主打提神醒脑与舒缓头痛的经典配方。
- **[鹰标驱风油完整指南](eagle-brand-medicated-oil-complete-guide.md)** — 历史逾百年，与虎标齐名的传统驱风油，成分以薄荷脑与桉油为主。
- **[余仁生Cap Limau完整指南](yu-yee-oil-cap-limau-complete-guide.md)** — 主打婴幼儿及儿童适用的温和驱风油，以柑橘为基底。

## 中国大陆品牌

内地药油以中成药标准生产，配方多含传统中草药提取物，价格亲民，供应量大。

- **[风油精完整指南](fengyoujing-complete-guide.md)** — 内地最普及的日用药油，以薄荷脑与水杨酸甲酯为主，清凉止痒。
- **[云南白药气雾剂完整指南](yunnan-baiyao-aerosol-spray-complete-guide.md)** — 云南白药旗下外用喷剂，适用于急性扭伤与跌打止血。
- **[玉林正骨水完整指南](yulin-zheng-gu-shui-complete-guide.md)** — 广西玉林出品的跌打正骨水，以其独特草本配方闻名，运动员常用。
- **[三九系列](sanjiu/)** — 现代中成药企业旗下外用药油产品线。
- **[广东百花油](guangdong-baihua/)** — 源自广东传统配方，以多种花卉与草本精华调配。
- **[如意珍宝](ruyi-zhenbao/)** — 中药复方外用油，专注于活血化瘀与消肿止痛。
- **[白花油（大陆版）](baihuayou/)** — 大陆市场流通版本，配方与港版略有差异。
- **[正骨贴](zheng-gu-tie/)** — 以正骨活络为主的外贴剂型，常见于骨科辅助用途。
- **[玉树](yushu/)** — 专注于皮肤护理的中草药外用油。
- **[虎力油](hubiaowan-jin-you/)** — 大陆版配方驱风药油。
- **[复方樟脑](compound-camphor/)** — 以樟脑为主成分的复方外用制剂。
- **[祛风油](qufengyou/)** — 传统中药祛风止痛外用油。
- **[清凉油](qingliangyu/)** — 中国大陆家喻户晓的薄荷基清凉止痛膏。
- **[活络油](huo-luo-you/)** — 以活络通经为主治的复方外用油。
- **[麝香追风](shexiang-zhuifeng/)** — 含麝香成分的高端追风止痛外用制剂。

## 东南亚品牌

新加坡与泰国的药油多由南洋华人移民发展，配方简洁，在东南亚及全球华人社区广泛使用。

- **[虎标万金油完整指南](tiger-balm-complete-guide.md)** — 源自缅甸、发扬于新加坡的国际经典，红白两款各有功效定位，全球最知名的药油品牌。
- **[斧标驱风油完整指南](axe-brand-complete-guide.md)** — 新加坡创立、香港普及的经典绿色驱风油，以薄荷脑与桉油为核心，尤其适合消化不适与头痛。
- **[均隆驱风油完整指南](kwan-loong-oil-complete-guide.md)** — 东南亚华人家庭常备，驱风提神效果持久。
- **[暹罗纯油完整指南](siang-pure-oil-complete-guide.md)** — 泰国出产的经典驱风油，主打头痛与肌肉不适，在东南亚及香港广受欢迎。
- **[亚美津洋洋完整指南](ammeltz-yoko-yoko-complete-guide.md)** — 日本安摩力品牌，凝胶剂型便于局部涂抹，在东南亚市场广受欢迎，颈肩局部应用设计。

## 如何在品牌之间选择

不同品牌设计用途有别，选择时可参考以下维度：

**按主要需求：**
- 头痛提神 → 白花油、保心安油、清凉油
- 跌打扭伤 → 正红花油、黄道益活络油、万华油
- 肌肉关节痛 → 撒隆巴斯、虎标万金油（白色）、玉林正骨水
- 婴幼儿 → 余仁生Cap Limau（需确认年龄适用）
- 驱风肚痛 → 斧标、均隆、保心安油
- 晕车晕船 → 白花油、暹罗纯油、风油精

**按成分敏感度：**
- G6PD患者 → 避免含樟脑品牌，详见[安全使用指南](/zh-hans/safety/)
- 孕妇 → 大多数药油不建议使用，请咨询医生
- 婴幼儿（2岁以下）→ 几乎所有成人药油均禁用
