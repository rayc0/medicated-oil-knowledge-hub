# 斑蝥（Ban Mao / Mylabris phalerata / 南方大斑蝥 / 斑蝥素、羟基斑蝥素）药理全解

在《神农本草经》收录的所有药物里，能同时被『可证伪的药理活性』与『可证实的致死性』牢牢挂钩的并不多，**斑蝥**（*Mylabris phalerata* Pallas 或 *Mylabris cichorii* Linnaeus 的干燥全虫）正是其中最具代表性的一味。它是字面意义上的『发泡药』：仅一毫克主成分**斑蝥素**接触完整皮肤，数小时内即可隆起水疱；口服 6 mg 即可致成人死亡。然而正是这一味毒虫，被中医外用方持续运用了两千余年——拔疣去胼、贴敷瘰疬、天灸发泡治哮喘、近三十年又化身为治疗肝癌、食管癌的医院注射剂的母核。

本文要做的，是把这些看似离散的用途串成一根线：**一切都源于斑蝥素对蛋白磷酸酶 PP1 与 PP2A 极其精准的抑制**。理解这一机制，传统主治、现代化制剂、毒理警讯就会从一团散沙凝成一个连贯的故事。

## 一、基原、采收与炮制

斑蝥为芫青科（Meloidae）下两种发泡甲虫的干燥全体：

- ***Mylabris phalerata*** Pallas —— 即『南方大斑蝥』，体长 1.5–3 cm，鞘翅有黄色横纹三条
- ***Mylabris cichorii*** Linnaeus —— 即『黄黑小斑蝥』，体型较小，生态习性相近

两者均在体液中蓄积斑蝥素，被惊扰时从腿关节『反射性出血』释放该化合物；其分子与地中海产『西班牙苍蝇』（*Lytta vesicatoria*）所含完全一致——这也是为什么西方药典在历史上把两者统称为 *Cantharides*。

采集多在 7–8 月清晨低温、虫体尚未活跃时进行，蒸杀或以米酒蒸汽闷死。**关键炮制点：生斑蝥仅限外用**。少数仍保留的内服制剂均经『米炒法』处理——以糯米同炒至米黄，随后弃米。需要明白：斑蝥素热稳定至约 200 °C，米炒**并不分解主成分**，其作用是借米粒表面吸附部分斑蝥素以降低实际给药量，并向调剂者明确传递『此药已按内服规范处理』的信号。

## 二、主要活性成分：斑蝥素

斑蝥的全部药理几乎可以归约为一个分子：**斑蝥素**（cantharidin），一种小分子双环萜类酸酐（C₁₀H₁₂O₄，MW 196.2）。干燥虫体含量高达 1–1.2 %（按质量计），在动物源次生代谢物中属于异常高位。

次要同系物与炮制衍生物包括：

- **羟基斑蝥素**（hydroxycantharidin）—— 酸酐开环水解产物，发泡作用显著降低，但**仍保留磷酸酶抑制活性**，是国产半合成衍生物**去甲斑蝥素**（norcantharidin）的母体；
- **斑蝥酸**（cantharidic acid）—— 双羧酸形式，在溶液中可与斑蝥素互变；
- **棕榈酸、硬脂酸、脂肪醇** —— 甲虫角质层脂质，在临用浓度下药理惰性。

水疱、疣体松解、肿瘤细胞凋亡、肾损伤——所有重要的临床现象都指向同一个分子：斑蝥素。

## 三、作用机制：一个分子为何能做这么多事

斑蝥素是**丝氨酸/苏氨酸蛋白磷酸酶 PP1 与 PP2A 的高效、高选择性抑制剂**，IC₅₀ 在 0.1–1 µM 之间。PP4 与 PP5 同样被抑制；PP2B（钙调神经磷酸酶）、PP2C 与全部酪氨酸磷酸酶在治疗浓度下基本不受影响。

这一**单一生化作用**足以解释斑蝥的所有用途：

### 3.1 发泡级联（外用）

PP2A 是表皮角质形成细胞内 MAPK 与 NF-κB 信号的关键去磷酸化负调控者。当斑蝥素穿透角质层、抑制基底层与基底上层细胞内 PP2A 后，会迅速发生三步级联：

1. 中性丝氨酸蛋白酶的酶原激活脱离磷酸酶依赖性约束，**胞内大量释放活性蛋白酶**；
2. 这些蛋白酶**切割桥粒芯糖蛋白（desmoglein）与桥粒斑蛋白（desmoplakin）**——即把角质形成细胞之间『铆钉』固定住的关键蛋白；
3. 基底层以上的表皮**整片脱开**，浆液渗出填充裂隙，**2–10 小时内形成角质层下到表皮内水疱**，24 小时达峰，并自动『封顶』。

关键之处在于：水疱位于**表皮内**而非真皮内，基底膜完整保留。这意味着**愈合不留瘢痕**——这正是斑蝥素在面部等美容敏感区治疗病毒性皮损的最大优势所在。

### 3.2 拔疣与治软疣机制

发泡级联本身就是疗法。斑蝥素把感染 HPV 的角质形成细胞（寻常疣）或含 MCV 软疣小体的细胞与下方健康真皮**整层撬开**，疣体随疱顶脱落，真皮血管与附属器毫发无损。

这正是『冷冻不愿做、刮匙不敢做』的儿科与面部皮损病例的理想方案，也是美国 FDA 于 2023 年批准 Verrica 制药公司**YCANTH（0.7 % 斑蝥素膜剂）**用于 2 岁及以上传染性软疣患者的核心药理依据。

### 3.3 凋亡与抗增殖作用

PP1/PP2A 处于细胞周期『磷酸化—去磷酸化』平衡的核心。被斑蝥素抑制后：

- **p53 过度磷酸化、稳定**（逃离 MDM2 介导的降解）；
- **JNK 与 p38 MAPK 持续磷酸化**——促凋亡；
- 线粒体释放细胞色素 c，激活 caspase-9 与 caspase-3；
- Cdk1 无法去磷酸化 → **G2/M 期阻滞**。

这一机制为国内三级医院使用**去甲斑蝥素**与**羟基斑蝥素钠**作为肝细胞癌、食管癌、原发性肺癌辅助注射制剂提供了理性基础。去甲斑蝥素保留约 60 % 的 PP2A 抑制活性，发泡作用却减弱 50–100 倍——这是医药化学层面的一次有意义的『取舍设计』：保留抗肿瘤价值，剔除皮肤毒性。

## 四、传统中医理论与主治

斑蝥气味**辛热，大毒**，归肝、胃、肾经。功效条目：

- **攻毒蚀疮**（*gōng dú shí chuāng*）
- **破血散结**（*pò xiě sàn jié*）
- **引赤发泡**（*yǐn chì fā pào*）—— 经典的发泡药指征

传统主治与现代机制可一一对应：

| 传统主治 | 现代解读 |
|---|---|
| 寻常疣、扁平疣、胼胝、鸡眼 | 发泡将 HPV 感染表皮整层撬离 |
| 顽癣、白癜风外搽 | 发泡 + 细胞毒作用于浅层感染表皮 |
| 瘰疬（颈淋巴结结核）外贴 | 反向刺激 + 对增生组织的促凋亡 |
| 痈疽久溃不敛——『拔毒』之用 | 局部血管扩张性炎症 + 借水疱清创 |
| 牙痛（粉撒齿龈） | 神经性局部麻醉样作用——**风险极高，现代弃用** |

## 五、外用剂型——斑蝥素如何抵达皮肤

斑蝥**绝不**作为『药油』日常使用，其治疗窗远不允许。临床实际只见以下三类受限剂型：

### 5.1 点涂酊剂（拔疣去胼）

经典处方为 1–5 % 斑蝥素的火棉胶或乙醇—丙酮溶液，用细木签**只点疣体本身**，绝不能蔓延至周围正常皮肤。覆盖胶布 4–24 小时后揭除。Ycanth 把这一剂型现代化为单次使用的预填充涂药器，递送 0.7 % 斑蝥素于成膜溶媒中。

### 5.2 膏药（瘰疬、慢性溃疡）

将斑蝥粉以 0.5–2 % w/w 与木鳖子、槟榔、大风子等同入热摊膏药中，贴敷于淋巴结或窦道之上 6–12 小时。**有意诱发可控水疱**即是治疗终点。

### 5.3 天灸（穴位发泡）

源自《灵枢》理论的非燃烧『天然灸法』。在『三伏』最热的几天，将斑蝥、白芥子、姜汁调成糊剂，贴敷于大椎、肺俞、膏肓等穴 2–4 小时，故意发泡——用于慢性哮喘、过敏性鼻炎、反复发作的支气管炎。**水疱本身即为疗效载体**，传统理论以经络刺激与免疫调节并称，现代解释指向局部反向刺激诱发的全身性 Th1/Th2 平衡再校准。

## 六、安全性——这一节是本文存在的主要理由

斑蝥的治疗指数（TI）是本知识库收录所有药物中**最窄**的一味。下列要点必须被任何调配或使用者熟记：

### 6.1 急性外用毒性

- **正常皮肤被波及**是最常见不良事件——哪怕极小范围的横向渗漏都会在健康皮肤上形成痛性水疱；
- **大面积水疱的系统性吸收**可致血尿、排尿困难乃至**急性肾小管坏死**；斑蝥素以原形随尿排泄，肾小管上皮是其首要外肾靶器官；
- 儿科病例报告显示，**患儿触碰治疗部位后揉眼**可造成严重角膜损伤，是 Ycanth 类制剂的明确警讯。

### 6.2 系统性中毒（误食/大面积吸收）

**口服致死量约为 10–60 mg 斑蝥素**（≈ 1–5 g 干虫）。中毒进程：

1. 口咽与食管灼烧痛；
2. 呕血、腹绞痛、剧烈水泻；
3. 血尿、急性肾衰竭；
4. 出血性休克——可在 24–48 小时内死亡。

另一更隐匿的毒性是**骨髓抑制**：斑蝥素抑制造血前体细胞 DNA 合成，慢性或反复系统暴露可致全血细胞减少与出血倾向。临床早已观察到，**即使是大面积外用斑蝥膏药**也曾导致可逆——偶有不可逆——的骨髓抑制。

### 6.3 绝对禁忌证

- 妊娠（堕胎药，动物实验显示致畸性）；
- 哺乳期；
- 儿童（除皮肤科医师直接监护下的拔软疣外）；
- 肝、肾功能不全；
- 出血性疾病或抗凝治疗中；
- 面部、生殖器、黏膜（除受控临床场景外）；
- **破损皮肤超过 5 cm²** 范围的应用。

### 6.4『春药』民间用法——必须单独警告

斑蝥与西班牙苍蝇都背着一个共同的民俗包袱：被当作春药。其所谓『机制』不过是泌尿生殖道与盆腔黏膜被刺激损伤后引发的反射性勃起——本质上是中毒。**斑蝥不存在合法的春药用途**。任何以此为卖点的含斑蝥素产品都是危险品，且在多数司法辖区均属违法。

## 七、监管现状

- **中国大陆**：《中华人民共和国药典》2020 年版收载；内服日剂量 0.03–0.06 g（1–2 只炮制后虫体），几乎仅以胶囊形式用于肿瘤辅助治疗。外用无明文剂量上限，但仅限执业医师调配；
- **香港 / 台湾**：列为一级毒性中药，仅持牌中医师可调配；
- **美国**：斑蝥素长期以来仅作院内合成调配药使用；**YCANTH（0.7 % 斑蝥素外用液）**于 2023 年 7 月获 FDA 正式批准用于 2 岁以上传染性软疣患者；整虫斑蝥不属于 OTC 商品；
- **欧盟**：斑蝥素属处方医药品；全虫制剂禁止以食品/补充剂形式销售；
- **澳大利亚**：列入 Schedule 7（剧毒物）。

## 八、配方实务（仅供专业教育参考）

提供给在合法执业范围内运作的医师与配方师：

- **按斑蝥素含量而非按虫体重量标准化**——种间差异与季节波动可使斑蝥素含量在 0.6–1.4 % 之间浮动；
- **载体比浓度更重要**——火棉胶与乙醇—丙酮体系会随溶剂蒸发将斑蝥素**浓缩**进病变组织内，局部暴露量远高于软膏体系；
- **绝不与 DMSO 等强渗透促进剂同用**——这会把一个受控的表皮发泡药变成系统性毒物；
- **永远先小面积试用**，并告知患者保持治疗部位**干燥、覆盖、避免摩擦**；
- **预先规划『揭疱』步骤**——斑蝥素水疱张力高，护理不当极易继发感染；规范做法是 24–48 小时无菌剪除疱顶，敷以无粘连敷料。

## 九、斑蝥在现代外用药谱中的定位

斑蝥**不是**药油原料，不属于万花油、保心安油、虎标万金油、跌打酒或任何留置型涂搽剂的配伍。它是一种**定点、定病灶、单次给药**的精准发泡药，用以诱发可控、可愈合的水疱。把它与本知识库中其他温通走窜类外用主药混为一谈，是会带来**灾难性后果**的范畴错误。

斑蝥真正『锚定』的，是一个相当窄但极有特色的临床用途集合：

- 斑蝥素点涂酊剂治疗寻常疣与传染性软疣（已被 Ycanth 标准化）；
- 经典中医外用方中以瘰疬与慢性溃疡为目标的定点膏药；
- 三伏天灸的穴位发泡疗法用于慢性呼吸道疾病；
- 国内三级医院以**去甲斑蝥素、羟基斑蝥素钠**为代表的肝癌、食管癌注射辅助用药。

斑蝥素以一个清晰的 PP2A 靶点连接起两千年传统外用与当代肿瘤药理，在中药材里属于**机制最清楚**的少数几种之一；同时它又因极窄的治疗指数而成为**最值得敬畏**的少数几种。两个事实不可分割——任何对斑蝥的现代化使用，若忽视其中任何一面，都注定是不完整的。
