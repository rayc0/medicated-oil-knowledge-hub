# 当归（Dang Gui / 当归）药理全解 — 那一缕藏在跌打酒、八宝活络油与产后温经油底下的"血家圣药"

如果你曾在香港旧式药行的后铺打开一瓶**跌打酒（跌打药酒）**，或者在台北的中药铺里看长辈把一瓶**温经油**摁在小腹上慢慢推开，那股近乎芹菜根、又带着一丝甘甜与暖香的气味，几乎都来自同一味药——**当归（*Angelica sinensis* (Oliv.) Diels）**。它在汉语世界被称为"血家圣药"，是东亚地区使用量最大、临床覆盖面最广的根类药材之一。和大多数被归类为"补益药"的中药不同，当归在外用配方里同样地位稳固：翻开几乎任何一张**跌打酒**、**八宝活络油**、**妇科调经油**、**产后温经搽剂**的处方，当归基本都坐在前几味，常与川芎、红花、乳香、没药相邻。

本文是当归"汤药故事"之外的另一半——**外用药理学**的那一半。我们要看清 Z-藁本内酯、阿魏酸、正丁烯基苯酞这几个名字到底在炎症组织和微血管层面做了什么；为什么中医所说的"活血化瘀"在现代血管内皮和镇痛通路上能找到对应；以及当你在一瓶药油的成分表上看到"当归"二字时，应该怎么合理地估算它能做到什么、做不到什么。

## 当归到底是什么

*Angelica sinensis* 是伞形科（Apiaceae，旧称繖形科）的多年生草本，原生于中国西部高海拔冷凉山谷——**甘肃岷县**至今仍是公认的最佳产区，云南、四川、陕西次之。"岷归"在药材市场被认为油性最足、香气最浓，价格亦最高。药用部位是干燥根，传统饮片把它分为三段，各有所主：

- **归头（Gui Tou）**——根头部，传统认为"补血"力最厚，亦有"止血"之说。
- **归身（Gui Shen）**——主根中段，介于补血与行血之间，是大多数复方里"全当归"的主体。
- **归尾（Gui Wei）**——侧根、根梢与须根，传统视为**活血、行血**最强的部分，**跌打、外伤、外用药酒优先取归尾**。

这种"一根三性"的传统区分，并非完全玄学：现代分析显示根尾的挥发油含量与阿魏酸分布与根头确有差异，归尾的活血药理活性确实更突出。优质当归的判断标准——切面油润、断面起油线、香气浓郁——在化学层面对应的就是**挥发油（特别是 Z-藁本内酯）**含量。欧洲药品管理局（EMA）在《当归专论》中将**阿魏酸**与 **Z-藁本内酯**列为药材的两个核心定量指标。

## 三类化合物决定了外用药理的全部故事

现代植物化学已从当归中鉴定出 70 种以上的化合物，但对外用配方来说，真正承担主要药理作用的，是以下三个家族。

### 1. 苯酞类（Z-藁本内酯、正丁烯基苯酞、洋川芎内酯）

当归挥发油的主体是**Z-藁本内酯（Z-ligustilide）**，常占挥发油总量的 50% 以上，配以**正丁烯基苯酞（n-butylidenephthalide）**与**洋川芎内酯（senkyunolides）**。这一类是分子量小、亲脂性强、芳香气浓烈的苯酞化合物，与川芎共享同一骨架——这也是为什么"当归—川芎"在跌打与活络处方里几乎是固定搭档。

Z-藁本内酯的现代药理证据可以归纳为四条：

- **扩张血管、复原微循环**。Z-藁本内酯能松弛血管平滑肌，在动物微循环淤滞模型中可以恢复毛细血管直径、重新打开关闭的毛细血管床。这一画面在中医语言里就是"瘀血开"。
- **平滑肌解痉**。它对血管平滑肌与子宫平滑肌都有抑制收缩作用，与当归用于痛经、产后腹痛时把药油揉在小腹的传统用法一致。
- **抗炎与镇痛**。在啮齿类炎症痛和神经病理性痛模型中，藁本内酯均能减轻疼痛行为，并抑制 IL-6、TNF-α 等促炎细胞因子释放。机制是多靶点的——部分通过抑制 NF-κB 信号、部分通过下调 COX-2 与 iNOS 表达——而非作用于单一受体。
- **可穿过血脑屏障**。Z-藁本内酯分子小、脂溶性高，因此当归在脑缺血与神经保护研究中亦被广泛使用。对外用而言，正是这种脂溶性让它能较好地透过角质层渗入浅表筋膜与肌肉。

苯酞类的一个工艺难点是**不稳定**——遇光、遇热、遇空气会氧化、二聚乃至聚合。新鲜蒸馏的当归油是清亮的、芹菜香与暖香兼具的；放置一年的同一瓶药油，气味会逐渐被樟脑、薄荷与其他配伍盖过。外用配方靠**酒精萃取（跌打酒）**、**稳定的脂质基质**、**棕色玻璃避光**这三条工艺约束来延长它的有效期。

### 2. 酚酸类（阿魏酸为主）

**阿魏酸（ferulic acid）**是 EMA 收录的第二个定量指标，也是当归"水溶性—脂溶性"两个相之间的桥梁分子，并且是整个药材里被研究最多的小分子。

外用与系统性药理的证据包括：

- **强自由基清除剂、抗氧化剂**。阿魏酸在皮肤外用研究里早已是经典——欧美护肤品里它常被用作维生素 C 与维生素 E 的稳定剂。
- **抗血小板聚集、抗血栓**。在动物模型里，阿魏酸通过促进内皮一氧化氮（NO）释放、抑制血栓素 A2，降低血小板聚集——这是现代意义上的"活血"。
- **抗炎**。下调 NF-κB 与 MAPK 通路；这是它在跌打瘀肿外用配方里和挥发油协同的另一条路径。
- **促进创伤修复**。在皮肤伤口愈合模型里，阿魏酸能加速角质细胞迁移与上皮化。

阿魏酸的水溶性中等，因此**酒精浸渍（药酒）**几乎是把它从干燥饮片里溶出的最有效手段——这从化学层面解释了为什么"跌打酒"必须用酒、而不是单纯的水煎或植物油浸。

### 3. 当归多糖（APS）

**当归多糖（Angelica polysaccharides，APS）**是当归"补血"传统功效的现代主力——动物研究中能促进造血、提高红细胞与血红蛋白水平。但 APS 是大分子水溶性多糖，**几乎不能透过完整的角质层**，因此在外用药理学的语境里它的作用极其有限。当你看到产后温经油、调经油等内服外用并行的配方，多糖的"补血"作用主要由内服承担，而外用做的更多是局部活血、解痉、镇痛。

这一点对消费者很重要：**不要因为"当归补血"而高估一瓶药油的全身性补益作用**。外用当归提供的，是局部循环改善与镇痛，而不是"补血"。

## 中医把这味药放到哪里：四个外用经典场景

### 跌打酒（跌打药酒）

岭南、香港、广府跌打门派的传统跌打酒里，当归与川芎、红花、桃仁、乳香、没药、苏木、骨碎补等同用。当归在这个配伍里的角色是**底层活血与镇痛**——它的挥发油提供穿透与解痉，阿魏酸提供抗血小板与抗炎，多种酚酸贡献抗氧化。配方上常用归尾或全当归，酒精浓度多在 50% 以上，浸渍数周至数月。

### 八宝活络油 / 跌打风湿油

八宝、十三味这类老牌活络油里，当归常与肉桂油、樟脑、薄荷脑、丁香油、桂枝油等"温—凉"双相成分同处一瓶。这里当归承担的是**温经活血**这条线——挥发油提供血管舒张与组织灌注，与丁香、桂枝等温性挥发油协同；同时阿魏酸的抗炎作用在反复劳损的颈肩、腰腿处提供慢性消炎支持。

### 妇科调经油 / 产后温经油

闽粤与南洋华人圈仍在使用的"调经油"、"产后温经油"，当归几乎是必入之药。Z-藁本内酯对子宫平滑肌的解痉作用，加上整体的微循环改善，构成了这类药油在小腹与腰骶外揉时的核心机制。这是**外用而非内服**：经皮吸收的剂量远低于内服汤药，安全窗口更宽，但**孕妇仍须避免**（见下文安全章节）。

### 现代复方搽剂与"循环油"

近十年东亚（尤其是台湾、新加坡）出现的"循环油"、"末梢循环搽剂"等新派外用产品，常以当归、川芎、姜黄精油、辣椒油为基底，用于久坐人群的下肢循环与"冰冷手脚"。从药理上说，这条产品线的主要支撑就是当归—川芎挥发油这一对苯酞类，加上辣椒素诱导的局部 TRPV1 激活与血流增加。

## 把"活血化瘀"翻译成现代语言

中医说的"活血"在分子层面其实是几条平行机制叠加：

1. **微循环开放**——藁本内酯舒张小动脉与毛细血管前括约肌；
2. **抗血小板聚集**——阿魏酸抑制 TXA2 与血小板黏附；
3. **抗炎降低血管渗出**——下调 NF-κB、COX-2 与 iNOS；
4. **平滑肌解痉**——降低伴随疼痛的痉挛性收缩；
5. **局部镇痛**——多靶点抑制疼痛信号传导。

这五条在跌打瘀肿的临床表现里同时发生：肿胀消退、瘀青退色加快、压痛减轻、活动度回复——也就是中医所说"瘀去肿消、气血周流"。

## 外用安全要点

当归外用的安全窗口**比红花宽**，但仍有几条不能忽略的红线。

### 1. 孕妇

虽然当归外用的子宫平滑肌作用远弱于内服汤药，但因为整体配方常含红花、桃仁、乳香、没药等强行血药，**孕妇应避免在腹部、腰骶部使用任何含当归的跌打外用产品**。这是与内服一致的安全态度，与"宁可不用、不可冒险"原则。

### 2. 抗凝/抗血小板药物使用者

阿魏酸有抗血小板作用，与华法林、阿司匹林、氯吡格雷等同用时，**理论上**可能增加出血风险。外用经皮吸收量虽远低于内服，但在大面积长期外用（例如全身按摩级用量）时，仍建议告知医生。

### 3. 皮肤敏感与光毒性

伞形科植物（如白芷、欧前胡）的呋喃香豆素（furanocoumarins）有光毒性。当归的呋喃香豆素含量较低，但**敏感肤质或外用后即将日晒的部位，仍建议避免**，或选择室内、夜间使用。

### 4. 破损皮肤与黏膜

含酒精的跌打酒不应施用于开放伤口、新鲜创面或眼周、口腔黏膜。

### 5. 婴幼儿

跌打类外用药酒含挥发油浓度较高，**两岁以下不建议使用**；学龄前儿童如必须使用，应稀释、小面积测试、避免胸腹大面积涂抹。

## 怎么看一瓶药油成分表里的"当归"

读到"当归"两个字时，可以按这几条做"质量判断"：

- **位置**——成分表是按重量降序排列时，当归在前五位，说明用量足；排在十几位之后，多半是"标记性"添加。
- **形式**——是"当归油（Angelica sinensis essential oil）"还是"当归提取物（extract）"还是"当归"？前两者代表挥发油与/或阿魏酸已经被提纯进入成品；后者只能假设它来自整体浸渍。
- **配伍逻辑**——和川芎、红花、乳香、没药等同列，说明走的是"跌打活血"路线；和黄芪、党参等同列，说明走的是"温补行气"路线。
- **气味**——一瓶有效的当归外用品在开盖瞬间应能闻到芹菜—暖香—甘甜的复合气味；如果只闻到樟脑与薄荷而毫无其他底香，挥发油可能已经氧化失效。

## 一句话总结

**当归不是"补血"那么简单。**它的外用价值由 Z-藁本内酯（解痉、扩张微循环、镇痛）与阿魏酸（抗血小板、抗氧化、抗炎）共同支撑，在跌打酒、活络油、调经油、温经油里承担"活血化瘀"这条主线的分子级实现。把它正确读懂——既不神化它的"血家圣药"标签，也不忽视它在外用药理学层面的真实作用——是理解中式医药油的一把钥匙。
