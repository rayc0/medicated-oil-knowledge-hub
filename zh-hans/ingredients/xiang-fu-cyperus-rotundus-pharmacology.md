# 香附（Xiang Fu / Cyperus rotundus / 莎草根）药理全解 —— 撑起『痛经热敷膏』『四物汤外用油』『妇科药酒』和现代 NF-κB / 软骨细胞研究的『气病之总司』

如果你打开一罐用于经期腹痛、小腹冷痛、产后瘀阻的中式妇科药油或膏方，那种悬浮在红花、益母草、当归之上、带着甜润木质感、微微樟脑气的底香，几乎肯定来自**香附**——莎草科植物**香附 *Cyperus rotundus* L.** 的干燥根茎，又称莎草根、雷公头、香头草。这种全球最难根除的农业杂草之一，却在中医方剂学中占据极其核心的位置。明代李时珍在《本草纲目》中称香附为**「气病之总司，女科之主帅」**——这八个字至今没有被任何一味理气药真正取代。

对于药油、膏药、外用酒剂这一类外治剂型而言，香附正是那一味让四物汤系列、痛经热敷膏、产后腹部药油、乳房胀痛敷剂之所以能够**先通后补、先散后养**的关键。本文从倍半萜化学型出发，梳理现代药理学对香附挥发油的研究成果，并将这些成果与香附在外治方剂中的传统功效一一对照，最后讨论 NF-κB 抑制、软骨保护、神经病理性疼痛、子宫平滑肌调节这几个正在把香附推向新临床方向的研究前沿。

## 植物学与药材识别

*Cyperus rotundus* 为多年生莎草科草本，原产非洲、欧洲南部及亚洲南部，现已遍布全球温带与亚热带地区。药用部位为其细小、暗紫褐色的纺锤形或卵形根茎，长约 1.5–3.5 cm，表面有残存的叶基纤毛和明显的环节，断面可见同心环纹。中国药典收载的香附为秋季采挖、燎去须毛后，分为**生香附**（直接晒干）、**醋香附**（米醋拌制）、**酒香附**（黄酒拌制）以及**四制香附**（醋、酒、姜汁、盐四种辅料同制）四种规格。这套炮制系统不是装饰——四制香附在外用膏方中比生香附更易于皮肤吸收，醋香附则把药力导向肝经与下焦。

香附那种沉稳、木质、微甜的特殊香气，并非来自其多糖或皂苷主体，而几乎完全来自一组复杂的挥发油，约占干药重量的 0.5–1.5%。正是这部分挥发油，决定了一切含香附的外用制剂究竟在皮肤上释放了什么活性成分。

## 香附挥发油的四大化学型

与大多数主要成分相对单一的中药不同，香附的药理活性高度依赖**化学型（chemotype）**。全球多项植物化学普查反复确认了至少四种主要化学型：

- **M 型**（中国大陆、香港、日本、台湾、越南）：以 **α-香附酮（约 30%）**、**香附烯酮（约 19%）**、**β-芹子烯（约 18%）**、**香附烯（约 7%）**、**香附醇（约 6%）** 为主——这是东亚中医临床实际使用的化学型。
- **O 型**（日本、台湾、泰国、夏威夷、菲律宾）：**香附烯（约 31%）** 占主导，香附烯酮约 13%、β-榄香烯约 5%。
- **H 型**（印度、部分非洲产区）：**广藿香酮（patchoulenone）** 与轮烯酮类含氧倍半萜含量较高。
- **K 型**（部分地中海、伊朗产区）：**香附烯（约 38%）** 富集，香附烯酮约 11%。

对于中式药油、膏药与药酒，**M 型**才是真正具备「香附味道」的那种。临床意义在于：两批香附即使都符合药典标准，其**α-香附酮含量可能因产地与采收季节相差 3–5 倍**。正因如此，高端「四制香附丸」与妇科外用油的制造商通常会指定使用广东或山东产 M 型饮片。

除倍半萜外，香附中还含有：
- **黄酮类**（木犀草素、槲皮素糖苷），贡献抗氧化能力
- **酚酸类**（迷迭香酸、阿魏酸、对香豆酸）
- **微量生物碱**（治疗意义有限）
- **三萜与甾醇**（β-谷甾醇、豆甾醇）
- **微量环烯醚萜与单萜**，构成挥发油顶香

## 把传统功效用倍半萜重新读一遍

传统中医文献为香附归纳出四大功效：**疏肝理气、调经止痛、温中止痛、散结消滞**。现代药理学发现，这四类功效大致可以对应到香附挥发油的不同活性轴：

**理气作用 ↔ 平滑肌调节。** α-香附酮与香附烯在离体组织实验中可剂量依赖性地舒张被催产素、PGF2α 和乙酰胆碱诱发收缩的子宫平滑肌。其机制涉及钙通道阻滞与前列腺素通路干预。这正是香附经典适应症——**痛经、乳房胀痛、腹部痉挛**——的现代药理学注脚。

**止痛作用 ↔ NF-κB 与 MAPK 通路抑制。** 这是现代证据最扎实的环节。α-香附酮已被证实通过负向调控 NF-κB 信号、抑制 LPS 诱导的巨噬细胞 COX-2 表达与 PGE2 释放，并下调 p38、ERK、JNK 的磷酸化。2021 年发表于 *PMC8312409* 的一项研究显示，α-香附酮可显著减轻 IL-1β 诱导的软骨细胞 COX-2、TNF-α、IL-6 与 iNOS 上调，并改善小鼠手术造模骨关节炎进展。落到外治剂型上，这意味着：当香附加在痛经热敷膏或膝关节药油中时，它的倍半萜其实在做与系统性 NSAIDs 同一类的上游细胞因子与介质抑制工作，只是发生在皮肤与浅筋膜层。

**温中作用 ↔ 挥发油的产热扩散。** β-芹子烯与香附烯组分挥发性强、亲脂性高，在加温的膏药基质中可较顺畅地穿透腹直肌鞘，是患者主观「温通感」的物质基础之一。

**散结作用 ↔ 抗增殖与抗纤维化活性。** β-芹子烯与 α-香附酮在多种肿瘤细胞系中显示抗增殖活性，并能调节纤维化模型中的 TGF-β/Smad 信号。这些目前主要停留在体外层面，但为香附在乳癖、癥瘕外敷方中的传统角色提供了生物学上的合理框架。

## α-香附酮专档——为什么这一倍半萜值得特别关注

在香附挥发油的数十种成分中，α-香附酮是迄今为止药理学评价最高的一种，原因有以下几条：

1. **在中国临床常用的 M 型中含量最高**，约占挥发油总量的 25–35%。
2. **直接抗炎机制清晰**——分子对接研究（PubMed 38747858）显示，α-香附酮与 NF-κB p65、p38、ERK、JNK 均能稳定结合，为多个实验室在体外与动物模型中观察到的活性提供了结构依据。
3. **神经病理性疼痛模型中的镇痛活性**——2024 年发表于 *PMC11679560* 的研究显示，腹腔注射 α-香附酮或口服香附提取物，均可阻止紫杉醇所致的冷觉与机械痛觉过敏发展，且机制涉及去甲肾上腺素通路。这一点对外治剂型有重要提示意义：α-香附酮富集的制剂，其镇痛作用可能超越单纯抗炎，触及中枢痛觉调节，这是薄荷脑、水杨酸甲酯等常规外用成分所不具备的维度。
4. **抗疟原虫红内期活性**——与外治无关，但说明 α-香附酮生物活性谱之广。
5. **抗菌活性**——对金黄色葡萄球菌、大肠杆菌及若干食源性致病菌有中等强度抑制，这是香附在药酒、酊剂中具备一定防腐力的化学基础。

## 香附在外治剂型中的实际应用

### 妇科外用膏药与药油

香附四物汤（香附、当归、川芎、白芍、熟地）的外用化是中式妇科药油的核心配方之一。在传统配方中，香附通常以醋制饮片浸入麻油，与四物汤的当归、川芎共同先煎滤渣，再加入红花、益母草、艾叶等动血温经药材。临床上这一配方用于：

- **原发性痛经**——下腹部、腰骶部加热敷帖，经前 3–5 日开始预防性应用
- **产后腹痛**与**恶露不尽**——配合益母草，外敷小腹
- **乳房胀痛**与**乳癖**——制成软膏外敷于乳房（避开乳头乳晕）

### 腹部热敷膏与温阳贴

针对寒凝气滞型腹痛，香附常与小茴香、肉桂、丁香、艾叶共同打粉，以蜂蜡和麻油调成软膏，或铺于布袋制成「腹部温敷包」。这类产品在台湾、香港的中药房和近年的电商平台上非常常见。从化学层面看，香附的 α-香附酮与肉桂的桂皮醛、小茴香的茴香脑形成「平滑肌调节 + 局部热感 + 抗炎」三联组合。

### 跌打风湿药酒中的辅助配伍

虽然香附并非常规的跌打主药，但在不少行气化瘀型药酒中（尤其针对中老年妇女腰膝寒痛、月经期间加重的关节痛），香附常以 5–10% 比例加入，与红花、川芎、当归同泡。乙醇本身有助于将其倍半萜成分萃取得更彻底。

### 现代外用复方与新研究方向

近五年来，针对香附倍半萜的微乳剂（microemulsion）、纳米脂质体（NLC）、透皮贴剂等新剂型研究开始增多，目标是解决 α-香附酮亲脂性极强、皮肤渗透不均的工业化难题。已有研究证实，将香附提取物负载于 NLC 后，其透皮通量可提升 2–4 倍，并显著改善对炎症模型的疗效。这类研究为下一代妇科外用产品提供了药剂学路径。

## 配伍逻辑与传统组合

香附在外治方中很少单用，其经典配伍包括：

- **香附 + 当归 + 川芎**——「妇科气血双调」轴心，几乎所有妇科药油的标配
- **香附 + 艾叶 + 小茴香**——温中散寒理气，针对宫寒、腹冷
- **香附 + 乳香 + 没药**——理气兼活血止痛，跌打型外用酒中常见
- **香附 + 柴胡 + 青皮**——疏肝解郁，治疗因情志因素加重的乳房胀痛与胁肋痛
- **香附 + 益母草 + 红花**——产后通用方向

值得注意的是，**醋香附**与**生香附**的功效侧重不同：醋制后的香附挥发油总量略降，但对肝经与下焦的导向更明确；外用妇科产品中，醋香附是首选。

## 安全性、禁忌与注意事项

香附整体安全等级较高，毒理学研究显示其急性毒性极低，LD50 高，长期使用未见明显器官毒性。但外用时仍需注意：

1. **孕期慎用**——香附具理气活血特性，孕期内服或大面积外敷小腹均不建议。少数妇科药油说明书将「孕妇」列为禁忌。
2. **过敏体质**——倍半萜类成分偶有接触性皮炎报告，首次使用建议小范围测试。
3. **化学型差异**——前述 M、O、H、K 四型药理活性存在差异；正规厂商应以挥发油气相色谱指纹图谱控制质量，消费者购买时应优先选择有产地标识与质量批次信息的产品。
4. **与抗凝药物的相互作用**——香附本身无显著抗凝活性，但当与红花、当归、桃仁等活血药联合大剂量使用时，应注意与华法林等口服抗凝药物的潜在累积作用，外用面积过大时尤为重要。
5. **儿童与婴幼儿**——不建议用于 2 岁以下婴幼儿小腹外敷。

## 总结：香附之所以仍然是「气病之总司」

从挥发油化学型差异、α-香附酮的 NF-κB / MAPK 抑制、子宫平滑肌调节、软骨细胞保护到神经病理性疼痛模型中的镇痛活性，现代药理学几乎在每一个分支上都给李时珍那句「气病之总司，女科之主帅」找到了分子层面的注脚。但更重要的是，它揭示了一个外治制剂工程师应该记住的事实：香附并不是单一活性分子，而是一组**化学型决定活性**的倍半萜混合物。两批名称相同、来源不同的香附饮片，可以在同一张配方中产出活性差异显著的成品。

这正是为什么在妇科药油、腹部热敷膏、四制香附丸为代表的中式外用产品体系中，原料溯源、产地认证、挥发油指纹图谱质量控制并非过度工程化——它们是这味「气病之总司」真正发挥临床价值的化学前提。
