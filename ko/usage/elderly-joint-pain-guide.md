# 노인 관절통 약용 오일 가이드: 무릎·어깨·손가락 안전 사용법

노인 인구에서 관절통은 가장 흔한 만성 통증 중 하나입니다. 약용 오일과 파스(외용 진통제)는 내복 진통제의 전신 부작용을 줄이면서 국소 통증 완화를 제공할 수 있어, 다약제 복용 노인에게 특히 가치 있는 선택입니다. 그러나 노인의 피부와 약물 대사 특성을 이해하고 올바르게 사용하는 것이 중요합니다.

---

## 1. 노인 관절통의 종류와 외용 진통제의 역할

| 관절통 유형 | 특성 | 외용 진통제 역할 |
|---|---|---|
| 퇴행성 관절염(골관절염) | 연골 마모로 인한 통증, 무릎·엉덩이·손 흔함 | 국소 통증 완화의 주요 선택지 |
| 류마티스 관절염 | 자가면역 염증, 여러 관절 동시, 아침 강직 | 보조적 통증 완화 (전신 치료 주도) |
| 통풍 | 요산 결정에 의한 급성 염증 | 급성기에는 외용 NSAIDs 활용 가능 |
| 어깨 건염·점액낭염 | 어깨 주변 힘줄·낭 염증 | NSAIDs 외용제 효과적 |

---

## 2. 노인의 피부 특성과 외용제 안전성

노인의 피부는 다음과 같은 특성으로 인해 외용 진통제 사용 시 특별한 주의가 필요합니다:

| 피부 특성 | 외용 진통제에 미치는 영향 |
|---|---|
| 피부 두께 감소 | 경피 흡수율 증가 — 동일 용량에서 더 많은 전신 흡수 |
| 피지 분비 감소 | 피부 보호 장벽 약화 — 자극에 더 민감 |
| 혈관 분포 감소 | 국소 혈류 감소 — 흡수 속도가 느릴 수 있음 |
| 탄력 감소·상처 취약성 | 파스 제거 시 피부 손상 위험 증가 |

**실전 팁**: 노인에게 파스를 제거할 때는 피부를 손으로 지지하면서 천천히 180도 방향으로 떼어냅니다.

---

## 3. 관절 부위별 제품 선택 가이드

| 관절 부위 | 권장 제품 | 사용법 |
|---|---|---|
| 무릎 (퇴행성) | NSAIDs 외용 겔(케토프로펜·볼타렌), 온감 파스 | 무릎 주변에 가볍게 도포 또는 패치 부착 |
| 어깨 (건염·결림) | Tiger Balm Red, 메틸살리실산 파스, NSAIDs 겔 | 삼각근·어깨 주변 도포, 회전근개 부위는 삼가 |
| 손가락·손목 | 저농도 멘톨 겔, NSAIDs 겔 | 소량 도포, 광범위 사용 금지 |
| 허리 | 케토프로펜/메틸살리실산 파스 | 넓은 면적에 부착 가능 (단, 신장 기능 확인 후) |
| 발목 | Tiger Balm White + 냉찜질 (급성), 온감 파스 (만성) | 급성 염증은 냉감; 만성 통증은 온감 |

---

## 4. 노인 다약제 복용과 외용 진통제 상호작용

노인은 평균 5–6종 이상의 약을 복용하는 경우가 많습니다. 주요 상호작용 위험:

| 복용 약 | 위험한 외용 성분 | 위험 | 권장 행동 |
|---|---|---|---|
| 와파린(항응고제) | 메틸살리실산 (Tiger Balm Red, 파스) | INR 상승 → 출혈 위험 | 광범위 사용 금지, 의사 상담 |
| 아스피린 | 메틸살리실산 | 과도한 살리실산 축적 가능 | 소면적 사용만 |
| NSAIDs 내복제 | NSAIDs 외용 겔 (케토프로펜 등) | 전신 NSAIDs 과다 노출 | 의사 상담 후 결정 |
| 혈압약(일부) | 고농도 외용제 대량 사용 | 드물지만 혈압에 영향 가능 | 사용량 최소화 |
| 당뇨 약 | 일반적으로 상호작용 없음 | — | 피부 감각 이상 시 확인 |

> **특히 와파린 복용 노인**: Tiger Balm Red와 사론파스 등 메틸살리실산 고함유 제품의 광범위·반복 사용은 INR 상승과 출혈 위험을 높입니다. 반드시 주치의에게 사용 중인 외용 진통제를 알리십시오.

## 5. 노인 안전 사용 원칙

### 도포량과 면적

| 원칙 | 이유 |
|---|---|
| 손바닥 크기 이내의 면적에 도포 | 노인의 경피 흡수율이 높아 광범위 사용 시 전신 흡수 과다 |
| 하루 2–3회로 제한 | 과도한 반복 도포는 국소 자극 및 전신 흡수 증가 |
| 가능한 저농도 제품 선택 | Tiger Balm Red(25%) 대신 Tiger Balm White나 일반 파스 활용 |

### 파스 사용 시 특별 주의

- **밀봉 금지**: 파스 위에 랩이나 붕대로 감싸지 않음
- **동일 부위 반복 사용**: 같은 부위에 계속 붙이면 피부 자극 발생 → 하루 이상 휴식 후 재사용
- **열 병행 사용 금지**: 파스 위에 핫팩이나 전기담요를 사용하면 흡수가 급격히 증가
- **제거 시 천천히**: 피부를 손으로 지지하며 천천히 180도로 떼어냄

### 노인 피부 보호를 위한 사전 준비

- 도포 전: 피부 상태 확인 (균열·상처·발진 없는지)
- 도포 후: 손 씻기 (눈·입 접촉 방지)
- 도포 부위: 상처·개방창·습진 부위 금지

## 6. 언제 외용 진통제를 넘어 병원에 가야 하는가

| 상황 | 권장 행동 |
|---|---|
| 2주 이상 지속되는 관절 부종·열감 | 류마티스내과 또는 정형외과 방문 |
| 관절통이 점점 심해지는 경우 | 영상 검사 필요 (X-ray, MRI) |
| 외용제 사용에도 수면 방해하는 통증 | 내과 또는 통증클리닉 방문 |
| 파스 부위에 발진·수포·심한 가려움 | 즉시 사용 중단, 피부과 방문 |
| 와파린 복용 중 출혈 증상 (멍, 혈변, 잇몸 출혈) | 응급실 또는 담당의 즉시 연락 |

---

## 7. 한국 장기요양 현장 적용

| 상황 | 실천 방법 |
|---|---|
| 요양원 노인 관절통 관리 | 야간 파스 사용 시 피부 상태 매일 확인, 기록 |
| 재가 요양 방문 | 가족에게 파스 제거법·와파린 상호작용 교육 |
| 치매 노인 | 자해 방지를 위해 파스 제거 금지 위치(손이 닿지 않는 곳)에 부착 |
| 약사·의사에게 알리기 | 현재 사용 중인 모든 외용 진통제 목록 지참 |

---

## 총정리

노인의 관절통에서 외용 진통제는 전신 부작용 없이 국소 통증 완화를 제공하는 가치 있는 선택입니다. 핵심 원칙은 **소면적 사용, 저농도 제품 선호, 와파린 복용자는 메틸살리실산 제품 광범위 사용 금지**입니다. 노인의 얇아진 피부와 다약제 복용 특성을 항상 고려하고, 사용 중인 모든 외용 진통제를 주치의와 약사에게 알리는 것이 안전한 사용의 핵심입니다.
