# 藥油防治暈車暈船暈機指南

## 點解會暈車暈船？

暈動症（Motion Sickness，又稱暈動病）的核心機制係**前庭系統與視覺系統之間的訊號衝突**。

人體嘅平衡與空間定向，主要靠三套感覺系統共同配合：內耳前庭系統（感知頭部運動與加速度）、視覺系統（感知外部環境嘅運動狀況）以及本體感受系統（肌肉同關節感知身體位置）。當這三套系統傳遞嘅訊號唔一致時，大腦就難以整合出一個一致嘅「運動狀態」，從而引發混亂，觸發暈動症反應。

**典型場景例子：**
- 坐喺車後排睇書：內耳感覺到顛簸運動，眼睛卻盯住靜止嘅書頁，兩者訊號衝突
- 坐船喺船艙內：身體感受到浪漲浪落，視野入面卻係靜止嘅艙壁
- 飛機遇上亂流：前庭系統感到突然嘅三維加速度變化，座位周圍嘅視覺資訊卻冇相應改變

訊號衝突可激發延腦嘔吐中樞（chemoreceptor trigger zone）及孤束核（nucleus tractus solitarius），產生惡心、嘔吐、冷汗、頭暈等症狀。呢個係人體進化留低嘅一種「毒素預警」機制——大腦會將感覺衝突解讀為「可能有神經毒素引致嘅幻覺」，所以觸發嘔吐去排走「毒素」。

## 邊款藥油有助舒緩暈動症？

唔係所有藥油都對暈動症一樣有效。對暈動症最有幫助係**以薄荷腦（menthol）為主要成分嘅清涼型藥油**，原因如下：

1. **薄荷腦嘅神經鎮定效果**：薄荷腦能激活皮膚及嗅覺黏膜嘅冷覺受體（TRPM8），帶嚟清涼感，同時對自主神經系統有輕微穩定作用，有助緩解惡心引起嘅胃部不適。

2. **嗅覺通路抑制效果**：吸入薄荷腦嘅揮發氣體，透過嗅神經刺激大腦額葉及邊緣系統，可在一定程度上抑制惡心感嘅主觀體驗，效果類似芳香療法中使用薄荷精油嘅原理。

3. **反刺激轉移注意力**：清涼感帶來嘅皮膚刺激，有助將注意力從惡心感轉移，降低暈動症嘅主觀不適程度。

**建議藥油類型：**
- 虎標萬金油白色配方（清涼感強，薄荷腦含量高）
- 清涼油
- 含薄荷腦嘅白花油（亦含水楊酸甲酯，有一定胃部舒緩效果）
- 驅風油（風油精）——薄荷腦含量較高

含高比例樟腦嘅紅色萬金油或以溫熱感為主嘅藥油，對暈動症效果相對有限，唔係首選。

## 正確使用方法：穴位按摩技巧

藥油配合穴位按摩係民間最常見嘅暈動症緩解方法之一，其中**內關穴（P6 / Neiguan）**嘅效果有一定臨床研究支持。

### 內關穴（P6）位置

內關穴位於**前臂內側，腕橫紋上方約三橫指（約5厘米）處，兩條肌腱（掌長肌腱同橈側腕屈肌腱）之間**。用拇指按壓時通常會有輕微酸脹感。

多項隨機對照試驗（包括針灸及指壓研究）顯示，刺激P6穴對化療後惡心、術後惡心及暈動症誘發嘅惡心均有顯著緩解效果，優於安慰劑對照組。

### 使用步驟

1. 將少量薄荷腦藥油塗於兩側內關穴
2. 用另一隻手拇指適度用力按壓，做圓圈狀按摩，每側持續約1–2分鐘
3. 同時在太陽穴（兩側太陽穴）輕揉並塗抹少量藥油
4. 可將少量藥油塗於臍周圍（神闕穴），民間認為有助穩定腸胃

### 出發前30分鐘使用

呢一步好重要。藥油及穴位刺激效果需要時間建立，**建議喺乘車/船/機前約30分鐘開始使用**，唔好等到開始唔舒服先補救。預防性使用嘅效果，明顯優於已經出現症狀後嘅補救性使用。

## 與抗暈藥物比較

| 對比維度 | 藥油（薄荷腦） | 苯海拉明（乘暈寧/暈海寧） | 東莨菪鹼貼片 |
|---|---|---|---|
| 作用機制 | 嗅覺/皮膚刺激，配合穴位 | 抗組胺，抑制延腦嘔吐中樞 | 抗膽鹼能，抑制前庭衝動 |
| 起效時間 | 5–15分鐘 | 口服30–60分鐘前使用 | 出發前6–8小時貼上 |
| 效果強度 | 輕至中度 | 中度 | 強（適合較嚴重暈動症） |
| 副作用 | 極少（皮膚刺激） | 嗜睡、口乾 | 口乾、視力模糊、嗜睡 |
| 適用人群 | 輕度暈動症，兒童（注意年齡） | 大部分成人及兒童 | 成人，需處方或藥師指引 |
| 可駕駛 | 是 | 否（有鎮靜效果） | 否 |

對於輕度暈動症患者，藥油係安全且無嗜睡副作用嘅輔助首選；對於中重度暈動症患者，建議同醫生討論是否需要聯合使用藥物治療。

## 邊款藥油最適合暈動症用途？

綜合薄荷腦含量、清涼感強度及使用便利性，以下排序供參考：

1. **白花油**：薄荷腦含量高，清涼感持久，滾珠型更方便塗抹於穴位
2. **清涼油**：薄荷腦＋桉葉油組合，清新氣味有助提振精神，緩解惡心
3. **虎標萬金油白色配方**：效果均衡，國際旅行攜帶較常見
4. **驅風油（風油精）**：液態，薄荷腦濃度高，揮發快，吸入效果強

建議旅途前備一款隨身攜帶，必要時可內關穴+太陽穴聯合使用，效果優於單點塗抹。
