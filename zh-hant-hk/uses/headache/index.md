# 藥油舒緩頭痛指南

頭痛是香港都市人最普遍的日常困擾之一。無論是因為長時間對著電腦、搭車時手機用太久、工作壓力，還是香港夏天的悶熱潮濕，頭痛往往在最不便的時候找上門。在未達到需要服藥的程度時，不少港人會先以藥油舒緩，這是一個流傳多代的生活習慣。

本文介紹如何正確使用藥油舒緩頭痛，以及在香港屈臣氏、萬寧等藥房可以找到的產品選擇。

## 藥油為何能緩解頭痛？

藥油舒緩頭痛的機制主要依賴以下幾個原理：

**清涼感分散注意力（感覺替代）**
薄荷腦（Menthol）接觸皮膚後，會刺激皮膚上的TRPM8受體，產生清涼感。大腦同時處理清涼感覺與頭痛信號時，會在一定程度上減低痛感的感知強度。

**促進局部血液循環**
輕力按摩太陽穴配合藥油，能促進局部微循環，有助舒緩因血管收縮引起的緊張性頭痛。

**芳香療法效果**
薰衣草油及桉樹油的香氣，對部分人有輕度鎮靜效果，有助放鬆情緒，間接緩解壓力型頭痛。

## 適合舒緩頭痛的常見藥油

### 白花油（Pak Fah Yeow / White Flower Oil）

**最適合**：緊張性頭痛、壓力頭痛、輕度偏頭痛
**氣味**：清涼淡雅（薰衣草+薄荷）
**用法**：取1-2滴於指尖，輕輕塗於太陽穴及前額中央，以畫圈方式按摩30秒

白花油是香港最普遍的舒緩頭痛選擇，清淡氣味在辦公室使用較不擾鄰。

**香港購買**：屈臣氏（全線分店）、萬寧（全線分店）、百佳藥品區、HKTVmall

### 虎標萬金油白色款（Tiger Balm White）

**最適合**：頭痛、鼻塞伴隨的頭痛
**氣味**：辛涼濃烈（樟腦+薄荷+薑）
**用法**：取少量軟膏塗於太陽穴，避免接觸眼睛周圍

虎標氣味較濃，在開放式辦公室需留意對同事的影響。蠶豆症患者請先閱讀成分表，樟腦成分可能有風險。

**香港購買**：屈臣氏、萬寧、各大超市、藥行

### 清涼油（薄荷腦膏）

**最適合**：廣泛性頭痛、暑熱頭痛
**特點**：價格便宜，體積小巧，方便攜帶
**香港購買**：藥行、部分萬寧分店

### 德國寶Hansaplast薄荷舒緩貼布

**最適合**：不習慣藥油氣味者
**特點**：含薄荷醇成分，貼於額頭使用，氣味較淡，適合在辦公室低調使用

## 正確使用方法（以白花油為例）

1. **準備**：洗手或確保手部清潔，避免雜菌帶入眼口鼻
2. **取量**：瓶口傾斜，讓藥油自然滴出1-2滴於食指指腹，切勿過量
3. **塗抹部位**（可選其一或組合）：
   - **太陽穴**：耳朵與眼角之間的凹陷處，兩側各塗一次
   - **前額**：眉心至前額中央，薄薄一層
   - **後頸**：頸部兩側肌肉，尤其適合頸源性頭痛
4. **按摩**：以指腹輕輕打圈，力度適中，持續1-2分鐘
5. **閉目休息**：有條件的話閉眼靜坐5-10分鐘，效果更佳
6. **洗手**：塗抹後立即洗手，防止觸碰眼睛

## 哪類頭痛適合用藥油？

| 頭痛類型 | 藥油效果 | 建議行動 |
|---|---|---|
| 緊張性頭痛（工作疲勞） | 有效 | 可先嘗試藥油+按摩 |
| 壓力/情緒頭痛 | 有一定效果 | 配合休息，藥油輔助 |
| 暈車/暈船引起 | 有效（塗人中及手腕） | 適合 |
| 輕度偏頭痛（初期） | 部分人有效 | 可嘗試，如無改善需就醫 |
| 嚴重偏頭痛 | 效果有限 | 應就醫或按醫囑服藥 |
| 發燒伴頭痛 | 不建議依賴藥油 | 應就醫 |
| 突然劇烈頭痛（雷擊性） | 無幫助 | 立即召喚救護車 |

## 在香港藥房選購的實用貼士

**屈臣氏（Watsons）**
- 分店遍佈全港，大型分店設有「家庭護理」或「藥品」專區
- 較常見白花油大、中、小三種瓶裝，可按需要選購
- 會員有積分，可定期留意會員特賣

**萬寧（Mannings）**
- 同樣全港覆蓋，部分分店藥品區較屈臣氏更齊全
- 不定期推出藥品類買一送一或折扣優惠

**百佳超級市場（ParknShop）**
- 部分分店（尤其大型店）設有藥品區，白花油及萬金油均有供應
- 適合順道購買生活用品時一併入手

**街坊藥行**
- 香港各區仍有不少傳統藥行，通常備有各種藥油，包括一些較難在連鎖藥房找到的品牌
- 店員通常對藥油用法較熟悉，可即場請教

**HKTVmall**
- 適合不想出門購物或需要送貨上門者
- 可使用搜尋功能比較不同品牌藥油的價格

## 注意事項與禁忌

- **蠶豆症（G6PD缺乏症）患者**：部分藥油成分可能觸發溶血，使用前請先諮詢醫生。詳見本站〈蠶豆症患者用藥油注意事項〉。
- **兩歲以下嬰幼兒**：禁止使用含薄荷腦或樟腦的藥油
- **孕婦**：建議避免使用，或先徵詢醫生意見
- **眼睛周圍**：任何藥油均不可接觸眼睛，若不慎接觸應立即用清水沖洗
- **破損皮膚**：不可塗於傷口或紅腫破損的皮膚上
- **頭痛持續超過72小時或劇烈難忍**：應立即就醫，藥油只適合輕度舒緩，不能取代醫療診斷

## 何時必須就醫？

以下情況不應依賴藥油，必須立即求醫：

- 頭痛伴隨發燒超過38.5°C
- 頭痛突然劇烈，如「一生中最嚴重的頭痛」（可能是蜘蛛網膜下腔出血徵兆）
- 頭痛伴隨視力模糊、說話困難或肢體無力
- 兒童或老人頭痛劇烈且持續
- 頭痛伴隨頸部僵硬及怕光（腦膜炎警號）

**緊急情況請撥999。**
