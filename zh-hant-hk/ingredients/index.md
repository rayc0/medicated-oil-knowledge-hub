# 藥油成分大全

藥油嘅功效主要由其成分決定。了解主要成分嘅藥理作用同安全邊界，有助於更合理咁選擇藥油，避免唔必要嘅風險。本頁彙整藥油中最常見嘅活性成分，按成分類別分組，方便日常使用及查閱。

- **[薄荷腦藥理完整指南](menthol-pharmacology.md)** — 薄荷腦嘅鎮痛、清涼同止癢機制，以及同薄荷精油嘅區別。

- **[廣藿香藥理完整指南](patchouli-huo-xiang-pogostemon-cablin-pharmacology.md)** — 廣藿香醇嘅抗炎與芳香化濕特性。
- **[桂枝藥理完整指南](gui-zhi-cinnamon-twig-pharmacology.md)** — 桂枝揮發成分嘅溫通經脈與散寒止痛作用。
- **[花椒藥理完整指南](sichuan-pepper-hua-jiao-zanthoxylum-bungeanum-pharmacology.md)** — 花椒麻木感嘅成分基礎與外用鎮痛機制。
- **[兩面針藥理完整指南](liang-mian-zhen-zanthoxylum-nitidum-pharmacology.md)** — 兩面針活性生物鹼與消炎止痛作用。
- **[薄荷油與留蘭香油比較](peppermint-versus-spearmint-comparison.md)** — 兩種薄荷精油成分差異與外用效果對比分析。
- **[薄荷油與薄荷腦晶體對比指南](peppermint-oil-versus-menthol-crystal-guide.md)** — 精油與提純薄荷腦嘅功效、純度與用途差異。
