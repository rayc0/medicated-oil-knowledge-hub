# 薑精油（薑油）藥理 — 月子油、薑活絡油與亞洲溫通油嘅真正主角

如果你喺廣東婆婆嘅月子房裡替產婦推過腰背，在泰國清邁嘅泰式按摩床上聞過 luk pra kob 草藥包嘅薑味蒸氣，或者喺印度南部阿育吠陀診所畀人用 Mahanarayan 油按過膝關節，你其實早已見過整個亞洲外用醫藥裡最古老、亦最普遍被誤讀嘅一味：**薑**。

薑油，源自薑科植物 *Zingiber officinale* 嘅根茎，喺中醫裡有四種身份（生薑、乾薑、炮薑、薑炭）；喺國際藥油標籤上通常只剩一行字 — "ginger oil" 或 "薑根油"。問題係：消費者以為薑油辣，但**蒸餾出來嘅薑精油其實並唔辣**。你印象中「灼熱鑽進皮下」嘅感覺，其實唔住喺精油入面；佢住喺另一個好多人體會叫唔出名嘅馏分。

呢篇文章就係要將呢件事講清楚 — 薑油到底係乜嘢、打到邊個受體、佢同薑辣素差咩，以及點解佢會同時出現在港式月子油、印尼 minyak jahe、泰式草藥包同阿育吠陀關節油裡。

## 一、四種薑，對應四種外用邏輯

中醫對薑嘅處理遠比西方香料學細緻。同一條薑，炮製方式一改，歸經、性味、臨床定位全部改咗：

- **生薑（鮮薑）**：辛、微溫，歸肺、脾、胃。揮發油含量最高，含檸檬醛、橙花醛、薑烯等倍半萜與單萜。內服止嘔散寒，外用走表，常拌入嬰幼兒驅風油、月子薑醋油、跌打急救油裡「開毛孔、散表寒」。
- **乾薑**：辛、熱，歸脾、胃、腎、心、肺。生薑陰乾後薑辣素（6-gingerol）會脫水生成更猛烈嘅薑烯酚（6-shogaol），所以乾薑走裡、溫中、回陽。外用層面，乾薑浸出油更辣、更深、更持久——呢個就係廣東「老薑活絡油」比鮮薑油更刺辣嘅原因。
- **炮薑**：乾薑炒至外焦褐、內黃，主入血分，溫經止血。喺月子方劑入面炮薑比乾薑更常見，因為佢「辛而不竄」，溫而不耗氣血。粵港嘅產後薑醋、薑油、薑灸條幾乎都用炮薑而唔係生薑。
- **薑炭**：炒至存性，止血用，外用藥油幾乎唔用。

將整條薑送去蒸餾出嚟嘅「**薑精油（薑油）**」，又係另一回事 — 從化學層面更接近「薑嘅香氣骨架」，未必係「薑嘅火力」。

## 二、薑精油 vs. 薑油樹脂：同一條薑，兩種靈魂

工業上從薑根茎可以得到兩種完全不同嘅外用萃取物：

1. **蒸餾薑精油（steam-distilled essential oil）**  
 產率約 1.5–3%，主要成分係**倍半萜烃**：薑烯（zingiberene 25–35%）、β-倍半水芹烯、ar-薑黃烯、β-沒藥烯，再加少量單萜（樟烯、α/β-蒎烯、β-水芹烯）同檸檬醛。色淡黃至淺琥珀，氣味溫辛帶檸檬尾韻。**幾乎不辣**，皮膚上只有輕微溫感。

2. **CO₂ 超臨界油樹脂（oleoresin）**  
 將整株鮮薑或乾薑用超臨界二氧化碳萃取，連同揮發性成分**同**非揮發性薑辣素一齊帶出。顏色深褐近黑，質地黏稠，**辣得很**。入面除咗上述倍半萜之外，仲有**薑辣素（gingerols）、薑烯酚（shogaols）、薑酮酚（paradols）、薑酮（zingerone）**。

呢兩樣嘢喺化妝品級 INCI 表度睇嚟都叫「Zingiber Officinale Root Oil / Extract」，但藥理幾乎係兩種完全不同嘅物種。

**關鍵事實**：薑辣素與薑烯酚分子量大、極性高，**根本冇法隨水蒸氣揮發**。蒸餾精油入面你搵唔到有任何有意義濃度嘅 6-gingerol 或 6-shogaol。所以如果有桶月子薑油聲稱「含薑精油」但又強調「溫熱透骨」，要唔就佢另外加咗油樹脂，要唔就加咗辣椒素，要唔就佢所謂「溫熱」主要係靠樟腦/桉油/水楊酸甲酯，而唔係薑本身。

懂得呢件事，會令你喺睇跌打油、活絡油、關節膏嘅成分表時，立即分辨到——呢家配方師係認真定係只係玩宣傳。

## 三、薑精油到底打邊個受體？三套機制並行

薑喺皮膚上嘅藥理行緊三條路，**行邊條取決於你用嘅係精油定油樹脂**。

### 1. TRPV1：溫感與鎮痛同源

**6-薑辣素、6-薑烯酚、薑酮**係 **TRPV1 受體嘅部分激動劑**。TRPV1 就係辣椒素受體，專責「熱」同「灼」訊號。被部分活化時：

- 局部血管擴張（rubefacient 紅皮作用），皮溫上升 1–3°C
- 初始溫/灼感
- 持續刺激下傷害感受神經末梢去敏感化，痛覺訊號反而衰減

呢個就係乾薑油樹脂抹喺膝蓋上會「先熱後鎮痛」嘅神經學基礎。生理上佢同辣椒素同源，但溫度只有辣椒素嘅 1/3 到 1/2，所以風險窗口較闊，更適合中老年人長期外用。

**重點**：純蒸餾薑精油幾乎唔打 TRPV1。所以如果你用嘅係單方薑精油配基礎油，佢畀到你嘅係溫和擴張血管嘅「暖」，唔係辣椒素式嘅「焮」。

### 2. 抗炎：抑制 COX-2、5-LOX、NF-κB

薑辣素與倍半萜都能抑制環氧合酶-2（COX-2）和 5-脂氧合酶（5-LOX），同時下調 NF-κB 通路。呢係骨關節炎、慢性肌筋膜痛人群外用薑油有效嘅分子學解釋。研究入面 6-薑烯酚嘅抗炎活性比 6-薑辣素更強，所以**乾薑系制劑**喺慢性炎症外用上比生薑系更有力。

倍半萜（薑烯、ar-薑黃烯、β-沒藥烯）則提供另一層抗炎與抗氧化作用，同時改善精油喺皮膚上的展布與滲透。

### 3. 血管動力學與「溫通」

喺中醫語境，薑嘅「溫經散寒、化瘀通絡」喺現代解釋大致對應：

- 局部血管擴張 → 增加皮下與肌肉血流灌流
- 神經末梢 P 物質釋放被調節 → 慢性痛訊號下行
- 抑制血小板聚集（輕度，與阿司匹林作用機制不同但方向相同）

最後呢一點，正係月子薑油、產後溫經油、痛經外用油嘅核心機制 — 它唔係表層「熱敷」，而有真實嘅血流動力學改變。

## 四、幾種亞洲傳統配方入面嘅薑，到底係做乜嘢

### 港式 / 廣式月子薑油

廣東人產後必拜嘅「薑醋雞蛋」係內服，而月子房裡俾產婦推拿嘅薑油係外用。配方常見以**炮薑或老薑**為基底，浸入花生油或茶油，再加少量艾葉油、肉桂油、當歸油。功能定位：溫腰腎、推惡露、防風寒入骨。藥理上：炮薑嘅薑辣素（已轉化為含氧度更高嘅薑酮、6-薑烯酚）+ 艾葉油（側柏酮）+ 肉桂醛，三套 TRPV1 / TRPA1 激動劑疊加，可喺產後腰骶部穩定提高 2–3°C 嘅局部皮溫，約 60–90 分鐘。

### 跌打油入面嘅薑

廣式跌打油同川派跌打油都會加薑，但加法唔同。廣式偏向直接用**乾薑或老薑浸油**，再同紅花、川芎、三七一齊浸出。川派則更可能用**薑油樹脂**配合川烏、草烏、樟腦去做「快速透熱」。兩者喺皮膚見到嘅紅皮反應唔同 — 廣式偏柔和持續，川派偏猛烈短促。

### 印尼 minyak jahe / jamu 薑油

爪哇式 jamu（草藥）傳統入面，薑叫做 *jahe*。**Minyak jahe** 通常係用整株 *jahe merah*（紅薑，*Zingiber officinale* var. rubrum）連皮搗碎，用椰子油低溫浸出最少 14 日。佢同蒸餾精油完全不是一回事 — 幾乎所有薑辣素同薑烯酚都會被椰子油萃取出嚟，所以佢喺皮膚上又辣又持久，是巴厘島傳統按摩 *pijat* 畀勞累農夫嘅標準配方之一。

### 泰式 luk pra kob 草藥包

清邁、清萊嘅草藥蒸壓包配方，鮮薑同南薑（*Alpinia galanga*）、香茅、薑黃、羅望子葉一齊蒸熱。蒸氣一散開，薑烯、樟烯、桉油精同薑黃素同時喺角質層上發揮。呢種濕熱環境下薑辣素都可以從鮮薑析出一部分，所以草藥包一壓喺肩背上，係「濕熱 + 倍半萜 + 微量薑辣素」三層效應疊加，比單一薑精油抹上去更強。

### 阿育吠陀 Mahanarayan / Mahabhringaraj taila

南印度嘅多年生芝麻油基底油入面，薑（*Sunti* 乾薑）同近三十味草藥一齊經 *kalka*（草藥泥）+ *kvatha*（草藥水）+ *taila*（油）三相 *paka* 工藝慢火熬煮。最終成品除咗有鮮薑嘅揮發萜烯，仲有乾薑喺長時間加熱中析出嘅 6-薑烯酚。喺阿育吠陀分類上係 *vata* 鎮定油，專用於關節強直、坐骨神經痛、產後腰背痛。用現代藥理睇，佢係一款天然多組分 TRPV1 緩釋制劑。

## 五、外用安全：呢段唔好跳過

薑唔係「絕對安全嘅天然溫通」，佢嘅外用風險有幾個明確邊界：

### 濃度上限

- **蒸餾薑精油**喺面部、頸部不超過 1%，身體不超過 2%。嬰幼兒（< 2 歲）禁用純精油外擦。
- **薑油樹脂**作用強 5–10 倍，外用濃度通常做到 0.1–0.3% 已足。
- 月子薑油、產後腰部按摩油因為浸出工藝濃度低同基礎油稀釋，使用上比單方精油寬鬆，但每次用量以**手掌大面積 5–10 滴**為限。

### TRPV1 類受體叠加

如果你配方裡同時有薑油樹脂 + 辣椒素 + 樟腦 + 水楊酸甲酯，**佢哋會喺受體水平疊加**。皮膚敏感人群可能直接出現紅斑、燒傷甚至神經性疼痛反跳。呢就係某啲「猛料跌打油」喺年輕人群體入面翻車嘅真實原因——唔係單一草藥出問題，而係受體被多次刺激。

### 與抗凝藥嘅交互作用

口服薑對血小板聚集抑制有較多文獻支持。外用薑油經皮吸收嘅 6-薑辣素 / 6-薑烯酚雖然量細，但長期大面積使用，對而家服用華法林、阿司匹林、氯吡格雷嘅人群仍有理論上嘅累加風險。月子產婦如同時用低分子肝素，請先諮詢醫生再用薑油按摩。

### 光毒性與檸檬醛過敏

薑精油入面嘅檸檬醛（geranial + neral）含量雖唔高，但對**檸檬醛 / 香茅醛過敏人群**仍可能引起接觸性皮膚炎。建議初次使用前喺前臂內側做 24 小時貼紋試驗。薑精油本身**唔屬於光毒性精油**（同佛手柑、萊姆冷壓油唔同），白天使用冇禁忌。

### 孕期

孕中後期腹部、腰骶部使用濃度 ≤ 1% 嘅薑精油按摩，主流芳療同傳統中醫都視為可接受。但**孕早期（< 12 週）腹部、內踝三陰交附近避免使用薑油樹脂級別嘅高濃度配方**，理論上有活血風險。月子期反而係薑油嘅主戰場。

## 六、俾消費者嘅一句藥理結論

讀完組成表後，可以咁判斷你手上呢支薑油屬邊一類：

- **只寫「Ginger Oil / Zingiber Officinale Root Oil」** → 幾乎肯定係蒸餾精油。香氣重、溫感輕、抗炎為主、TRPV1 弱激活。適合日常頸肩、面部遠端使用。
- **寫「Ginger Extract / Oleoresin / CO₂ Extract」** → 含薑辣素 / 薑烯酚。辣、熱、持久。適合關節、腰背、慢性肌肉勞損。皮膚敏感者需降濃度。
- **寫「炮薑浸油 / Dried Ginger Infused Oil」** → 傳統月子油嘅主流形態。溫和持續，適合產後調理與中老年腰膝保養。
- **同時含薑油 + 辣椒素 + 樟腦 + 水楊酸甲酯** → 高強度跌打 / 活絡油，分子機制係 TRPV1 + TRPM8 + TRPV1 + COX 抑制四件套。效果猛，風險亦疊加，按部位用，唔好全身大面積抹。

薑由一條平平地嘅廚房根茎，轉化成「生薑、乾薑、炮薑」三套外用藥理截然不同嘅中藥材，再到現代精油、油樹脂、CO₂ 萃取三種工業品，背後其實係 5000 年臨床觀察沉澱嘅細緻辨證。下次有人隨口將「薑油」兩個字寫喺配方表上，最少會多問一句：**係邊種薑，邊種萃取，打緊邊個受體。**

懂呢個道理，你已經先人一大截，喺呢個市場入面走得比 99% 消費者同 80% 零售藥房櫃檯同事更前。
