# 冬青油（冬青醇 / 水楊酸甲酯）解析

## 什麼是冬青油與水楊酸甲酯

冬青油（Wintergreen Oil）是從冬青植物（*Gaultheria procumbens*，又稱東方冬青或加拿大冬青）的葉片中蒸餾提取的精油，主要成分為**水楊酸甲酯（Methyl Salicylate）**，含量可高達98%。

水楊酸甲酯是一種水楊酸衍生物，與我們熟悉的**阿士匹靈（乙酰水楊酸）**同屬水楊酸家族。兩者在化學結構上高度相近，藥理機制也有重疊：均能在體內轉化為水楊酸，發揮抗炎和鎮痛效果。事實上，水楊酸甲酯在體內的代謝路徑可類比為「局部吸收版的阿士匹靈」，這一特性令它成為外用肌肉止痛藥膏中最有效的成分之一。

除了天然提取，現代生產中絕大多數水楊酸甲酯以化學合成方式製備，由水楊酸與甲醇在酸催化下酯化反應得到，成本低廉，純度更易控制。

## 作用機制：深入皮膚的止痛原理

與許多只停留在皮膚表層的成分不同，水楊酸甲酯有著優異的經皮滲透能力。

**步驟一：經皮吸收**
水楊酸甲酯的脂溶性較高，能穿透皮膚角質層並進入真皮層，繼而被毛細血管吸收進入局部循環。其滲透速率在所有外用止痛成分中屬於較高水平，遠勝於單純依靠揮發性氣味起效的成分（如薄荷腦的清涼感）。

**步驟二：水解為水楊酸**
在皮膚組織中，酯酶將水楊酸甲酯水解為水楊酸（salicylic acid）和甲醇。水楊酸在局部組織中發揮抗炎效果，抑制前列腺素（prostaglandins）的合成——而前列腺素正是炎症與疼痛感的重要介質。

**步驟三：局部抗炎鎮痛**
透過抑制前列腺素的生成，水楊酸減少局部組織的炎症反應，從而紓緩肌肉痠痛、關節疼痛及運動後的遲發性肌肉痠痛（DOMS）。這種深層作用是單純依靠「清涼感反刺激」的薄荷腦無法做到的。

## 在各類藥油中的濃度

水楊酸甲酯是多種經典藥油的核心成分，但在不同產品中濃度差異顯著：

| 產品 | 水楊酸甲酯含量 | 特點 |
|---|---|---|
| 白花油（白花活絡油） | 約40%–60% | 濃度最高，深層肌肉止痛效果最強 |
| 虎標萬金油（紅色） | 約3%–5% | 作為輔助成分，主要止痛依靠樟腦 |
| 虎標萬金油（白色） | 較低或不含 | 以清涼感為主 |
| 驅風油（風油精） | 約20%–35% | 兼顧清涼感與深層滲透 |
| 肌肉止痛貼／噴霧（如Icy Hot、Bengay） | 10%–30% | 歐美市場專注於運動損傷恢復 |

白花油的水楊酸甲酯濃度在亞洲市場藥油中屬於最高水平，這正是白花油相較於虎標萬金油，在紓緩深層肌肉痠痛和關節不適方面往往更為有效的原因。

## 與血液稀釋藥物的重要交互警告

這是使用含水楊酸甲酯產品最重要的安全問題之一，但極少被使用者留意到。

**華法林（Warfarin）與其他抗凝血藥**

水楊酸家族成分（包括外用途徑吸收的水楊酸甲酯）會與華法林（warfarin）發生顯著藥物相互作用。水楊酸能抑制維生素K依賴性凝血因子的活性，並可能置換華法林與血漿蛋白的結合，導致華法林血藥濃度升高，增加出血風險。

大面積、頻密塗搽含高濃度水楊酸甲酯的產品（如白花油），其經皮吸收量足以對華法林用藥者產生臨床意義的影響。醫學文獻中已有相關病例報告。

**阿士匹靈長期使用者**

長期服用阿士匹靈（例如心血管疾病預防）的患者，同時使用高濃度水楊酸甲酯外用產品，可能進一步增加水楊酸相關副作用（如腸胃刺激、出血傾向）的風險。

**實用建議**

- 正在服用華法林、氯吡格雷（Plavix）、達比加群（Pradaxa）等抗凝／抗血小板藥物的患者，使用任何含水楊酸甲酯成分的外用藥油前，**必須諮詢醫生或藥劑師**
- 不建議大面積或每日多次使用高濃度水楊酸甲酯產品
- 使用後避免用密封材料包裹塗搽部位，以防經皮吸收量急劇增加

## 皮膚吸收率與劑量考量

水楊酸甲酯的經皮吸收率受多種因素影響：

- **濃度**：濃度越高，單位面積吸收量越大
- **塗搽面積**：面積越大，總吸收量越高，全身影響越顯著
- **皮膚狀態**：皮膚有破損、炎症或用密封敷料覆蓋時，吸收率大幅增加
- **使用頻率**：每日多次塗搽的累積劑量可能產生系統性影響

## 注意事項

- **阿士匹靈過敏者禁用**：水楊酸甲酯與阿士匹靈同屬水楊酸家族，存在交叉過敏風險
- **兒童慎用**：大面積使用可能引起水楊酸中毒（salicylism），症狀包括耳鳴、噁心、呼吸急促
- **孕婦禁止大量使用**：高劑量水楊酸類藥物在妊娠期有潛在風險
- **皮膚破損處禁用**：可引起刺激並加劇全身吸收
- **不可內服**：水楊酸甲酯口服毒性極強，5毫升純品即可危及兒童生命
