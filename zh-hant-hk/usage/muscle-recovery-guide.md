# 運動後肌肉恢復藥油指南：減輕DOMS同加速恢復

做完gym、跑完步、打完波之後第二日全身痠痛——呢個就係**延遲性肌肉痠痛（DOMS, Delayed Onset Muscle Soreness）**。DOMS通常喺運動後12-24小時開始出現，48-72小時達到高峰，之後逐漸消退。

藥油喺DOMS管理中嘅角色主要係**促進血液循環、減輕痠痛感覺、加速代謝廢物清除**。配合正確嘅按摩手法，可以明顯縮短恢復時間。

---

## DOMS嘅科學背景

DOMS嘅成因唔係「乳酸堆積」（呢個係過時嘅說法）。真正嘅原因係：

1. **微觀肌肉損傷**：高強度或新動作造成肌肉纖維微撕裂
2. **炎症反應**：免疫系統對損傷嘅修復反應
3. **神經敏感化**：損傷區域嘅痛覺神經閾值降低

藥油主要作用於第三點——透過皮膚刺激（溫感/涼感）同促進血流，調節痛覺信號。

## 運動後藥油使用時機

| 時間點 | 建議 | 原因 |
|---|---|---|
| 運動後即刻 | ❌ 唔好馬上搽 | 肌肉仍處於充血狀態 |
| 運動後1-2小時 | ⚠️ 涼感藥油可以 | 幫助降溫同舒緩 |
| 運動後6-12小時 | ✅ 最佳時機 | DOMS開始前介入最有效 |
| DOMS高峰期（24-48hr） | ✅ 積極使用 | 配合按摩效果最好 |
| DOMS消退期（48-72hr） | ✅ 可以繼續 | 加速恢復收尾 |

## 各部位按摩手法

### 大腿（股四頭肌/腿後肌群）

呢個係跑步、踩單車、深蹲後最常痠痛嘅部位。

**手法**：將虎標萬金油（紅色）搽喺手掌搓暖，然後用手掌從膝蓋上方向大腿根部方向推——即係**由下往上**推。每條腿推10-15次，力度由輕到中等。

### 小腿（腓腸肌/比目魚肌）

跑步同行山後最常見嘅痠痛位置。

**手法**：坐低，一隻腳擱喺另一隻腳膝蓋上。搽藥油後用拇指從腳跟上方沿小腿中線向上推至膝蓋後方。每條腿8-10次。

### 肩膊（三角肌/斜方肌）

游水、打羽毛球、做overhead press後。

**手法**：搽藥油後用對側手從肩膊外側向頸部方向推壓，同時轉動頭部。配合聳肩-放鬆動作重覆5-8次。

### 腰背（豎脊肌）

深蹲、硬舉、划船後。

**手法**：搽藥油喺腰背位置，用雙手拇指沿脊柱兩側從腰部向上推至肩胛骨下方。唔好直接按壓脊柱骨。

## 藥油效果比較（肌肉恢復專用）

| 藥油 | 溫感/涼感 | 止痛效果 | 促進血流 | 氣味 | 適合部位 |
|---|---|---|---|---|---|
| 虎標萬金油（紅） | 溫感為主 | ★★★★☆ | ★★★★★ | 強 | 大肌群（腿、背） |
| 虎標萬金油（白） | 涼感為主 | ★★★★☆ | ★★★☆☆ | 中等 | 肩頸、小肌群 |
| 黃道益活絡油 | 溫感 | ★★★★★ | ★★★★★ | 強 | 關節附近、深層 |
| 均隆驅風油 | 涼感強烈 | ★★★☆☆ | ★★★★☆ | 強烈 | 大範圍痠痛 |
| 正紅花油 | 溫感 | ★★★★☆ | ★★★★★ | 中等 | 瘀傷、腫脹 |
| 白花油 | 涼感 | ★★★☆☆ | ★★☆☆☆ | 清新 | 輕度痠痛、提神 |

**推薦選擇**：DOMS最佳係虎標萬金油（紅色）或黃道益——溫感促進血流嘅效果對DOMS恢復最有利。

## 藥油配合其他恢復方法

| 恢復方法 | 配合藥油 | 效果加成 |
|---|---|---|
| 熱水浸浴 | 浸完之後搽藥油 | 毛孔擴張增強吸收 |
| 泡沫軸（Foam Roller） | 搽藥油後滾壓 | 雙重促進血流 |
| 伸展運動 | 搽藥油後做靜態伸展 | 溫感幫助肌肉放鬆 |
| 壓力衣 | 先搽藥油，乾透後著壓力衣 | 加壓 + 藥油雙效 |
| 按摩槍 | 搽藥油後用按摩槍 | 深層滲透效果更好 |
| 冰浴 | 唔好配合溫熱藥油 | 效果互相抵消 |

## 常見錯誤

1. **運動完即刻大範圍搽藥油**：等1-2小時先，等肌肉降溫
2. **搽完藥油包保鮮紙**：會造成過度刺激同皮膚灼傷
3. **每日搽超過4次**：藥油成分累積可能造成皮膚敏感
4. **痠痛超過5日仍然只搽藥油**：可能係肌肉撕裂或其他傷害，需要求醫
5. **搽喺有傷口嘅位置**：藥油入傷口會刺激同延遲癒合

## 總結

| 重點 | 建議 |
|---|---|
| 最佳使用時機 | 運動後6-12小時（DOMS前介入） |
| 最佳藥油 | 虎標紅色 / 黃道益（溫感促血流） |
| 按摩方向 | 由下往上（向心臟方向） |
| 最佳配合 | 熱水浸浴後 + 藥油 + 伸展 |
| 避免 | 運動後即刻搽、包保鮮紙、用過量 |
| 何時求醫 | 痠痛超過5日、腫脹、活動受限 |
