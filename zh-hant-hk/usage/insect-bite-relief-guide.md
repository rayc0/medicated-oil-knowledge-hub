# 藥油治蚊蟲咬傷完全指南：止癢消腫與正確使用

夏天一到，蚊蟲咬傷幾乎是每個家庭都無法避開的煩惱。被咬後皮膚又紅又痕，越搔越腫，甚至影響訓眠。對不少港式家庭來說，藥油（跌打、萬金油、白花油）都是應對蚊蟲咬傷最快的選擇。本文從生理機制講解為何會痕癢、為何會腫，詳盡介紹藥油止癢消腫原理，並針對不同蟲類咬傷提供處理方法，幫助全家人在咬蟲高發季節更安心。

---

## 一、被蚊蟲咬後為何會痕、會腫？

要正確使用藥油，首先要明白咬傷反應的生理機制。

### 1.1 組胺釋放：痕腫的根源

當蚊子刺穿皮膚吸血時，會將唾液注入傷口。唾液中含有抗凝血成分和異體蛋白，人體免疫系統會將其視為外來入侵者，隨即啟動防禦反應：皮膚內的**肥大細胞（mast cells）**被活化，大量釋放**組胺（histamine）**。

組胺是引發咬傷症狀的核心物質，主要作用包括：

- **擴張局部毛細血管**：引起皮膚發紅、發熱
- **增加血管通透性**：血漿滲出到組織間隙，形成腫包（風團）
- **刺激感覺神經末梢**：活化癢覺受體（尤其是 H1 受體），產生強烈的搔癢感

所以你會發現被咬後幾分鐘內便出現紅腫及劇烈搔癢——這是免疫系統在快速反應，並非一定代表感染。

### 1.2 為何越抓越癢？

抓搔皮膚會造成輕微機械性傷害，刺激更多肥大細胞脫顆粒，釋放更多組胺，形成惡性循環。再者，抓搔還會破壞皮膚屏障，增加細菌感染風險，導致咬傷處繼發感染，紅腫進一步加重。

## 二、藥油止癢消腫的科學原理

傳統藥油之所以能紓解蚊蟲咬傷，關鍵在於其活性成分對皮膚神經受體的作用。

### 2.1 薄荷腦：激活 TRPM8 受體產生涼感，覆蓋癢感

薄荷腦（menthol）是多數清涼藥油的核心成分，其止癢機制建立在**神經競爭性抑制**原理上：

薄荷腦可選擇性活化皮膚感覺神經元上的**TRPM8 離子通道**（又稱「冷感受體」）。TRPM8 通道在低溫（通常低於 26°C）時自然開啟，向腦部傳遞「涼」的訊號。薄荷腦在常溫下即可活化該通道，帶來明顯涼感。

這種涼感訊號由同一條感覺神經傳入腦部，**在神經層面與癢感訊號形成競爭**，有效「覆蓋」或「抑制」癢覺的感知，亦因此塗抹薄荷腦後會覺得「涼涼」而痒感即時減輕。此外，薄荷腦亦有輕度局部麻醉作用，可直接降低神經末梢興奮性，進一步減少不適。

**關鍵點**：薄荷腦的止癢效果是透過神經訊號干預達成，並非直接抑制組胺，因此見效快，但持續時間有限，需要按需重複塗抹。

### 2.2 樟腦：激活 TRPV3 受體，溫熱減腫

樟腦（camphor）在藥油中往往與薄荷腦協同使用，兩者機制不同、效果互補。

樟腦主要活化**TRPV3 離子通道**（一種溫熱感受體），產生溫和的溫熱感。這種溫熱效應可：

- **促進局部血液循環**：加快淋巴回流，幫助組織液消散，減輕腫脹
- **加速組胺代謝**：局部血流加快，有助組胺等炎症介質的稀釋與代謝清除
- **輕度抗炎**：樟腦本身具一定抗炎特性，可減少炎症介質持續釋放

值得注意，樟腦於低濃度（1%–3%）時可展現止癢及溫熱效應，但**高濃度使用或大面積塗抹可能產生刺激**，因此藥油中的樟腦含量需嚴格控制。

### 2.3 其他常見活性成分

除薄荷腦和樟腦外，優質藥油常還含下列成分：

| 成分 | 作用 |
|------|------|
| **丁香油（eugenol）** | 天然止痛，溫和麻醉局部神經末梢 |
| **桉葉油（cineole）** | 抗菌消炎，減少咬傷處繼發感染 |
| **水楊酸甲酯（methyl salicylate）** | 抗炎鎮痛，促進局部血液循環 |
| **薰衣草精油** | 鎮靜舒緩，適合兒童及敏感肌膚 |

## 三、不同咬傷的處理方法

不同昆蟲咬傷機制、注入成分與危險程度各異，處理方式亦應有所分別。

### 3.1 蚊子咬傷

**特徵**：紅色小圓包，周圍有風團，癢覺明顯，通常在數小時內逐步退去。

**處理步驟**：
1. **勿抓搔**：立即用指腹輕壓（非抓撓）咬傷處，短暫阻斷神經訊號可稍為減輕癢感
2. **清潔**：用清水沖洗咬傷部位，去除皮膚表面殘留分泌物
3. **塗抹藥油**：取少量藥油直接點塗於咬點中心，輕輕揉開至約1–2厘米範圍
4. **待涼感出現**：約30秒至1分鐘，薄荷腦的涼感起效後，癢感可明顯減輕
5. **按需重塗**：若1–2小時後癢感再起，可重複塗抹，每日不超過4–5次

### 3.2 螞蟻咬傷

**特徵**：螞蟻（尤其是紅火蟻）咬傷會注入螞蟻酸（甲酸），引起灼燒感和刺痛，部分人會出現充滿液體的小水泡。

**處理步驟**：
1. **立即離開螞蟻聚集區**，避免繼續被咬
2. **冷水沖洗**：螞蟻酸偏酸性，可用清水（或少量小蘇打）沖洗中和
3. **冷敷**：用冰袋隔布冷敷5–10分鐘，減輕局部灼燒感
4. **塗抹藥油**：冷敷後塗藥油，薄荷腦涼感可進一步舒緩刺痛與搔癢
5. **保護水泡**：若出現水泡，**切勿刺破**，以免感染；可輕薄塗藥油於水泡周邊

> **注意**：對螞蟻毒素過敏者可能出現嚴重反應，詳見「何時需要就醫」章節。

### 3.3 蜜蜂／黃蜂螫傷

**特徵**：蜜蜂螫後會留下毒刺（蜜蜂特有，黃蜂不留刺），傷口劇烈疼痛，快速紅腫。蜜蜂毒液偏酸性，黃蜂毒液偏鹼性。

**處理步驟（蜜蜂）**：
1. **首要步驟——拔除毒刺**：這是最關鍵的一步。蜜蜂毒囊留在皮膚上會持續注入毒素。**正確做法是用信用卡或指甲側邊橫向刮除**，切勿用手指捏住（會擠壓毒囊，加劇毒素注入）
2. **冷水沖洗＋冷敷**：降低局部溫度，減緩毒素擴散，舒緩疼痛
3. **弱鹼性中和**（蜜蜂毒為酸性）：可用小蘇打水（碳酸氫鈉溶液）輕敷傷口
4. **塗抹藥油**：沖洗後塗藥油，止癢消腫
5. **觀察30分鐘**：密切留意有無全身性過敏反應（參見就醫指徵）

**處理步驟（黃蜂）**：
1. **無毒刺可拔**，可直接冷水沖洗
2. **弱酸性中和**（黃蜂毒為鹼性）：可用稀釋食醋輕敷
3. 後續步驟同蜜蜂處理

### 3.4 蜈蚣咬傷

**特徵**：蜈蚣以毒爪咬人，傷口呈兩點狀，毒液含蜂毒肽和螞蟻酸，會引起劇烈疼痛、局部紅腫，部分人可出現發熱、淋巴結腫大等全身反應。

**處理步驟**：
1. **立即用肥皂水徹底沖洗傷口**：蜈蚣毒液偏酸性，肥皂的弱鹼性有助中和
2. **冷敷**：冰袋隔布冷敷，減輕劇痛（冷敷比藥油更重要，應先做）
3. **塗抹藥油**：冷敷後可塗藥油以減輕搔癢和輕度疼痛
4. **不要擠壓傷口**，以免毒液擴散
5. **密切觀察**：蜈蚣咬傷的全身反應風險較高，若出現發熱、嘔吐、心跳加速，應立即就醫

> **重要提示**：大蜈蚣（體長超過6厘米）咬傷往往較重，建議傷後直接就醫，不可只靠藥油處理。

### 3.5 跳蚤咬傷

**特徵**：跳蚤咬傷多出現在腳踝、小腿等下肢位置，常成排或成群分佈，皮膚出現小紅點，中心有刺點，搔癢可持續較久（可達數日）。

**處理步驟**：
1. **徹底清潔皮膚**：用肥皂及溫水清洗咬傷區域
2. **冷敷**：減輕初期腫脹及刺痛
3. **規律塗抹藥油**：因跳蚤咬傷的搔癢可維持較長時間，每天早晚各塗藥油一次，並在搔癢劇烈時隨時補塗
4. **處理感染源**：同步為寵物驅蟲、清潔地毯/床具，減少反覆咬傷

## 四、推薦藥油產品與主要成分參考

市面上藥油種類繁多，以下是常見的幾類及其適用情境：

### 4.1 清涼油（萬金油／白花油類）

**代表成分**：薄荷腦、樟腦、桉葉油、丁香油、肉桂油  
**濃度特點**：成分濃度較高，止癢效果較強  
**適用場景**：成人蚊蟲咬傷、螞蟻咬、輕微蜂螫（輔助處理）  
**注意事項**：氣味較濃烈，皮膚敏感者慎用；3歲以下兒童不建議使用

### 4.2 風油精

**代表成分**：薄荷腦、水楊酸甲酯、樟腦、桉葉油、丁香薄荷油  
**濃度特點**：液態，滲透性好，適合點狀塗抹局部  
**適用場景**：蚊蟲咬傷快速止癢，也可用於暈車、頭痛等  
**注意事項**：含水楊酸甲酯，不建議用於皮膚破損處；3歲以下兒童不建議使用

### 4.3 兒童專用驅蚊止癢產品

**代表成分**：薰衣草精油、金盞花萃取物、低濃度檸檬桉葉油（PMD）或低濃度薄荷腦（≤1%）  
**濃度特點**：活性成分濃度明顯低於成人款，配方更溫和  
**適用場景**：2歲以上兒童輕度蚊蟲咬傷  
**注意事項**：2歲以下不可使用（見下節兒童安全篇）

### 4.4 選購建議

- 優先選擇有標明**國藥准字**或**衛妝准字**的正規產品
- 查閱產品說明書，確認薄荷腦和樟腦含量（成人用品一般薄荷腦5%–20%，樟腦3%–10%）
- 首次使用前，建議於前臂內側小面積試用，觀察15分鐘有無過敏反應

## 五、兒童使用安全須知

兒童皮膚較薄，體表面積與體重比例高、神經系統發育未完善，對薄荷腦和樟腦的耐受性遠低於成人，使用時必須格外謹慎。

### 5.1 核心禁忌：2歲以下禁用含薄荷腦和樟腦產品

**2歲以下嬰幼兒嚴禁使用含薄荷腦（menthol）或樟腦（camphor）的任何外用產品。**

原因如下：

- **薄荷腦**：在嬰幼兒中可能誘發**喉頭痙攣**，導致呼吸困難，具致死風險。即使塗抹於皮膚，由於嬰兒皮膚滲透性高，薄荷腦仍可能經皮吸收後進入呼吸道
- **樟腦**：嬰幼兒對樟腦代謝能力極弱，經皮吸收後可引致**神經毒性反應**，包括抽搐、驚厥，嚴重時危及生命

> **特別警告**：不要將含薄荷腦／樟腦的藥油塗抹於嬰兒臉部、鼻孔附近、胸口或頸部。

### 5.2 2–12歲兒童的使用原則

- 使用**兒童專用產品**，或選擇活性成分濃度不超過1%的溫和配方  
- **塗抹量宜少**：一顆豌豆大小即可覆蓋一個咬點  
- **避開面部（尤其眼周）及手部**（兒童易將手放入口中）  
- 每日塗抹次數不超過3次  
- 使用後觀察兒童反應，若出現皮疹加重、持續哭鬧、呼吸異常，應立即擦除並就醫

### 5.3 孕婦與哺乳期婦女

孕期頭三個月建議避免大面積使用含樟腦的藥油（樟腦可能有致畸風險，雖然外用吸收量有限）。哺乳期婦女應避免將藥油塗於胸部區域。使用前建議諮詢醫生。

## 六、蚊蟲咬傷預防措施

止癢只治標，防範才是治本。以下措施可有效減少咬傷機會：

### 6.1 物理防護

- **著長袖長褲**：於黃昏蚊蟲活躍時段（黃昏前後約2小時）外出時，淺色衣物更佳（深色吸熱，蚊子較易停留）  
- **使用蚊帳**：夜間睡眠時用蚊帳是最安全、最無副作用的防護方式，特別適合嬰幼兒  
- **裝紗窗**：保持家中窗門紗網完好

### 6.2 清除孳生地

- 定期清理居家周圍積水（花盆托盤、水桶、廢舊輪胎等）
- 積水容器至少每週換水一次，打斷蚊子幼蟲生命週期

### 6.3 驅蟲產品

- **成人**：可使用含 DEET（避蚊胺，6%–30%濃度適合一般戶外活動）或派卡瑞丁（picaridin）的驅蚊產品，效能約可持續4–6小時  
- **2歲以上兒童**：可使用含低濃度 DEET（不超過10%）或派卡瑞丁的兒童驅蚊產品  
- **2歲以下嬰兒**：不建議使用化學驅蚊劑，以物理防護為主

## 七、何時需要就醫

大多數蚊蟲咬傷可透過家庭處理痊癒，但以下情況必須立即就醫：

### 7.1 嚴重過敏反應（過敏性休克）——立即撥打120

若被咬／螫傷後出現以下任一症狀，可能是**過敏性休克（anaphylaxis）**，必須立即撥打急救電話：

- 全身蕁麻疹或皮膚大面積潮紅
- 面部、唇、舌頭或喉嚨腫脹
- 呼吸困難、喘鳴、胸悶
- 頭暈、血壓急降、意識模糊或暈厥
- 劇烈腹痛、嘔吐、腹瀉

### 7.2 局部嚴重反應

以下情況建議當天或次日就醫：

- **大面積紅腫**：紅腫範圍超過10厘米，或持續擴大超過48小時
- **咬傷處出現膿液**：提示繼發細菌感染，可能需要使用抗生素
- **持續高熱**：體溫超過38.5°C，需排除其他病因，考慮感染或蟲媒傳播疾病
- **淋巴結腫大**：咬傷附近淋巴結明顯腫大、疼痛
- **咬傷性皮膚炎**：多處咬傷引起泛發性皮疹，已超出單純止癢藥油的處理範圍

### 7.3 特殊情況

- 被蜈蚣（尤其大型蜈蚣）或蠍子螫傷
- 被蜱蟲咬傷（蜱蟲須完整使用鑷子拔除，不可自行暴力扯；若咬傷位於華南／華東等萊姆病流行區，須到醫院取蜱並觀察）
- 兒童被蜜蜂群螫（多處螫傷），或老人、心臟病患者單次蜜蜂螫傷
- 咬傷發生在眼周、口腔黏膜等特殊部位

## 重點總結

| 要點 | 關鍵內容 |
|------|---------|
| **止癢原理** | 薄荷腦活化 TRPM8 受體產生涼感，競爭性抑制癢覺訊號傳導 |
| **消腫原理** | 樟腦活化 TRPV3 受體，促進局部血液循環，協助炎症消散 |
| **蜜蜂螫傷** | 先拔毒刺（側向刮除），再冷敷、塗藥油 |
| **兒童安全** | 2歲以下嚴禁使用含薄荷腦和樟腦產品；2歲以上使用兒童專用低濃度產品 |
