## 為什麼華人家庭都備一支藥油？

藥油（活絡油 / 風油 / 萬金油 / 跌打油 / 藥膏）是華人世界 1000+ 年的家庭常備品。從清末民初開始，數十個經典品牌歷經幾代人傳承，覆蓋跌打、感冒、暈車、頭痛、關節痠痛、蚊蟲叮咬等日常場景。

但全網沒有一個地方做到：
- **客觀對比**：每個品牌的官方頁都吹自己最好
- **成份透明**：實際藥用濃度、輔料、安全劑量極少公開
- **跨地區法規**：香港 / 台灣 / 大陸 / FDA / EMA 對同一成份的判斷可能完全不同
- **安全警示**：嬰幼兒 / 孕婦 / 蠶豆症（G6PD 缺乏）患者的禁忌

本文根據中國藥典（2020 版）、USP-NF、香港中醫藥管理委員會、台灣衛福部食藥署、新加坡 HSA、馬來西亞 NPRA 公開資料，對 8 大經典華人藥油做橫向對比。

## 成份背後的科學

### 1. 樟腦（Camphor）

- **作用**：激活 TRPV1 / TRPM8 受體，產生 cooling-then-warming 感覺，輕度局部麻醉
- **安全劑量**：成人外用 ≤ 11%（FDA OTC monograph 上限）
- **🚨 嬰幼兒警示**：FDA 明令禁止 2 歲以下使用樟腦製品。誤食 1 茶匙（約 1g）可致幼兒抽搐
- **孕婦**：避免大面積使用，可經皮吸收
- **參考**：FDA 21 CFR 341.14；American Academy of Pediatrics

### 2. 薄荷腦 / 薄荷醇（Menthol）

- **作用**：激活 TRPM8 冷感受體，分散痛感（gate control theory），輕度局部麻醉
- **安全劑量**：外用 ≤ 16%（FDA OTC）
- **🚨 嬰幼兒警示**：< 2 歲禁用於鼻部 / 胸口（曾有 laryngospasm 案例）
- **參考**：Eccles R., J Pharm Pharmacol 1994

### 3. 水楊酸甲酯 / 冬青油（Methyl Salicylate）

- **作用**：經皮吸收後轉化為 salicylate，具抗炎止痛效果
- **安全劑量**：外用 ≤ 30%（FDA），但**小面積使用**
- **🚨 高風險**：1 茶匙純冬青油 ≈ 21 顆 325mg 阿斯匹靈。曾有運動員因大面積塗抹後熱敷致死（2007 NYC 案例）
- **🚨 蠶豆症（G6PD 缺乏）**：避免使用 — 水楊酸類可誘發溶血
- **🚨 兒童**：< 12 歲不建議
- **參考**：Chan TY., Hum Exp Toxicol 1996

### 4. 桉葉油 / 尤加利油（Eucalyptus Oil）

- **作用**：1,8-cineole 主成份，輕度抗炎、祛痰、抗菌
- **安全劑量**：外用稀釋 ≤ 10%
- **🚨 嬰幼兒警示**：2 歲以下禁用於面部 / 鼻部
- **參考**：Australian TGA monograph

### 5. 松節油（Turpentine Oil）

- **作用**：α-pinene 為主，反刺激（counter-irritant），促進局部血流
- **🚨 純松節油**口服極毒，外用亦需稀釋至 ≤ 5%
- **參考**：USP-NF Turpentine Oil monograph

## 真假辨識：5 個共通信號

1. **批次號 + 條形碼**：所有正貨都有，可上品牌官網或當地藥監局查詢
2. **氣味層次**：正貨的精油有複雜層次感；假貨多為單一刺鼻香精味
3. **油液澄清度**：正貨多為澄清淡黃 / 琥珀色；假貨可能渾濁或顏色過深
4. **包裝印刷**：正貨字體銳利，封口完整；假貨常見模糊、錯字、封口鬆
5. **價格**：明顯低於市場均價的 50% 大概率是假貨

各品牌的具體真偽辨識細節，將在獨立的品牌頁陸續上線。

## 安全警示（必讀）

⚠️ **以下人群使用任何藥油前必須諮詢醫師**：

- **2 歲以下嬰幼兒** — 樟腦 / 薄荷腦神經毒性風險
- **孕婦** — 多種精油可經皮吸收，部分有催產 / 致畸風險
- **G6PD 缺乏（蠶豆症）患者** — 樟腦、薄荷醇、水楊酸類均需避免
- **皮膚破損 / 大面積燒傷** — 經皮吸收量大幅升高
- **與熱敷 / 電熱毯併用** — 可顯著增加水楊酸吸收，曾有致死案例
- **眼周、黏膜、生殖器附近** — 任何含樟腦 / 辣椒素 / 薄荷腦製品都禁用
