# 清涼油（清涼萬金油）介紹

## 什麼是清涼油

清涼油，學名「清涼萬金油」，是中國大陸家庭中最為普及的外用消炎止痛藥物之一。其綠色鐵皮小圓罐的包裝已成為幾代中國人共同的童年記憶。不論是夏天被蚊叮，還是出門感到頭暈腦脹，很多家庭的第一反應就是拿出清涼油來搽一搽。

清涼油的歷史可以追溯到20世紀初。最初是在東南亞華人萬金油配方基礎上，結合中國本土生產條件進行改良，逐步成為中國大陸本地化的藥油產品。1949年後，清涼油由國營藥廠批量生產，以極低的成本廣泛進入城鄉家庭，成為計劃經濟年代最具代表性的「國民藥品」之一。

## 核心成分

清涼油的配方在不同品牌及生產廠商之間存在輕微差異，但核心成分基本一致：

- **樟腦（Camphor）**：通常佔比8%–15%，提供溫熱與穿透感，是清涼油發揮鎮痛效果的主要成分之一。
- **薄荷腦（Menthol）**：通常佔比10%–15%，產生顯著的清涼感，並有輕度局部止痛及止痕效果。
- **桉葉油（Eucalyptus Oil）**：含有桉葉素（1,8-cineole），具有一定抗菌與消炎作用，同時為清涼油帶來清新的草藥氣味，有助於舒緩鼻塞。

輔料通常包括凡士林（白凡士林）作為基質，使產品保持軟膏狀態並易於塗抹。

部分配方還添加有微量松節油（turpentine oil）或薄荷油，進一步增強揮發性氣味的強度。

## 主要用途

清涼油在中國家庭中的使用範圍極廣：

**蚊叮蟲咬與皮膚痕癢**
這是清涼油最普遍的用途。薄荷腦的清涼刺激感可迅速壓制痕癢感，塗抹後數秒內即有效果。適用於蚊子、螞蟻、蜂類等輕微叮咬，不適用於嚴重過敏反應。

**頭痛與頭暈**
將少量清涼油塗於太陽穴（兩側）、印堂（眉間），輕揉按摩，揮發的薄荷腦與桉葉油氣味有助於舒緩緊張性頭痛及輕微偏頭痛。

**鼻塞與傷風感冒**
塗於人中穴或鼻翼兩側，深呼吸吸入揮發氣體，有助通鼻、提神，是感冒初期常見的輔助舒緩方法。

**暈車暈船**
乘車前將清涼油塗於肚臍或太陽穴，有一定預防暈動症的效果，民間使用歷史悠久。

**輕微肌肉疲勞**
塗抹於頸部、肩背等疲勞部位，短暫舒緩肌肉緊張感，但對深層肌肉酸痛的效果較有限。

## 清涼油 vs 虎標萬金油：有什麼不同？

兩者同源於華人傳統藥油文化，但在以下幾個維度存在明顯區別：

| 對比維度 | 清涼油 | 虎標萬金油（白色） |
|---|---|---|
| 產地 | 中國大陸（多個國營/民營藥廠） | 新加坡（恒生製藥/虎標集團） |
| 配方重點 | 清涼為主（薄荷腦、桉葉油） | 清涼+輕微溫熱（薄荷腦+樟腦） |
| 氣味風格 | 清新、偏桉葉草藥味 | 較辛辣、複合草藥香 |
| 包裝 | 綠色/白色鐵皮圓罐，極簡設計 | 白色/紅色圓罐，老虎商標，設計較精緻 |
| 價格 | 極低（人民幣幾元至十餘元） | 較高（HKD/SGD十餘至數十元） |
| 國際知名度 | 以中國大陸為主，海外華人社區有流通 | 全球100+國家正式銷售 |
| 藥用濃度 | 通常標註8–15%樟腦、10–15%薄荷腦 | 標準白色配方含25%樟腦、10%薄荷腦（紅色版） |

整體而言，清涼油的清涼感較突出，偏向於日常止痕與提神；虎標萬金油的配方更為複雜，兼顧清涼與溫熱雙重效果，對肌肉疼痛的針對性更強。

## 購買與價格

清涼油是中國大陸藥房與超市的常備品，幾乎所有連鎖藥房（如大參林、老百姓大藥房、益豐藥業）及各大商超均有售。常見品牌包括「風油精」（雖配方略有不同，但功效相近）、「白鳳凰清涼油」、「順氣清涼油」等。

價格極為親民，一罐標準規格（約3–10克）售價通常在人民幣2至15元之間，是國內最具性價比的家用急救藥品之一。

## 注意事項

- **2歲以下嬰兒禁用**：樟腦對嬰幼兒有神經毒性，不可使用於嬰兒皮膚或衣物。
- **避開眼睛、口腔及黏膜**：清涼油刺激性較強，誤入眼睛需立即用清水沖洗。
- **孕婦慎用**：部分成分可能對孕婦產生影響，建議諮詢醫生後使用。
- **皮膚破損處禁用**：有傷口、濕疹、皮炎等皮膚問題時不應塗抹。
- **不可內服**：誤食須立即求醫。
