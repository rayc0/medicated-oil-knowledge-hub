## 一、「紅花油」與「正紅花油」：名稱相近，配方迥異

市面上「紅花油」一詞涵蓋兩類截然不同的產品，購買前務必分清。

### 1. 普通紅花油

普通紅花油的主要成份為**丁香羅勒油、水楊酸甲酯、薑樟油、肉桂油、桂皮醛、檸檬醛、冰片**。其藥理以涼血活血、祛風濕為主，屬中華人民共和國衛生部藥品標準收載品種，主治風濕骨痛、肢體麻木、跌打損傷及蚊蟲叮咬。[^1]

### 2. 正紅花油

正紅花油的配方以**水楊酸甲酯、桂葉油、桂油、香茅油、松節油、辣椒油、血竭**為核心。相比普通紅花油，正紅花油含有**血竭**（Dragon's Blood Resin）這一名貴中藥材，止痛效力更強，更側重急救消炎、涼血止血。正紅花油原為進口藥品標準，後於內地註冊為國家標準。[^2]

### 速查對比

| 項目 | 普通紅花油 | 正紅花油 |
|------|----------|---------|
| 代表成份 | 丁香羅勒油、冰片 | 血竭、松節油、辣椒油 |
| 共同成份 | 水楊酸甲酯、桂皮醛 | 水楊酸甲酯、桂葉油 |
| 側重功效 | 涼血活血、日常舒緩 | 消炎止痛、急救止血 |
| 藥品標準 | 衛生部藥品標準 | 國家標準（原進口） |
| 代表品牌 | 五環牌、嶺南萬應 | 依馬打、永龍、獅馬龍 |

> **購買提示**：在香港藥房，「正紅花油」通常指含血竭配方的高規格版本。若店員問你要「紅花油定正紅花油」，記住帶「正」字的才是含血竭版。

## 二、品牌歷史：依馬打與正紅花油的六十年

### 依馬打（IMADA）

依馬打的經營始於 **1950 年代**，迄今逾七十年歷史。品牌早年的產品行銷南洋群島及東南亞礦場，服務海外華工社群。依馬打堅持選用純正地道名貴藥材，精提煉製，逐步建立起「香港製造」的正紅花油代名詞地位。[^3]

依馬打旗下產品線涵蓋正紅花油、四季平安油、活絡油、金龍驅風油等系列藥油，但正紅花油始終是其招牌產品。品牌現由**金活醫藥集團**（Kingworld Medicines Group）負責中國內地的分銷。

## 三、成份藥理深度解析

以依馬打正紅花油為例，每 1000 g 藥液含以下成份（根據藥品說明書）[^3]：

| 成份 | 含量（每 1000 g） | 佔比 |
|------|------------------|------|
| 水楊酸甲酯 | 713 g | 71.3% |
| 松節油 | 174 g | 17.4% |
| 桂油 | 47 g | 4.7% |
| 桂葉油 | 38 g | 3.8% |
| 香茅油 | 16 g | 1.6% |
| 辣椒油 | 9 g | 0.9% |
| 血竭 | 3 g | 0.3% |

### 3.1 水楊酸甲酯（Methyl Salicylate）— 71.3%

水楊酸甲酯是正紅花油的**絕對主成份**，佔配方逾七成。

**藥理機制**：外用後經皮膚滲透，在體內代謝為水楊酸（Salicylic Acid）。其止痛作用主要通過以下途徑：

- **反刺激效應（Counter-irritation）**：活化並脫敏皮膚感覺神經，透過「以刺激蓋過疼痛」的原理減低深層組織痛感，這是外用水楊酸酯類的主要止痛機制。[^4]
- **COX 抑制**：局部抑制環氧合酶（Cyclooxygenase），減少前列腺素合成，但臨床前研究顯示外用水楊酸酯的 COX 抑制力約僅為口服阿司匹林的 1/100。[^4]
- **血管擴張**：促進局部血液循環，加速代謝廢物清除。

**安全警示**：水楊酸甲酯毒性不容低估。1 ml 純品約含 1.4 g 水楊酸當量。大面積塗抹或誤服可導致水楊酸中毒（耳鳴、噁心、酸鹼失衡）。文獻記載有因過量使用含水楊酸甲酯產品致死的案例。[^5]

### 3.2 松節油（Turpentine Oil）— 17.4%

松節油為第二大成份，其主要活性成份為 α-蒎烯（α-Pinene）和 β-蒎烯。

- **反刺激與溫熱感**：塗抹後產生明顯的溫熱刺激，促進局部血流
- **滲透增強劑**：協助其他活性成份穿透角質層，提高藥效
- **傳統功用**：中醫以松節油「祛風除濕、活血止痛」，常見於跌打外用方

### 3.3 桂油與桂葉油（Cassia / Cinnamon Leaf Oil）— 合計 8.5%

兩者均富含**桂皮醛（Cinnamaldehyde）**，為肉桂的標誌性芳香成份。

**TRPA1 受體活化**：桂皮醛是 TRPA1（瞬時受體電位通道 A1）的天然激動劑。TRPA1 表達於感覺神經元，介導痛覺信號傳導。桂皮醛活化 TRPA1 後引發以下反應[^6]：

- **神經源性血管擴張**：透過一氧化氮合酶（NOS）途徑擴張皮膚微血管
- **CGRP 釋放**：刺激三叉神經元分泌降鈣素基因相關肽（CGRP），產生局部神經源性炎症反應
- **溫熱感**：塗抹後的「發熱感」部分來自桂皮醛的 TRPA1 活化效應

桂油同時具備抗菌活性，對金黃色葡萄球菌等常見皮膚病原菌有抑制作用。

### 3.4 香茅油（Citronella Oil）— 1.6%

香茅油富含香茅醛（Citronellal）和香葉醇（Geraniol），在正紅花油中主要起：

- **驅蚊蟲**：香茅醛是天然驅蚊成份
- **芳香矯味**：平衡水楊酸甲酯的刺鼻氣味
- **輕微抗炎**：動物實驗顯示有一定消炎活性

### 3.5 辣椒油（Capsicum Oil）— 0.9%

辣椒油含**辣椒素（Capsaicin）**，為 TRPV1 受體激動劑。

- **P 物質耗竭**：長期局部應用可耗竭感覺神經末梢的 P 物質（Substance P），從而降低痛覺信號傳導
- **深層溫熱感**：與松節油、桂皮醛協同，增強正紅花油的「辛熱」體感
- **含量極低**：0.9% 的辣椒油含量屬溫和水平，不會造成皮膚灼傷，但敏感膚質仍需留意

### 3.6 血竭（Dragon's Blood Resin）— 0.3%

血竭是正紅花油區別於普通紅花油的**靈魂成份**，來源於龍血樹屬（*Dracaena* spp.）植物的樹脂。

**藥理研究**[^7]：

- **抗炎**：抑制促炎細胞因子 TNF-α、IL-1β 及 COX-2 的產生；動物實驗顯示可顯著抑制巴豆油誘導的小鼠耳廓炎症
- **促傷口癒合**：20% 濃度外用可加速兔燒傷模型的組織修復，促進再上皮化
- **抗菌**：對多種常見病原菌有抑菌活性
- **抗氧化**：含豐富的黃酮類、原花青素、兒茶素等多酚化合物，清除自由基能力強

血竭在配方中含量雖僅 0.3%，但其活血化瘀、消腫止痛的傳統功效是正紅花油「正」字的核心所在。

## 四、主治症狀與適用場景

根據藥品說明書，正紅花油主治[^1][^2]：

- **風濕骨痛**：關節酸痛、天氣變化時加重的慢性痠痛
- **肢體麻木**：氣血不暢所致的手腳麻痹感
- **跌打損傷**：扭傷、撞傷、挫傷（須遵守 48 小時法則，見下文）
- **蚊蟲叮咬**：蚊叮蟲咬後的紅腫痕癢

### 使用方法

1. 取適量正紅花油（3–5 滴）滴於患處
2. 以指腹輕柔按揉 2–3 分鐘，至皮膚微微發熱
3. 每日 4–6 次（藥品說明書建議用量）
4. 可配合穴位按摩使用，如太陽穴（頭痛）、合谷穴（牙痛）
5. 使用後徹底洗手，避免觸碰眼睛及黏膜

## 五、急性扭傷 48 小時法則（極重要）

這是使用正紅花油最常見的誤區，亦是最危險的錯誤用法。

### 為何扭傷後不能立即用紅花油？

關節扭傷的急性期（受傷後 0–48 小時），軟組織破裂，微血管尚未完全閉合，局部仍有出血及滲出傾向。此時軟組織腫脹明顯、疼痛劇烈。[^8]

正紅花油的核心功效正是**活血化瘀、促進局部血液循環**。在急性出血期使用活血藥油，等於加速血液滲出，結果只會：

- **加重腫脹**：局部滲出液增加
- **延長恢復期**：出血範圍擴大
- **加劇疼痛**：腫脹壓迫神經

### 正確處理流程

| 時段 | 正確做法 | 禁忌 |
|------|---------|------|
| **0–48 小時**（急性期） | RICE 原則：休息（Rest）、冰敷（Ice）、加壓包紮（Compression）、抬高患肢（Elevation） | 禁用紅花油、禁止熱敷、禁止按揉 |
| **48–72 小時**（過渡期） | 觀察紅腫熱痛是否消退；若仍有明顯腫脹，繼續冰敷 | 視情況而定 |
| **72 小時後**（恢復期） | 開始熱敷 + 配合正紅花油按揉，活血化瘀、散腫止痛 | 若仍有明顯瘀青及出血傾向，延後使用 |

> **記住**：「先冷後熱，先止後通」—— 先用冰敷止血消腫，待急性期過後再用紅花油活血通絡。

## 六、安全禁忌與注意事項

### 絕對禁忌

| 人群 / 情況 | 原因 |
|------------|------|
| **孕婦** | 水楊酸甲酯、桂皮醛、血竭均有活血化瘀作用，可能增加流產風險 |
| **皮膚破損處** | 直接接觸傷口可導致劇烈刺痛及組織損傷 |
| **黏膜部位** | 禁止接觸眼睛、口腔、鼻腔內部 |
| **內服** | 正紅花油嚴禁口服，水楊酸甲酯誤服可致中毒 |

### 慎用人群

| 人群 | 注意事項 |
|------|---------|
| **兒童** | 必須在成人監護下使用；患感冒、水痘或發熱的兒童避免使用含水楊酸甲酯產品（與雷氏綜合症 Reye's Syndrome 風險相關） |
| **水楊酸過敏者** | 對阿司匹林過敏者禁用 |
| **出血傾向者** | 水楊酸抑制血小板聚集，有出血風險 |
| **經期婦女** | 活血成份可能延長經期或增加經血量 |
| **哺乳期婦女** | 慎用，成份可能經皮膚吸收進入乳汁 |
| **老年人** | 建議在醫師或藥劑師指導下使用 |
| **G6PD 缺乏症（蠶豆症）** | 部分成份可能誘發溶血，須諮詢醫師 |

### 飲食注意

使用正紅花油期間，中醫建議避免進食**生冷及油膩食物**，以免影響活血化瘀的藥效。

### 過敏反應

首次使用建議先在前臂內側小範圍試用，觀察 15–30 分鐘。若出現紅疹、瘙癢、水泡等過敏反應，應立即停用並以清水沖洗。

## 九、保存方法

- **避光密封**：存放於陰涼乾燥處，避免陽光直射
- **遠離火源**：水楊酸甲酯及松節油均為易燃液體
- **兒童觸及不到之處**：防止兒童誤服

## 十、常見問題

### Q1：正紅花油可以治療濕疹嗎？

不建議。正紅花油含高濃度水楊酸甲酯及辣椒油，對濕疹等炎症性皮膚病可能造成嚴重刺激，加重病情。

### Q2：正紅花油可以內服嗎？

**絕對不可以。** 正紅花油嚴禁口服。5 ml 水楊酸甲酯即可對幼兒產生致命毒性。即使成人誤服少量亦可引起噁心、嘔吐、耳鳴等水楊酸中毒症狀。如不慎誤服，應立即送院急救。

### Q3：紅花油和雲南白藥哪個治扭傷更好？

兩者定位不同。雲南白藥（噴霧劑）含三七等成份，可在急性期使用，有止血消腫功效。正紅花油則適合在急性期過後（48 小時以上）使用，以活血化瘀、促進恢復。兩者可先後配合使用：急性期用雲南白藥，恢復期用正紅花油。

### Q4：孕婦連聞紅花油的氣味都不行嗎？

短暫聞到氣味一般不會造成危害，但不建議長時間吸入。孕婦應避免使用正紅花油塗抹皮膚，因水楊酸甲酯可經皮膚吸收進入血液循環。
