# 藥油品牌大全

藥油品牌繁多，產地橫跨香港、新加坡、泰國、中國內地與日本，各有歷史傳承與配方特色。本頁彙整主要品牌，按產地區域分類，幫助你快速了解各品牌定位，找到最適合自己需求的選擇。

---

## 香港傳統品牌

香港藥油歷史悠久，以出口聞名海外，配方融合中西成分，注重多功能性。

- **[白花油完整品牌指南](white-flower-embrocation-complete-guide.md)** — 香港最具代表性的藥油，創立逾百年，以薄荷腦、水楊酸甲酯與桉油為核心，主打提神、止痛、通鼻塞三合一。
- **[正紅花油完整指南](red-flower-oil-complete-guide.md)** — 專為跌打瘀傷設計，以紅花提取成分為主要活性成分，活血散瘀效果突出。
- **[保心安油完整指南](po-sum-on-complete-guide.md)** — 香港老字號出品，以丁香與薄荷為主，提神驅風效果突出，廣受長者信賴。
- **[黃道益活絡油完整指南](wong-to-yick-complete-guide.md)** — 香港本地著名跌打油，由黃道益創辦，中醫活絡配方，專注筋骨關節痛症。
- **[萬華油完整指南](wan-hua-oil-complete-guide.md)** — 香港本地生產，廣泛用於肌肉痠痛與跌打損傷的多用途藥油。
- **[萬應止痛油](wan-ying-zhi-tong-gao/)** — 多功能外用止痛膏油，香港家庭常備。
- **[雙獅行軍散完整指南](double-prince-complete-guide.md)** — 香港老字號，主打提神醒腦與舒緩頭痛的經典配方。
- **[鷹標驅風油完整指南](eagle-brand-medicated-oil-complete-guide.md)** — 歷史逾百年，與虎標齊名的傳統驅風油，成分以薄荷腦與桉油為主。
- **[余仁生如意油完整指南](yu-yee-oil-cap-limau-complete-guide.md)** — 主打嬰幼兒及兒童適用的溫和驅風油，以柑橘為基底。

## 中國內地品牌

內地藥油以中成藥標準生產，配方多含傳統中草藥提取物，價格親民，供應量大。

- **[風油精完整指南](fengyoujing-complete-guide.md)** — 內地最普及的日用藥油，以薄荷腦與水楊酸甲酯為主，清涼止癢。
- **[雲南白藥氣霧劑完整指南](yunnan-baiyao-aerosol-spray-complete-guide.md)** — 雲南白藥旗下外用噴劑，適用於急性扭傷與跌打止血。
- **[玉林正骨水完整指南](yulin-zheng-gu-shui-complete-guide.md)** — 廣西玉林出品的跌打正骨水，以其獨特草本配方聞名，運動員常用。
- **[三九系列](sanjiu/)** — 現代中成藥企業旗下外用藥油產品線。
- **[廣東百花油](guangdong-baihua/)** — 源自廣東傳統配方，以多種花卉與草本精華調配。
- **[如意珍寶](ruyi-zhenbao/)** — 中藥複方外用油，專注於活血化瘀與消腫止痛。
- **[白花油（內地版）](baihuayou/)** — 內地市場流通版本，配方與港版略有差異。
- **[正骨貼](zheng-gu-tie/)** — 以正骨活絡為主的外貼劑型，常見於骨科輔助用途。
- **[玉樹](yushu/)** — 專注於皮膚護理的中草藥外用油。
- **[虎力油](hubiaowan-jin-you/)** — 內地版配方驅風藥油。
- **[複方樟腦](compound-camphor/)** — 以樟腦為主成分的複方外用製劑。
- **[祛風油](qufengyou/)** — 傳統中藥祛風止痛外用油。
- **[清涼油](qingliangyu/)** — 中國內地家傳戶曉的薄荷基清涼止痛膏。
- **[活絡油](huo-luo-you/)** — 以活絡通經為主治的複方外用油。
- **[麝香追風](shexiang-zhuifeng/)** — 含麝香成分的高端追風止痛外用製劑。

## 東南亞品牌

新加坡與泰國的藥油多由南洋華人移民發展，配方簡潔，在東南亞及全球華人社區廣泛使用。

- **[虎標萬金油完整指南](tiger-balm-complete-guide.md)** — 源自緬甸、發揚於新加坡的國際經典，紅白兩款各有功效定位，全球最知名的藥油品牌。
- **[斧標驅風油完整指南](axe-brand-complete-guide.md)** — 新加坡創立、香港普及的經典綠色驅風油，以薄荷腦與桉油為核心，尤其適合腸胃不適與頭痛。
- **[均隆驅風油完整指南](kwan-loong-oil-complete-guide.md)** — 東南亞華人家庭常備，驅風提神效果持久。
- **[上標油完整指南](siang-pure-oil-complete-guide.md)** — 泰國出產的經典驅風油，主打頭痛與肌肉不適，在東南亞及香港廣受歡迎。
- **[安美露完整指南](ammeltz-yoko-yoko-complete-guide.md)** — 日本安摩力 [待核實] 品牌，凝膠劑型便於局部塗抹，在東南亞市場廣受歡迎，針對肩頸局部應用設計。

## 如何在品牌之間選擇

不同品牌設計用途有別，選擇時可參考以下維度：

**按主要需求：**
- 頭痛提神 → 白花油、保心安油、清涼油
- 跌打扭傷 → 正紅花油、黃道益活絡油、萬華油
- 肌肉關節痛 → 撒隆巴斯、虎標萬金油（白色）、玉林正骨水
- 嬰幼兒 → 余仁生如意油（需確認年齡適用）
- 驅風肚痛 → 斧標、均隆、保心安油
- 暈車浪、暈船浪 → 白花油、上標油、風油精

**按成分敏感度：**
- 蠶豆症（G6PD缺乏症）患者 → 避免含樟腦品牌，詳見[安全使用指南](/zh-hant-hk/safety/)
- 孕婦 → 大多數藥油不建議使用，請諮詢醫生
- 嬰幼兒（2歲以下）→ 幾乎所有成人藥油均禁用
