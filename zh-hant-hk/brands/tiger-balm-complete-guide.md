# 虎標萬金油完整指南：成分、歷史、辨別真假、安全使用（2026年版）

## 摘要

虎標萬金油（Tiger Balm）由緬甸華僑藥師胡子欽於1870年代研創，其子胡文虎、胡文豹兄弟於1918年正式命名並推向東南亞市場。核心活性成分為樟腦（Camphor）與薄荷腦（Menthol），輔以丁香油、桉葉油及肉桂油。FDA規定外用樟腦濃度上限為11%，2歲以下嬰幼兒禁用。G6PD（蠶豆症）患者亦應避免使用。

## 一、從緬甸仰光到香港：虎標萬金油150年史

### 1.1 永安堂：一切的起點

1870年代，福建永定客家人**胡子欽**（Aw Chu Kin）只身南渡緬甸仰光謀生，以草藥行醫為業，在仰光開設「**永安堂**」（Eng Aun Tong）藥舖。他以中草藥調配出一種外用止痛膏劑，命名為「萬金油」（Ban Kim Yew），這便是虎標萬金油的前身。

### 1.2 胡文虎、胡文豹兄弟接手

1908年胡子欽病故，遺囑囑託兩個兒子完善配方：

- **胡文虎**（Aw Boon Haw，1883—1954）：名字中「虎」字即品牌由來，英文意為「溫和的老虎」（Gentle Tiger）
- **胡文豹**（Aw Boon Par，?—1944）：英文意為「溫和的豹子」（Gentle Leopard）

胡文虎遊歷中國、日本、暹羅（泰國），考察中西藥經營，延聘藥劑師改良配方。1918年，產品正式更名為「**Tiger Balm（虎標萬金油）**」，以虎為圖騰進行品牌包裝，並憑藉大規模廣告攻勢，迅速在東南亞華人社區打開市場。

### 1.3 香港：品牌的重要根據地

虎標萬金油與香港的淵源，比許多人以為的更深：

| 年份 | 事件 |
|---|---|
| 1923 | 永安堂總行從仰光遷至**新加坡** |
| 1932 | 總行再遷**香港**，胡文虎以香港為主要業務基地 |
| 1935 | 胡文虎在香港大坑道建造「**虎豹別墅**」（Tiger Balm Garden / Haw Par Mansion），集私人別墅、花園及免費公眾樂園於一身，以中國神話故事塑像聞名 |
| 1937—1945 | 二戰期間，胡氏兄弟大部分時間均在香港渡過 |
| 1944 | 胡文豹在香港日佔時期病故 |
| 1954 | 胡文虎在波士頓術後乘機返港途中心臟病發去世，享年72歲，在港留下大量慈善遺產 |
| 1969 | 家族成立 **Haw Par Brothers International Limited**，於馬新交易所上市 |
| 2000年代 | 虎豹別墅由地產發展商重建，原別墅主樓及部分花園結構保留，現為法定古蹟 |
| 2018 | 虎標產品覆蓋全球 **100+** 國家及地區 |
| 2024 | 成為拜仁慕尼黑東南亞區域合作夥伴 |

**虎豹別墅小知識**：香港虎豹別墅原址位於銅鑼灣大坑道，以十八層地獄、西遊記等彩色神話塑像聞名，一度為香港最受歡迎的免費景點之一。舊址現已重建為住宅，但主樓（Haw Par Mansion）已列為香港法定古蹟，開放公眾參觀。

胡文虎同時創辦《星島日報》（Sing Tao Daily）等多家華文報章，被譽為「萬金油大王」與「報業大王」，其商業版圖與慈善事業深深影響了香港及整個東南亞華人社會。

### 1.4 現代公司架構

虎標萬金油現由新加坡上市公司 **Haw Par Corporation Limited**（虎豹集團，SGX：H02）運營，業務涵蓋醫藥保健、休閒及投資。產品於印度海得拉巴 Makson 工廠等多地製造，由授權分銷商分銷至全球各地。

| **肌肉痠痛、關節不適** | 紅色萬金油 / 超強版 | 塗患處，充分按揉3—5分鐘，每日3—4次 |

| **暈車、暈船** | 白色萬金油 | 塗人中穴或太陽穴，或直接開蓋吸聞 |

| **輕度扭傷（48小時後）** | 紅色萬金油 | 先冰敷48小時，再用紅色款配合按摩 |

### 2.3 其他產品線

虎標近年大幅拓展產品線，除經典軟膏外還包括：

- **Tiger Balm Muscle Rub** — 含水楊酸甲酯（Methyl Salicylate）+ 薄荷腦 + 樟腦，乳霜質地，塗抹範圍更廣
- **Tiger Balm Patch（貼布）** — 含樟腦 + 薄荷腦，部分型號含辣椒素（Capsicum），適合背部、肩頸等大面積部位
- **Tiger Balm Neck & Shoulder Rub** — 樟腦 + 薄荷腦乳霜，專為頸肩設計
- **Tiger Balm Pain Relieving Spray** — 水楊酸甲酯 + 薄荷腦 + 樟腦，噴霧劑型，方便難以直接觸及的部位

**注意**：以上延伸產品的配方與安全指引與原裝萬金油罐有所不同，使用前請獨立閱讀各自包裝標籤。

## 三、各活性成分的作用原理

**樟腦（Camphor，11%）**

- 激活皮膚的 TRPV1 及 TRPM8 受體，產生先涼後暖的感覺
- 輕度局部麻醉及止痛作用
- 反刺激作用（counter-irritant）：刺激皮膚表面感覺神經，分散深層痛感
- 輕度抗炎及抗菌活性
- **警告**：樟腦經皮吸收迅速，口服極少量即可引致抽搐，嬰幼兒尤其危險

**薄荷腦（Menthol，8%—11%）**

- 選擇性激活 TRPM8 冷感受體，產生持續清涼感
- 透過閘控理論（gate control theory）干擾痛覺信號傳導
- 在鼻腔可產生通氣感，緩解鼻塞症狀（但不改變實際氣道阻力）
- 輕度局部麻醉

**丁香油（Clove Bud Oil，1.5%—5%）**

- 主成分丁香酚（eugenol）具強效局部麻醉及抗炎活性
- 紅色版含量更高（5%），增強暖感及止痛效果

**白千層油（Cajuput Oil，7%—13%）**

- 主成分1,8-桉葉素（cineole），輕度抗炎、抗菌
- 祛痰、緩解呼吸道不適；白色版含量更高（13%）

**肉桂油（Chinese Cinnamon Oil）**

- 含肉桂醛（cinnamaldehyde），激活 TRPA1 受體，產生溫熱感
- 僅存在於紅色版及超強版中
- 皮膚敏感人士可能引發接觸性皮炎，初次使用建議先在手腕內側小面積測試

## 六、安全警示

> ⚠️ **以下為強制性安全提示，使用虎標萬金油前務必細閱。**

### 6.1 嬰幼兒（2歲以下）— 嚴禁使用

- **樟腦對嬰幼兒具有神經毒性**。FDA明確規定2歲以下兒童禁用含樟腦製品
- 嬰幼兒肝臟代謝功能未成熟，樟腦經皮吸收後易積聚，可致抽搐、呼吸抑制，甚至死亡
- 誤食極少量（約500mg，約半茶匙）即可對嬰幼兒構成致命風險
- 薄荷腦塗抹於嬰幼兒鼻腔或胸口，可能引發喉痙攣（laryngospasm），阻礙呼吸
- **請將虎標萬金油存放於兒童無法觸及之處**
- **12歲以下兒童**使用前應先諮詢醫生或藥劑師（FDA DailyMed標籤要求）

**參考**：FDA 21 CFR 341.14；American Academy of Pediatrics, *Pediatrics* 1994;94(1):127-128

### 6.2 孕婦及哺乳期婦女

- 樟腦及薄荷腦均可經皮吸收進入血液循環
- FDA DailyMed標籤明確要求：**孕婦或哺乳期婦女使用前應諮詢醫療專業人員**
- 部分精油成分（如肉桂油）在高劑量下具有子宮收縮風險
- 建議：小面積、低頻率使用；避免塗抹腹部；如有任何不適立即停用並就醫
- 香港婦女可向**母嬰健康院（MCH）**或婦產科醫生查詢

### 6.3 G6PD缺乏症（蠶豆症）患者 — 高度警惕

蠶豆症（G6PD缺乏症）在香港南方華人群體中並不罕見，估計男性患病率約為4—5%。使用萬金油前，蠶豆症患者必須了解以下風險：

- 樟腦及薄荷腦均具有促氧化（pro-oxidant）特性
- 研究顯示：G6PD缺乏新生兒接觸含薄荷腦製品後，黃疸發生率顯著升高
- 動物模型中，薄荷腦可誘發G6PD缺乏個體出現溶血反應及器官水腫
- **建議**：G6PD缺乏患者（尤其嬰幼兒）應避免使用虎標萬金油及所有含樟腦/薄荷腦的外用製品；如有需要，請先諮詢血液科或遺傳科醫生
- **香港蠶豆症資源**：香港遺傳科學會或各大公立醫院血液科可提供相關指引

**參考**：Olowe SA et al., *Acta Paediatr Scand* 1980;69(3):341-345；Youngster I et al., *Drug Saf* 2010;33(9):713-726

### 6.4 其他禁忌與注意事項

- **皮膚破損、傷口、濕疹患處** — 嚴禁使用，經皮吸收量急劇增加
- **眼睛、口腔、黏膜、生殖器附近** — 嚴禁使用
- **切勿同時使用加熱設備** — 熱敷、電熱毯、密封包紮均會顯著增加藥物吸收，增加中毒風險
- **切勿與其他外用止痛藥疊加使用** — 避免水楊酸類藥物累加中毒（特別是超強版 Muscle Rub）
- **若出現嚴重皮膚刺激、疼痛持續超過7日，或疼痛消退後復發** — 停用並就醫
- **誤食處理** — 立即致電**香港中毒控制中心（HKPCC）**：(852) 2382 0000，或到就近急症室

## 七、常見問題（FAQ）

**Q1：紅色和白色萬金油到底買哪一罐？**

白色偏涼，適合頭痛、傷風鼻塞、蚊叮蟲咬等「涼感」需求，亦適合炎熱天氣使用；紅色偏暖，丁香油和肉桂油含量更高，適合肌肉痠痛、關節不適等需要溫熱滲透的場景，也適合在香港寒冷冬日使用。如只購一罐，白色版應用範圍更廣。

**Q2：新加坡版與香港版成分一樣嗎？**

不完全一樣。新加坡本地版的樟腦濃度可達25%，而美國版、澳洲版及部分香港銷售版本為11%，受各地法規限制。其他輔料比例亦可能有所調整。建議直接查閱所購版本的包裝標籤。

**Q3：虎標萬金油可以口服嗎？**

**絕對不可以。** 虎標萬金油為外用製品，口服樟腦可致抽搐、肝損傷，甚至死亡。誤食後應立即致電**香港中毒控制中心（2382 0000）**或到就近急症室求診。

**Q4：萬金油過期了還可以用嗎？**

不建議。過期後活性成分（尤其各種精油）可能氧化變質，療效下降，且可能產生刺激性分解物。請按包裝標示的有效期使用。

**Q5：用完萬金油後手碰到眼睛怎麼辦？**

立即以大量清水沖洗眼睛至少15分鐘。如持續刺痛、流淚、視力模糊，應盡快就醫。

**Q6：萬金油可以治燙傷嗎？**

**不可以。** 燙傷後塗萬金油是常見誤解——樟腦和薄荷腦會刺激受損皮膚，加重損傷。正確處理方法：以流動冷水沖洗至少20分鐘，然後就醫，切勿塗抹任何藥膏或油脂。

**Q7：可以同時使用萬金油和其他外用止痛貼（如撒隆巴斯）嗎？**

不建議疊加使用。撒隆巴斯等產品同樣含水楊酸甲酯及薄荷腦，疊用可能導致皮膚刺激或藥物過量吸收。如需同時使用，請先諮詢藥劑師。
