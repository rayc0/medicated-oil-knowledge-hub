# 白花油完全指南

白花油（Pak Fah Yeow）是香港家家戶戶幾乎必備的外用藥油，無論是頭痛、肌肉酸痛、蚊叮蟲咬，還是暈船浪，都有人第一時間拿出白花油來應急。它由余仁生（Hoe Hin）品牌生產，源自廣東傳統藥油配方，在香港、新加坡及東南亞一帶有逾百年歷史。

## 白花油的起源與歷史

白花油由廣東商人胡文虎家族於二十世紀初引入東南亞市場。相傳配方源自民間跌打藥油，由祖傳秘方改良而成，後由余仁生公司正式生產及推廣。因其療效受民間廣泛認可，白花油迅速成為香港市民熟悉的家庭常備藥。

在香港，白花油長期在屈臣氏（Watsons）、萬寧（Mannings）及百佳等藥房及超級市場上架，不少長輩更視之為萬能藥油，家中永遠備有一支。

## 主要成分

白花油的成分主要為天然植物精油，透過揮發性芳香物質發揮舒緩效果：

- **薰衣草油（Lavender Oil）**：主要成分，約40%，具鎮靜及輕微止痛作用
- **薄荷腦（Menthol）**：帶來清涼感，有助舒緩頭痛及鼻塞
- **桉樹油（Eucalyptus Oil）**：具抗炎及清新鼻腔作用
- **水楊酸甲酯（Methyl Salicylate）**：俗稱冬青油，深層舒緩肌肉及關節不適
- **石蠟油（Paraffin Oil）**：作為基底載體，幫助精油均勻滲透

正因為這些成分，白花油帶有清涼薄荷加淡雅花香的獨特氣味，與虎標萬金油（Tiger Balm）的辛辣氣味明顯不同。

## 主要用途

### 頭痛及偏頭痛

用少量白花油塗於太陽穴及前額，以指腹輕力打圈按摩。薄荷腦的清涼感有助促進局部血液循環，暫時舒緩緊張性頭痛。此方法在香港辦公室及學校廣泛流傳，尤其考試季節更為常見。

### 暈車浪及暈船浪

乘坐交通工具前，在人中、耳後及手腕內側塗少量白花油，有助紓緩暈眩感。天星小輪、往返離島的渡輪上，不時可見乘客使用白花油止暈。

### 蚊叮蟲咬

塗於患處可即時帶來清涼止癢感，適合香港夏季蚊患嚴重時使用。惟對於已有紅腫或過敏反應者，應先諮詢醫生或藥劑師。

### 肌肉酸痛及筋骨不適

運動後或長時間伏案工作後，取適量白花油塗於頸、肩、腰等部位，輕力按摩，有助緩解肌肉繃緊感。

### 鼻塞及傷風初期

少量白花油塗於鼻翼兩側或嗅聞瓶口，桉樹油成分有助暫時紓緩鼻塞。不宜塗於鼻腔內部。

## 白花油 vs 虎標萬金油

| 比較項目 | 白花油 | 虎標萬金油（白色） |
|---|---|---|
| 氣味 | 清涼淡雅（薰衣草+薄荷） | 辛辣濃烈（樟腦+薄荷） |
| 質地 | 液態油 | 軟膏 |
| 主打成分 | 薰衣草油、薄荷腦 | 樟腦、薄荷腦 |
| 適用場景 | 頭痛、暈浪、蚊叮 | 肌肉酸痛、頭痛 |
| HK售價（約） | HK$25–35 / 小瓶 | HK$30–45 / 小罐 |

兩者各有擁躉，香港家庭不少同時備有兩款。

## 使用注意事項

1. **純外用**：白花油切勿口服，成分具毒性。
2. **嬰幼兒禁用**：兩歲以下嬰幼兒不宜使用任何含薄荷腦或樟腦的藥油，可能引致呼吸困難。
3. **蠶豆症（G6PD缺乏症）患者**：薄荷腦及水楊酸甲酯可能對G6PD患者有潛在風險，使用前應先諮詢醫生。詳見本站〈蠶豆症患者用藥油注意事項〉。
4. **懷孕及哺乳期**：建議孕婦及哺乳期婦女使用前先徵詢醫生意見。
5. **避免接觸眼睛及破損皮膚**：若不慎接觸眼睛，應立即用清水沖洗。
6. **密封存放**：避免日光直射，存放於陰涼乾燥處，精油成分揮發後效果會減弱。

香港各大藥房均有售，包括：

- **屈臣氏（Watsons）**：全港各分店均有存貨，部分分店有大、中、小瓶裝

- **百佳超級市場**：部分分店設有藥品區

海外港人可透過亞馬遜英國/美國、YesStyle 或 iHerb 等平台訂購，惟價格較貴。

## 總結

白花油是香港人幾代同堂共用的家庭良伴，其清涼舒緩的效果及獨特香氣，已成為不少人的成長記憶。雖然它並非萬能，亦有其使用限制，但作為日常小傷小病的輔助舒緩工具，白花油依然是香港藥箱中不可或缺的一員。

如有嚴重病況或症狀持續，請及時就醫，切勿以藥油取代正規治療。
