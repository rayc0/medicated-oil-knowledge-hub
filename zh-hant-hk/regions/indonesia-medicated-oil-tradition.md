# 印尼藥油傳統完整指南：白樹油、Telon 嬰兒油與 Jamu 草藥體系

> **摘要** — 印尼是全球藥油文化最深厚的國家之一，其傳統比荷蘭殖民檔案能記錄的更古老。理解印尼藥油，必須先分清四個本地類別：**白樹油 Minyak Kayu Putih（卡療油 / 白千層精油）**作為家家戶戶的萬用油；**Minyak Telon** 三油配方專供新生兒與嬰幼兒；**Minyak Gosok / Minyak Urut** 是成人按摩驅風油；以及 **Minyak Angin**（「風油」）用於暈車暈船與感冒鼻塞。本文梳理植物學源頭、活性成份、Cap Lang 鷹標 / Cap Kapak 斧標 / Konicare 等代表品牌，並解釋印尼藥油與 Jamu（佳木）草藥傳統的共生關係，以及華人買家、孕嬰家庭與穆斯林消費者最關心的清真認證與安全問題。

## 一、印尼藥油的四個本地類別——千萬別混為一談

西方旅遊指南往往把印尼藥油籠統稱為「巴爾姆 / balm」，但印尼本地市場對它們的分類極為嚴格：

| 印尼名稱 | 中文俗稱 | 實質 | 典型使用者 |
|---------|---------|------|-----------|
| Minyak Kayu Putih | 白樹油 / 卡療油 / 白千層油 | 接近純品的白千層精油（含 1,8-桉葉素 50–65%） | 全家通用，從嬰兒到老人 |
| Minyak Telon | 「Telon 三合一」嬰兒油 | 白樹油 + 茴香油 + 椰子油的三油配方 | 新生兒至 2 歲嬰幼兒 |
| Minyak Gosok / Urut | 按摩油 / 跌打油 | 多成份紅腫溫熱油（樟腦、薄荷、桂皮、丁香） | 成人，運動後或感冒受寒 |
| Minyak Angin | 風油 | 嗅吸 + 塗抹兩用的清涼油 | 嬰兒以上全年龄 |

「Minyak」在印尼語和馬來語中只是「油」的意思，關鍵在修飾詞。一個雅加達祖母給孫仔腹絞痛塗的 *minyak kayu putih*，與一位肌肉勞損的祖父睡前塗的 *minyak gosok*，從藥理上完全不是同一類東西，儘管它們在便利店貨架上瓶身長得差不多。**這是華人買家在印尼旅遊或托代購時最常踩的坑**——把嬰兒用的白樹油當成成人按摩油，或反過來把含高濃度樟腦的成人按摩油給嬰兒用。

## 二、白樹油（Minyak Kayu Putih）——印尼版的「萬金油基底」

### 2.1 植物學與命名

*Kayu putih* 在印尼語 / 馬來語中字面意思就是「白色的木」，得名於白千層（*Melaleuca cajuputi*，及商用近緣種 *M. leucadendra*）那層層剝落的紙狀白色樹皮。英文裡「cajuput / cajeput」就是馬來原文的直接音譯。

這種樹原產於印尼群島，主產區集中在 **馬魯古群島（Maluku，舊稱香料群島 / Moluccas）、蘇拉威西島（Sulawesi）與巴布亞（Papua）**。其中布魯島（Buru Island）上由印尼林業部經營的國營白千層油廠（PMKP）已經持續生產超過一個世紀，今天蘇拉威西仍是產量最大的產區。

**與澳洲茶樹油的區別**：白千層油（cajuput）、互生葉白千層 / 茶樹油（*M. alternifolia*）和綠花白千層（*M. quinquenervia* / niaouli）雖然同屬白千層屬，但化學組成完全不同。白樹油主打 1,8-桉葉素，茶樹油主打松油烯-4-醇，**兩者不可互換**。

### 2.2 化學成份

藥典級印尼白樹油的主要成份：

- **1,8-桉葉素（Eucalyptol / Cineole）50–65%** — 主要活性，與桉樹油共有的擬刺激、祛痰、溫和發紅劑作用
- α-蒎烯、β-蒎烯 — 微量驅風作用
- α-松油醇、檸檬烯 — 抗菌與香氣調和
- 痕量松油烯-4-醇

可以把白樹油理解為「印尼版的桉樹油，但氣味更甜更柔，對兒童更友好」。詳細的桉葉素藥理可參考[桉油藥理完整指南](/zh-hant-hk/ingredients/eucalyptus-oil-pharmacology)。

### 2.3 印尼人怎麼用

- **沐浴後塗抹**：印尼媽媽給嬰兒洗澡後例行塗抹背部、胸口與腳底，「防寒風」（*masuk angin*）
- **腹絞痛 / 脹氣**：在肚臍周圍順時針打圈按摩
- **感冒鼻塞**：胸口與背部薄塗，或滴幾滴在枕頭上嗅吸
- **蚊叮蟲咬**：直接點塗患處止痕
- **肌肉痠痛**：與按摩同步使用（成人）

**重要差別**：與中國的清涼油 / 白花油不同，印尼白樹油常常 **接近 100% 純白千層精油**（部分產品會用椰子油稀釋）。這意味著它的桉葉素濃度比清涼油裡的桉葉素含量高出十幾倍，使用時**用量要小**。

## 三、Minyak Telon——新生兒出院第一天就開始塗的油

### 3.1 配方

Telon 在爪哇語中意為「三」，指三種油的固定配比：

- **白樹油（Minyak Kayu Putih）約 30–62%** — 主要驅風成份
- **茴香油（Minyak Adas / Fennel Oil）約 13–16%** — 助消化、緩脹氣
- **椰子油（Minyak Kelapa）約 22–55%** — 基底油，稀釋 + 滋潤

各品牌微調比例與額外香料（如薰衣草、洋甘菊），但核心三油不變。Konicare、Cap Lang Telon Plus、Caplang GPU、My Baby 是幾大主流品牌。

### 3.2 為什麼是這三種油

印尼傳統育兒觀念認為新生兒「體寒易受風」（*masuk angin*）。三油協同作用：

- 白樹油的 1,8-桉葉素提供溫和的「溫感」（實際上是 TRPM8 / TRPV1 受體激動產生的感覺錯覺，**不會真的提高體溫**）
- 茴香油的反式茴香腦（trans-anethole）是**唯一被多項兒科 RCT 證實可緩解嬰兒腸絞痛的精油成份**
- 椰子油作為大分子載油，降低活性物的透皮速度，形成保護性油膜

### 3.3 用法與禁忌

**標準用法**：嬰兒沐浴後 5–10 分鐘內，將 Telon 油倒在大人手掌中搓熱（不可直接倒在嬰兒身上），薄塗前胸、背部、肚臍周圍、腳底與腳踝。

**安全紅線**：

1. **不可塗鼻孔下方與人中**——含桉葉素的油用於 2 歲以下嬰幼兒面部，有誘發喉痙攣與呼吸抑制的報告。這條規則在印尼也常被忽略，但兒科文獻立場明確。
2. **避開眼睛、生殖器與開放傷口**
3. **G6PD 缺乏症（蠶豆症）患儿慎用**——Telon 中常見的薰衣草版本含芳樟醇，部分配方還會添加微量樟腦。詳見 [G6PD 缺乏症與藥油安全指南](/zh-hant-hk/safety/g6pd-deficiency)
4. **不要「為了更暖」加大用量**——透皮吸收劑量隨面積線性上升

## 四、按摩油 Minyak Gosok / Minyak Urut——成人專用的紅腫溫熱油

### 4.1 與白樹油的本質區別

白樹油單一成份（白千層精油），按摩油則是**多成份紅腫溫熱複方**，典型成份譜：

- 樟腦（Camphor）3–10%
- 薄荷醇（Menthol）3–8%
- 水楊酸甲酯（Methyl Salicylate）少量至 10%
- 丁香油、桂皮油、薑油、肉豆蔻油 — 提供溫感與活血印象
- 椰子油或礦物油基底

代表品牌：**Cap Tawon（蜜蜂牌）、Cap Kapak（斧標）GPU、Tresnojoyo、FX Balsem**。其中 Cap Kapak GPU 在馬來西亞與新加坡華人圈知名度最高，常被誤認為是「印尼版的虎標」。

### 4.2 使用場景：Kerokan 刮痧

按摩油在印尼最具文化辨識度的用法是 **kerokan**——用一枚硬幣（*koin*）蘸油，在背部沿膀胱經反覆刮擦，直到皮膚出現暗紅色條狀瘀痕。這是華人移民帶來的刮痧文化在印尼本土化的形態，幾乎每個爪哇家庭都會做。

**與中國刮痧的區別**：中國刮痧多用刮痧板與萬花油 / 紅花油，印尼 kerokan 則使用硬幣（舊荷蘭盾或現代 Rupiah 硬幣）配合按摩油，刮拭力度通常更重，留下的痧痕更深。

### 4.3 成人用法注意

- 單次塗抹面積不超過手掌大小
- 含水楊酸甲酯的產品**禁用於 12 歲以下兒童**與**正在服用華法林等抗凝藥的患者**——參考[藥油與抗凝藥相互作用安全指南](/zh-hant-hk/safety/medicated-oils-and-anticoagulant-medication-safety)
- 塗後避免直接日曬（部分柑橘類精油有光毒性）

## 五、Minyak Angin——印尼版的「嗅吸式白花油」

Minyak Angin 字面意思就是「風油」，專門用於 **暈車、暈船、頭暈、感冒鼻塞、突發噁心**。與新加坡的虎標萬金油 Aromatherapy 系列、中國的白花油同屬一類，但印尼版本通常更稀（樟腦 + 薄荷 + 桉葉素的酒精溶液），強調「開蓋嗅吸即起效」，而非長時間塗抹。

代表產品：**Cap Lang Minyak Angin Aromatherapy、FreshCare、Plossa（嗅吸 + 塗抹 + 刮痧三合一棒）**。Plossa 是近年大熱的便攜產品，瓶身設計成唇膏狀，一端可作為 kerokan 的刮痧頭使用。

## 六、Jamu 草藥傳統——印尼藥油背後的文化骨架

要理解為什麼印尼藥油如此普及，必須知道 **Jamu（佳木）草藥體系**。Jamu 是印尼自 8 世紀起延續至今的傳統草藥醫學，2023 年被 UNESCO 列入人類非物質文化遺產名錄。

Jamu 與中醫、印度阿育吠陀並列為亞洲三大本土醫學傳統，但 Jamu **更去儀式化、更家庭化**：

- 沒有中醫的「經絡臟腑」理論框架
- 沒有阿育吠陀嚴格的 Vata / Pitta / Kapha 體質分型
- 核心是「對症使用 + 經驗配方 + 女主導的家庭傳承」

歷史上 Jamu 幾乎完全由女性掌握。荷蘭殖民時期（17–20 世紀），荷蘭婦女與印尼婦女共同經營 Jamu 業務，留下了大量殖民地藥典記錄。其中最重要的早期文獻是德國-荷蘭博物學家 **Rumphius** 在馬魯古群島安汶（Ambon）寫成的 *Herbarium Amboinense*（《安汶植物誌》，1741 年），其中詳細記載了白千層的提油工藝。

**藥油是 Jamu 體系中「外用方」的代表**——內服用煎劑 / 粉劑 / 蜜丸（Jamu Gendong 流動小販至今仍背着藥籃在巴厘島的市場叫賣），外用則用 minyak（油）+ pilis（額頭敷藥）+ param（身體塗膏）。

## 八、華人買家與海外代購的實操要點

### 8.1 真偽辨別

印尼藥油的假冒主要發生在馬來西亞與新加坡的小雜貨店。三個核心識別點：

1. **BPOM 註冊號**——印尼藥監局 BPOM 的註冊號格式為 *POM TR / TI XXXXXXXXXX*（草藥類）或 *POM HT XXXXXXXXX*。瓶身或包裝盒上必須印有
2. **MUI 清真認證（Halal）**——Cap Lang、Konicare、My Baby、Cap Kapak 都有印尼烏理瑪理事會 MUI 的清真標識，假貨往往遺漏或印刷模糊
3. **批號與生產日期**——壓印在瓶底的鋼印批號應清晰，噴印模糊或日期與瓶身不符多為翻新假貨

### 8.2 華人體質適用性

印尼藥油的桉葉素 + 茴香油配方與中國傳統的樟腦 + 薄荷醇配方在感官上有明顯差異——**印尼油更「甜」、更「暖」，華人初次使用可能覺得「唔夠攻」**。但實際功效不弱，特別是 Minyak Telon 在嬰兒脹氣與腸絞痛上的循證基礎比中國的小兒驅風油更扎實（fennel anethole 的 RCT 數據）。

### 8.3 與中國 / 香港藥油的搭配建議

許多印尼華人家庭會同時備着：

- **印尼白樹油**（嬰兒與日常）
- **香港黃道益活絡油**（成人跌打）
- **新加坡虎標白色萬金油**（頭痛蚊咬）

三者互補——白樹油負責嬰兒與全家通用，活絡油負責跌打勞損，虎標負責蚊蟲叮咬與暈車。詳細對比參考[東南亞華人藥油傳統綜述](/zh-hant-hk/regions/southeast-asia-chinese)。

## 九、安全須知（不可忽略）

1. **白千層精油原液不可口服**——印尼民間偶有「飲幾滴白樹油治胃痛」的偏方，但 1,8-桉葉素口服劑量超過 5 mL 可引起嚴重中樞神經抑制
2. **嬰幼兒用量極少**——Minyak Telon 單次 0.5–1 mL（約 8–15 滴）足夠全身使用
3. **G6PD 缺乏症兒童**慎用所有含薰衣草、樟腦、薄荷腦成份的嬰兒油——這是華南、東南亞華人家庭高發的代謝病
4. **孕婦前三個月**避免大面積使用所有按摩油（含水楊酸甲酯、丁香油、肉桂油的產品風險更高）——參考[孕婦與藥油安全指南](/zh-hant-hk/safety/pregnancy-guide)
5. **過敏體質**——白千層精油在桃金娘科過敏者中可引起接觸性皮炎，初次使用建議手腕內側斑貼測試 24 小時

## 十、結語

印尼藥油不是單一商品，而是一整套貫穿嬰兒 → 成人 → 老人、覆蓋日常保健 → 跌打損傷 → 文化儀式（kerokan）的家庭健康體系。它根植於 Jamu 千年草藥傳統，在荷蘭殖民與華人移民的雙重影響下完成了現代化，今天通過 Cap Lang、Konicare、Cap Kapak 等品牌走向全球。

對中國與香港的消費者而言，最值得收藏的入門組合是 **一支 60 mL 的 Cap Lang Minyak Kayu Putih + 一支 Konicare Minyak Telon Plus**——前者是全家通用的「印尼萬金油基底」，後者是世界少見的、循證基礎扎實的嬰兒專用油。
