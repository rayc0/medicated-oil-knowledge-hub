# 長者藥油安全使用指南：年齡相關風險同注意事項

香港長者係藥油使用量最高嘅群體。根據非正式調查，**超過85%**嘅65歲以上香港長者定期使用藥油——比任何其他年齡組別都高。虎標萬金油喺床頭櫃、保心安油喺手袋、白花油喺衫袋——藥油係好多長者日常生活嘅一部分。

但70歲嘅身體同30歲嘅身體處理藥物嘅方式完全唔同。皮膚變薄增加藥物吸收、肝腎功能下降減慢代謝、長期病藥物同藥油可能產生交互作用——呢啲都係長者同佢哋嘅家人需要了解嘅重要安全資訊。

---

## 年齡帶嚟嘅生理變化同藥油安全

| 生理變化 | 對藥油使用嘅影響 | 實際風險 |
|---|---|---|
| 皮膚變薄（表皮減少20-30%） | 藥物透皮吸收增加 | 同樣嘅搽法，長者吸收嘅劑量可能係年輕人嘅1.5-2倍 |
| 皮下脂肪減少 | 脂溶性成分蓄積減少 | 血液中藥物濃度更快升高 |
| 肝功能下降 | 代謝藥物嘅速度減慢 | 水楊酸甲酯、樟腦等成分清除更慢 |
| 腎功能下降 | 排泄減慢 | 代謝物喺體內停留時間更長 |
| 溫度感覺減退 | 對灼熱感反應遲鈍 | 藥油造成嘅皮膚灼傷而唔自知 |
| 嗅覺退化 | 用藥量唔自覺增加 | 因為「聞唔到味」而反覆搽，導致過量 |

## 長者最常見嘅藥油安全問題

### 1. 搽太多、搽太密

好多長者因為痛感強烈或者嗅覺退化，會**不斷重複搽藥油**——有時候一日搽5-6次甚至更多，每次搽嘅面積都好大。

**風險**：水楊酸甲酯累積中毒症狀包括耳鳴、噁心、呼吸急促、混亂。嚴重個案可以致命。

**建議**：
- 每日最多3次
- 每次搽嘅面積唔應該超過一個手掌大小
- 家屬可以幫長者將藥油分裝到小樽，限制每日用量

### 2. 搽喺損傷皮膚上

長者皮膚容易乾燥龜裂、有傷口、濕疹、腳部潰瘍（尤其糖尿病患者）。搽藥油喺受損皮膚上會：

- 造成劇痛
- 增加全身性吸收
- 延遲傷口癒合
- 可能引起接觸性皮膚炎

**建議**：搽藥油之前檢查皮膚有冇傷口、紅疹、龜裂。

### 3. 同長期病藥物交互作用

詳見[藥油與處方藥物交互作用](/zh-hant-hk/safety/drug-interactions)文章。最危險嘅組合：

- 薄血丸（華法林）+ 含水楊酸甲酯藥油 → 出血
- 糖尿藥 + 大量水楊酸甲酯 → 低血糖
- 抗癲癇藥 + 樟腦 → 降低發作閾值

### 4. 誤食

認知障礙症（腦退化症）長者可能將藥油誤認為飲品而吞食。香港中毒諮詢中心每年接獲多宗長者誤食藥油嘅個案。

**建議**：
- 藥油遠離餐桌同飲品存放處
- 對有認知障礙嘅長者，藥油由照護者保管同施用
- 萬一誤食：唔好催吐，立即致電999或中毒諮詢熱線

## 長者各品牌安全評級

| 藥油 | 主要風險成分 | 長者安全性 | 特別注意 |
|---|---|---|---|
| 白花油 | 薄荷腦、水楊酸甲酯（少量） | ★★★★☆ | 相對安全，適合長者日常使用 |
| 虎標萬金油（白） | 樟腦、薄荷腦 | ★★★★☆ | 適量使用安全 |
| 保心安油 | 薄荷腦、肉桂油 | ★★★★☆ | 長者最愛，相對安全 |
| 虎標萬金油（紅） | 樟腦、水楊酸甲酯 | ★★★☆☆ | 華法林/薄血丸患者注意 |
| 斧標驅風油 | 薄荷腦、樟腦 | ★★★☆☆ | 適量使用 |
| 均隆驅風油 | 水楊酸甲酯15% | ★★☆☆☆ | 薄血丸患者應避免 |
| 黃道益活絡油 | 水楊酸甲酯 | ★★☆☆☆ | 大面積使用風險較高 |
| 跌打酒/跌打油 | 水楊酸甲酯高濃度 | ★☆☆☆☆ | 長者應避免或極少量使用 |

## 照護者/家屬指南

如果你照顧緊一位長者，以下建議可以幫助管理藥油安全：

1. **了解長者用緊咩藥**：帶齊藥物清單去問藥劑師有冇交互作用
2. **檢查藥油成分**：重點睇水楊酸甲酯同樟腦嘅含量
3. **限制用量**：幫長者分裝小樽或設定每日用量上限
4. **觀察皮膚反應**：定期檢查長者常搽藥油嘅部位有冇紅疹、灼傷
5. **安全存放**：認知障礙症患者嘅藥油要鎖起
6. **記錄使用情況**：覆診時告知醫生長者嘅藥油使用習慣

## 緊急處理

| 情況 | 處理 |
|---|---|
| 皮膚過敏反應 | 用清水徹底沖洗，唔好用熱水 |
| 眼睛接觸藥油 | 用大量清水沖洗15分鐘，求醫 |
| 誤食藥油 | 唔好催吐，致電999 |
| 大面積搽藥油後頭暈/耳鳴 | 清洗皮膚，求醫，帶埋藥油樽去 |
| 使用後出血異常 | 如正服用薄血丸，立即求醫 |

| 重點 | 內容 |
|---|---|
| 長者風險高嘅原因 | 皮膚薄、肝腎慢、藥物多、感覺退化 |
