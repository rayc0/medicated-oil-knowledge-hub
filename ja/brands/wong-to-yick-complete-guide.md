# 黄道益（Wong To Yick）活絡油完全ガイド

黄道益活絡油（Wong To Yick Wood Lock Medicated Oil）は、1927年に香港で創業した老舗薬用オイルブランドです。「活絡油」という名称は中国伝統医学の概念「通絡活血（経絡を通し血行を促す）」に由来し、筋肉痛・関節痛・捻挫・打撲を中心とした運動器系症状への特化製品として香港・広東地区で長く支持されてきました。現在はリッチモンド薬業（Ricky Chemical）グループが製造し、香港・マカオ・東南アジア・北米の中国系コミュニティで広く流通しています。

## 1. 黄道益（Wong To Yick）とは？

| 項目 | 内容 |
|---|---|
| ブランド名（英語） | Wong To Yick Wood Lock Medicated Oil |
| 中国語名 | 黄道益活絡油 |
| 創業 | 1927年、香港 |
| 製造元 | Ricky Chemical Co., Ltd.（リッチモンド薬業） |
| 剤形 | 液体オイル（ロールオン・瓶タイプ） |
| 主な市場 | 香港、マカオ、東南アジア、北米華僑コミュニティ |
| 製品コンセプト | 筋肉痛・関節痛・捻挫・打撲への運動器系特化 |
| 伝統処方の根拠 | 中国伝統医学「通絡活血」思想に基づく配合 |

黄道益の最大の特徴は、**メチルサリチル酸（サリチル酸メチル）を50%という業界内でも高い濃度で配合している**点にあります。この高濃度配合により、深部筋肉・関節への浸透性と鎮痛効果が強く、スポーツ後の筋疲労や関節の炎症にターゲットを絞った設計となっています。

## 2. 成分分析表

黄道益活絡油の主要成分と各成分の薬理作用機序を以下に示します。

| 成分名 | 配合濃度（目安） | 薬理作用機序 | 主な効果 |
|---|---|---|---|
| メチルサリチル酸（Methyl Salicylate） | 約50% | COX（シクロオキシゲナーゼ）阻害によるプロスタグランジン産生抑制、皮膚からの深部浸透 | 抗炎症、鎮痛、筋肉痛・関節痛の緩和 |
| メントール（Menthol） | 約16% | TRPM8冷感受容体の活性化による清涼感・反刺激鎮痛、末梢血管への局所作用 | 冷感、清涼感、軽度鎮痛 |
| 樟脳（カンファー / Camphor） | 約10% | TRPV1およびTRPM8の二重刺激による温冷感、局所血流促進、反刺激鎮痛 | 温熱感、血行促進、筋肉緩和 |
| ユーカリ油（Eucalyptus Oil） | 少量 | 1,8-シネオールによる局所刺激・経皮吸収促進補助 | 補助的な通気感・吸収補助 |
| 冬緑油（Wintergreen Oil） | 微量含有の場合あり | メチルサリチル酸の天然由来補強 | 抗炎症補助 |

### 成分の特徴：メチルサリチル酸50%が意味すること

メチルサリチル酸は経皮吸収後、体内でサリチル酸に代謝され、COX阻害を介した局所抗炎症・鎮痛作用を発揮します。一般的な薬用オイル（Tiger Balm Redで約25%、斧標驅風油で約15–20%）と比較して、**黄道益の50%配合は約2倍の濃度**にあたります。

この高濃度は：
- 深部筋肉・靭帯・関節周囲組織への浸透性向上
- より持続的な鎮痛・抗炎症効果
- 一方で、皮膚刺激感・全身経皮吸収量の増加

という特性をもたらします。

## 4. 適応症（筋肉痛・関節痛・捻挫・打撲に特化）

黄道益が特に有効とされる症状と使用シーンを以下に示します。

| 症状 | 推奨度 | 使用タイミング | 期待される効果 |
|---|---|---|---|
| 運動後筋肉痛（DOMS） | ◎ 強く推奨 | 運動48–72時間後（急性炎症期後） | 深部筋肉の抗炎症・鎮痛 |
| 慢性的な関節痛（変形性関節症など） | ○ 推奨 | 入浴後など血行促進時 | 関節周囲の局所抗炎症 |
| 捻挫（亜急性期〜回復期） | ○ 推奨 | 受傷後48時間以降（腫脹消退後） | 靭帯周囲の炎症緩和 |
| 打撲（皮下組織損傷） | ○ 推奨 | 内出血安定後（受傷後24–48時間以降） | 局所血流改善・鎮痛 |
| 肩こり・筋緊張 | ○ 推奨 | 日常的使用 | 筋肉の緊張緩和・血行促進 |
| 腰痛（筋性腰痛） | ○ 推奨 | 入浴後に塗布・マッサージ | 腰部筋肉の鎮痛・温熱感 |
| 頭痛・鼻詰まり | △ 不向き | — | 設計対象外（White系製品を選ぶべき） |

**重要な注意**: 捻挫・打撲の**急性期（受傷直後〜48時間）は使用禁止**です。急性炎症期にメチルサリチル酸を塗布すると、血管拡張・血流増加により腫脹・内出血が悪化する可能性があります。受傷直後はRICE処置（安静・アイシング・圧迫・挙上）を優先してください。

## 5. ⚠️ ワルファリン服用者への絶対禁止

**黄道益活絡油はワルファリン（ワーファリン）服用中の方は使用禁止です。**

### なぜ危険なのか：メカニズムの解説

| リスク要因 | 詳細 |
|---|---|
| メチルサリチル酸濃度 | 50%（業界最高水準） |
| 経皮吸収後の代謝物 | サリチル酸（Salicylic acid） |
| ワルファリンとの相互作用 | サリチル酸はCYP2C9阻害によりワルファリン代謝を遅延 → 血中ワルファリン濃度上昇 |
| INRへの影響 | PT-INR（プロトロンビン時間国際標準比）の予測不能な上昇 |
| 臨床リスク | 出血リスクの急増（脳出血・消化管出血・皮下出血） |
| 特に危険な使用法 | 広範囲・長時間・閉塞包帯下の塗布（吸収量が激増） |

### 50%配合の意味するリスク

一般的な薬用バームに含まれるメチルサリチル酸25%（Tiger Balm Red）でも既にワルファリン相互作用リスクが報告されています。黄道益の50%配合はその約2倍の経皮吸収リスクがあり、臨床的により深刻な問題を引き起こす可能性があります。

**医薬文献の報告例**：メチルサリチル酸含有外用剤の過剰使用によるINR上昇・出血事象は複数の症例報告として記録されています（特に高濃度・広範囲使用）。

### ワルファリン服用者への代替提案

| 代替選択肢 | 理由 |
|---|---|
| Tiger Balm White | メチルサリチル酸を含まない（カンファー・メントール主体） |
| 斧標驅風油（Axe Brand） | サリチル酸系成分の比率が低い |
| 主治医・薬剤師への相談 | 非サリチル酸系外用鎮痛剤（ジクロフェナクゲル等）の検討 |

## 6. G6PD欠乏症（蚕豆病）注意事項

G6PD欠乏症（グルコース-6-リン酸脱水素酵素欠乏症）は、東南アジア・中国南部・地中海系に比較的多い遺伝性酵素異常です。

| 項目 | 内容 |
|---|---|
| G6PD欠乏症との関係 | 樟脳（カンファー）成分が酸化ストレスを誘発し溶血を引き起こす可能性がある |
| 黄道益の樟脳含有量 | 約10% |
| リスクレベル | 大量・長期使用、特に乳幼児への使用は禁忌 |
| 乳幼児への絶対禁止 | G6PD診断有無にかかわらず、2歳未満への使用は禁止 |
| 成人G6PD患者 | 少量・短時間の使用でも主治医に事前確認を推奨 |

## 7. 正しい塗布方法

### 基本塗布手順

1. **患部を清潔にする** — 入浴後・洗浄後の清潔な皮膚に塗布します
2. **適量を取る** — ロールオンタイプは直接患部に転がしながら塗布、瓶タイプは指先に数滴取る
3. **円を描くようにマッサージ** — 筋肉の走行に沿って10〜15回、適度な圧をかけながら塗り込む
4. **塗布後は手を洗う** — 目・粘膜・傷口に触れないよう必ず洗手
5. **使用回数** — 1日2〜3回を目安に（使用間隔は4時間以上）

### 温熱直後の使用禁止

**入浴中・直後（10〜15分以内）や電気毛布・カイロ使用直後は塗布禁止です。**

| 理由 | 詳細 |
|---|---|
| 血流増加による過剰吸収 | 温熱で皮膚の毛細血管が拡張し、メチルサリチル酸の経皮吸収量が急増 |
| 皮膚刺激・熱傷リスク | 高濃度サリチル酸 × 熱の複合刺激で皮膚炎・化学的熱傷様症状の可能性 |
| 全身毒性リスク | サリチル酸の血中濃度が安全域を超える可能性（特に広範囲使用時） |

**推奨タイミング**: 入浴後15〜30分経過し、皮膚温が落ち着いてから塗布してください。

### 使用禁止部位・状況

- 目・口・鼻・性器などの粘膜部位
- 傷口・湿疹・炎症を起こした皮膚
- 損傷した皮膚（すり傷・切り傷）
- 顔面（メントール高濃度のため、特に目周辺禁止）
- 2歳未満の乳幼児（全身）
- 気密包帯・ラップ処置下での使用（吸収量が危険レベルに増加）

日本国内では黄道益活絡油は一般薬局での流通がほとんどないため、以下の入手方法を検討してください。

| 購入方法 | 具体的な店舗・手段 | 注意点 |
|---|---|---|
| 横浜・神戸・長崎・大阪の中華街 | 中華系薬局・雑貨店 | 在庫は店舗により異なる |
| 日本の中国系食材店・スーパー | 新大久保（東京）、大阪中央区の中国系店舗 | 取り扱いない店も多い |
| 海外通販（Lazada・Shopee・Qoo10） | オンライン購入 | 正規品確認が必要、送料考慮 |
| 香港・マカオ旅行時の現地購入 | 最も確実で安価 | スーツケース内輸送に問題なし |
| 日本のアマゾン（Amazon.co.jp） | 並行輸入品として出品あり | 価格が割高になる場合あり |

**輸入規制について**: 黄道益活絡油は日本の薬機法上「外用消炎鎮痛剤」に相当する可能性があり、個人輸入の場合は自己責任となります。商業目的での輸入は薬機法上の許可が必要です。
