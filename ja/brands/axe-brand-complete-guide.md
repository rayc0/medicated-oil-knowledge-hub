# 斧標万金油（Axe Brand Universal Oil）完全ガイド

斧標万金油（アックスブランド）は、1915年にシンガポールで創業したLeung Kai Fook Medical社が製造する、110年の歴史を持つアジアの代表的薬用オイルです。シンガポール・マレーシア・香港を中心に東南アジア全域で広く親しまれており、日本では「斧マーク」として知られています。

---

## 1. ブランドの歴史と特徴

| 項目 | 詳細 |
|---|---|
| 創業年 | 1915年（シンガポール） |
| 製造元 | Leung Kai Fook Medical Co., Ltd. |
| 原産地 | シンガポール |
| 代表製品 | Axe Brand Universal Oil（万金油） |
| 主な市場 | シンガポール、マレーシア、香港、台湾、ASEAN全域 |
| 特徴 | Tiger Balmより穏やかな香り・刺激、ファミリー向け |

斧標は「万能オイル」のコンセプトで知られ、1本で頭痛・筋肉痛・乗り物酔い・虫刺されなど多用途に対応するよう設計されています。

## 2. 主要成分分析

| 成分 | 濃度 | 作用機序 | 効果 |
|---|---|---|---|
| メントール（Menthol） | 約7% | TRPM8受容体活性化 → 冷感シグナル | 即時の清涼感・かゆみ緩和 |
| 樟脳（Camphor） | 約7% | 鎮痛・抗かゆみ | 発赤・ひりつき緩和 |
| ユーカリプタス油 | 適量 | 抗菌・抗炎症 | 鼻詰まり緩和・二次感染予防補助 |
| クローブ油（オイゲノール） | 適量 | COX酵素阻害 → 抗炎症 | 腫れ（浮腫）軽減 |
| ナツメグ油 | 微量 | 温熱・鎮静 | 深部の筋肉弛緩補助 |

> **特徴**: Tiger Balm Whiteと近い成分構成だが、各成分濃度がやや穏やかで刺激が少ない。子ども・高齢者・敏感肌に適したバランス。

## 5. 症状別使用ガイド

### 頭痛・こめかみの張り
1. 斧標万金油を米粒大ほど指先に取る
2. こめかみ・額・後頭部に円を描くようにやさしくなじませる
3. 目・鼻・口周囲には絶対に付けない
4. 10〜15分で清涼感とともに頭痛が和らぐことが多い

### 筋肉痛・肩こり
- 患部に適量を塗布し、5〜10分かけてマッサージ
- 塗布後の温熱パッド併用は吸収率を高めるが、**ワルファリン服用者は禁止**
- 2〜3時間おきに再塗布可能

### 乗り物酔い
- 出発前：鼻下・手首内側（P6内関穴）に塗布
- 吐き気が出始めたら：こめかみと鼻下に追加塗布
- 香りを直接嗅ぐことで嘔吐反射を一時的に抑制する効果も

### 虫刺され（蚊・アリなど）
1. 刺された部位を清潔な水で洗う
2. 斧標万金油を少量だけ患部に直接塗布
3. 10〜15分でかゆみが落ち着く

### 鼻詰まり（軽度）
- 鼻の下（人中付近）に微量塗布
- または蒸気吸入：ボウルの熱湯に1〜2滴たらしてタオルで覆い吸入

---

## 6. 塗布の基本ルール

| ルール | 内容 |
|---|---|
| 使用量 | 米粒〜緑豆大。少量から始める |
| 再塗布間隔 | 2〜3時間以上あける |
| 塗布後の手洗い | 必ず手を洗い、目・粘膜に触れない |
| 密閉包帯禁止 | ラップ・包帯で覆うと吸収量が急増 |
| 温熱直後の使用 | 入浴・温湿布直後は血流↑→吸収↑ → 特にワルファリン服用者注意 |

### 絶対に使用しない部位
- 目・まぶた周囲
- 口唇・鼻の粘膜
- 傷口・発疹のある皮膚
- 性器周囲

---

## 7. 使用注意・禁忌

| 対象 | 注意事項 |
|---|---|
| **2歳未満** | すべての薬用オイル使用禁止（メントール・樟脳による呼吸抑制リスク） |
| **2〜12歳** | 斧標万金油は比較的穏やかだが、顔・鼻周囲は禁止。保護者の監視下で少量のみ |
| **妊婦** | 使用前に医師に相談。特に樟脳は胎盤通過性あり |
| **ワルファリン服用者** | **Axe Brand Universal Oil（万金油）はメチルサリチル酸を含まないため比較的安全**。ただしAxe Brand Red Oilは高濃度メチルサリチル酸含有 → 使用禁止 |
| **G6PD欠乏症** | メントール・樟脳は溶血誘発の可能性 → 医師に相談 |
| **敏感肌・アレルギー** | パッチテスト（内腕に少量塗布・24時間確認）を推奨 |

## まとめ

斧標万金油（Axe Brand Universal Oil）は、110年の歴史を持つシンガポール製の多用途薬用オイルで、メントール7%・樟脳7%の穏やかな配合が特徴です。Tiger Balm Whiteより刺激が少なく、ファミリー向け・高齢者・子ども（2歳以上）に適した選択肢です。頭痛・筋肉痛・乗り物酔い・虫刺されに幅広く対応します。**Axe Brand Universal Oil（万金油）はメチルサリチル酸を含まないため、ワルファリン服用者にも相対的に安全ですが、Axe Brand Red Oilは使用禁止**。2歳未満への使用は禁止。
