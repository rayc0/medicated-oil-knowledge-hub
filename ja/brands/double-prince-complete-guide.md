# 双獅標（Double Prince）活絡油完全ガイド

双獅標活絡油（Double Prince Medicated Oil）は、香港・広東地区を拠点とする老舗薬用オイルブランドです。「双獅」（二頭の獅子）というシンボルは強さと伝統を表し、香港の家庭薬文化の中で根強い支持を持ちます。Tiger Balm（虎標）や黄道益ほど国際的な知名度はありませんが、地元の香港人・広東系コミュニティには馴染み深い製品として長年愛用されています。

---

## 1. 双獅標（Double Prince）とは？

| 項目 | 内容 |
|---|---|
| ブランド名（英語） | Double Prince Medicated Oil |
| 中国語名 | 双獅標活絡油 |
| 原産地 | 香港 |
| 主な流通地域 | 香港、マカオ、広東省、東南アジアの広東系コミュニティ |
| 剤形 | 液体オイル（瓶・小型ロールオン） |
| 製品コンセプト | 日常的な筋肉痛・打撲・冷え性・疲労感への汎用薬用オイル |
| ブランドの特徴 | 低刺激寄りの処方、地元向けの手頃な価格設定 |

双獅標は、黄道益のような「高濃度・高刺激・運動器特化」ではなく、**日常的な体の不調に対して使いやすい、やや穏やかな処方**を特徴とします。香港の家庭では冬場の冷え性ケアや、軽度の打撲・筋肉疲労時の日常ケアとして使用される機会が多い製品です。

## 2. 成分分析表

双獅標活絡油の一般的な処方成分と各成分の作用を以下に示します。なお、製品バッチや販売地域によりわずかに配合比率が異なる場合があります。

| 成分名 | 配合濃度（目安） | 薬理作用機序 | 主な効果 |
|---|---|---|---|
| メチルサリチル酸（Methyl Salicylate） | 約20–30% | COX阻害によるプロスタグランジン産生抑制、経皮浸透による局所抗炎症 | 筋肉痛・関節痛の鎮痛、抗炎症 |
| メントール（Menthol） | 約10–15% | TRPM8冷感受容体の活性化、末梢神経への反刺激鎮痛 | 清涼感、軽度鎮痛、冷感リフレッシュ |
| 樟脳（カンファー / Camphor） | 約5–10% | TRPV1・TRPM8の二重刺激、局所血流促進 | 温感・血行促進・筋肉緩和 |
| ユーカリ油（Eucalyptus Oil） | 少量 | 1,8-シネオールによる局所刺激・経皮吸収促進補助 | 清涼感補助、通気感 |
| ハッカ油（Peppermint Oil） | 少量（配合品による） | メントールの補強、清涼感強化 | 芳香・清涼感強化 |
| テレビン油（Turpentine Oil） | 微量（配合品による） | 局所刺激・血行促進補助 | 筋肉への温感補助 |

### 黄道益との成分比較ポイント

双獅標のメチルサリチル酸は約20–30%と、黄道益（50%）の約半分から6割の濃度です。その分、**皮膚刺激が穏やかで、初めて薬用オイルを使う方や刺激に敏感な方でも比較的使いやすい**処方となっています。ただしワルファリン相互作用リスクは依然として存在するため、抗凝固薬服用者は注意が必要です。

## 4. 適応症と使用例

### 主な適応症

| 症状 | 推奨度 | 使用方法 | 期待される効果 |
|---|---|---|---|
| 筋肉疲労・軽度筋肉痛 | ◎ 強く推奨 | 入浴後、患部に薄く塗りマッサージ | 血行促進・筋肉緩和・軽度鎮痛 |
| 打撲（亜急性期） | ○ 推奨 | 受傷後24–48時間以降に使用開始 | 局所血流改善・鎮痛 |
| 冷え性・末梢冷感 | ◎ 推奨 | 足首・ふくらはぎ・手首に塗布してマッサージ | 局所温感・血行促進 |
| 肩こり・首こり | ○ 推奨 | 肩・頸部に塗布し軽くマッサージ | 筋肉緊張緩和・温感 |
| 腰の張り（筋性腰痛） | ○ 推奨 | 腰部筋肉に沿って塗布 | 筋肉の血行促進・軽度鎮痛 |
| 虫刺され（軽度） | △ 補助的 | 患部に少量点状塗布 | 清涼感・かゆみ軽減 |
| 頭痛・鼻詰まり | △ 不向き | — | 設計対象外（White系を選ぶべき） |

### 使用シーン別の実際の活用例

**冷え性ケア（冬季）**: 就寝前に足先からふくらはぎにかけて双獅標を薄く塗布し、ゆっくりとマッサージします。メントールの清涼感が一時的にあった後、樟脳の温感が続き、末梢の血行が促進されます。靴下を履いて就寝すると効果が持続しやすくなります。

**軽い打撲後のケア**: 転倒などによる軽い打撲で、皮膚に傷がなく腫脹が落ち着いた段階（受傷後48時間以降）から使用を開始します。患部に少量塗布し、過度な圧をかけずに優しくなじませます。

**デスクワーク後の肩こり**: 長時間のパソコン作業後、肩から首筋にかけて塗布します。清涼感でリフレッシュしながら、血行促進による筋肉緩和が期待できます。

## 5. 安全注意事項（メチルサリチル酸含有量によるワルファリン評価）

### ワルファリン（抗凝固薬）服用者への注意

双獅標活絡油のメチルサリチル酸含有量は約20–30%です。黄道益（50%）よりは低いものの、ワルファリンとの相互作用リスクは依然として存在します。

| リスク評価項目 | 内容 |
|---|---|
| メチルサリチル酸濃度 | 約20–30%（中程度） |
| ワルファリンリスクレベル | **中程度（使用禁忌ではないが要注意）** |
| 経皮吸収後の代謝物 | サリチル酸（Salicylic acid） |
| 相互作用機序 | サリチル酸がCYP2C9を阻害 → ワルファリン代謝遅延 → 血中濃度上昇 → INR上昇 |
| 特に危険な使用法 | 広範囲・長時間・閉塞処置下での塗布 |
| 推奨事項 | 抗凝固薬服用中の方は事前に医師・薬剤師へ相談 |

**ワルファリン服用者には Tiger Balm White（メチルサリチル酸不含）が最も安全な代替選択肢です。**

### その他の一般的な安全注意事項

| 注意事項 | 詳細 |
|---|---|
| 目・粘膜への接触禁止 | メントール・樟脳成分が粘膜を強く刺激 |
| 傷口・損傷皮膚への使用禁止 | 皮膚バリア欠損部では吸収量が激増し、刺激・毒性リスク上昇 |
| 温熱処置との同時使用禁止 | 入浴直後・カイロ使用直後は経皮吸収量が増加するため危険 |
| 使用後の手洗い | 必ず手を洗い、目や口に触れないよう注意 |
| アレルギー確認 | 初回使用時は腕の内側に少量塗布し、パッチテストを推奨 |

## 6. 子ども・妊婦への注意

### 小児への使用

| 年齢 | 使用可否 | 理由・詳細 |
|---|---|---|
| 2歳未満 | **絶対禁止** | 樟脳成分が乳幼児の中枢神経に毒性を示す可能性（樟脳中毒リスク） |
| 2〜6歳 | **使用禁止を強く推奨** | メチルサリチル酸の全身吸収リスク、皮膚面積比で吸収量が大人より相対的に多い |
| 6〜12歳 | **要注意・医師相談推奨** | 必要最小限の量・面積での使用、保護者の管理下に限定 |
| 12歳以上 | 成人に準じた使用可 | 少量から始め、皮膚反応を確認 |

**特に重要**: メントール・樟脳成分の乳幼児への使用は、国際的な小児科学会でも危険性が指摘されています。特に鼻・口周辺への塗布は乳幼児の呼吸抑制を引き起こす可能性があり、絶対に行わないでください。

### 妊婦・授乳中の方への注意

| 項目 | 内容 |
|---|---|
| メチルサリチル酸（妊婦） | 妊娠後期における大量経皮吸収は胎児への影響（動脈管早期閉鎖リスク）が懸念される |
| 樟脳（妊婦） | 胎盤通過性があり、過剰暴露は胎児毒性の報告あり |
| 妊婦への推奨 | **使用前に必ず産婦人科医に相談** — 特に妊娠中期以降は使用を避けることが賢明 |
| 授乳中 | 皮膚からの少量成分移行リスクあり、乳幼児への間接暴露を避けるため乳房周囲への使用禁止 |

双獅標活絡油は、香港の家庭薬文化の中で「特別な疾患治療薬」ではなく「日常の体のケアに気軽に使えるパートナー」として長く親しまれてきた製品です。強烈な鎮痛効果を求めるなら黄道益、多目的で使いやすさ重視ならTiger Balm Whiteや斧標、日常の軽いケアや冷え性対策には双獅標と使い分けることが、薬用オイルを賢く活用する一つの方法です。
