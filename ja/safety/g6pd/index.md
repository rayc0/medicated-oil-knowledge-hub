# G6PD欠損症と薬用オイル：安全な使い方ガイド

> **重要な免責事項** — 本記事は教育目的の情報提供であり、医療アドバイスの代替ではありません。G6PD欠損症の診断を受けた方、または疑いがある方は、薬用オイルの使用前に必ず医師または薬剤師に相談してください。

---

## 1. G6PD欠損症とは何か

**G6PD（グルコース6リン酸脱水素酵素）欠損症**は、赤血球の保護に関わる酵素「G6PD」が遺伝的に不足・機能低下している状態です。この酵素は赤血球を酸化ストレスから守る役割を担っており、欠損すると特定の物質・薬剤にさらされたときに**溶血性貧血（赤血球が破壊される状態）**を引き起こす可能性があります。

### 世界的な有病率

G6PD欠損症はX染色体連鎖の遺伝性疾患で、**世界で約4億人が罹患している**とされます。地域別の有病率は以下の通りです。

| 地域・民族 | 推定有病率 |
|-----------|-----------|
| 東アジア（中国・香港・台湾） | 男性の約3〜8% |
| 韓国 | 男性の約0.4〜2% |
| 日本 | 男性の約0.1〜0.5% |
| 東南アジア（マレーシア・シンガポール） | 男性の約4〜15% |
| 地中海沿岸 | 男性の約5〜20% |
| アフリカ系 | 男性の約10〜15% |

**日本での有病率は比較的低い**ですが、ゼロではありません。また、香港産・東南アジア産の薬用オイルを使用する際には、製品が想定する使用者集団（中国系・東南アジア系）では有病率が高いことを念頭に置く必要があります。

### 症状と発症メカニズム

通常時は無症状ですが、酸化ストレスを引き起こす物質にさらされると急性溶血発作が起こります。

**発症時の典型的な症状：**
- 突然の全身倦怠感・疲労感
- 尿の色が濃くなる（茶色・コーラ色の尿）
- 黄疸（皮膚・白目が黄色くなる）
- 急速な心拍数増加・息切れ
- 重篤な場合：腎不全・急性貧血

## 2. 薬用オイルの成分で特に注意すべきもの

### カンファー（Camphor / 樟脳）

**リスクレベル：高**

カンファーはG6PD欠損症患者において溶血を誘発する危険性が報告されている成分です。主に**白花油・Ho Yeow油・AxeBrand Universal Oil・タイガーバーム**などの薬用オイルに含まれています。

- 白花油（和興）：約8%含有
- AxeBrand Universal Oil：約11%含有
- 関平（Kwan Loong）油：低濃度含有（製品による）

**作用機序：** カンファーは赤血球の酸化的ストレスを増加させ、G6PD欠損患者では解毒できない活性酸素種が蓄積し、溶血を誘発します。

### 高濃度メントール（Menthol）

**リスクレベル：中〜高（用量依存性）**

メントール自体はG6PD欠損への影響が比較的小さいとされますが、**高濃度（20%以上）では酸化作用が懸念**されます。関平油のような高濃度メントール製品（57%以上）を広範囲に塗布する場合は注意が必要です。

### ウィンターグリーン油（サリチル酸メチル）

**リスクレベル：低〜中**

ウィンターグリーン油のサリチル酸メチルはG6PD欠損との直接的な関係は限定的ですが、経皮吸収量が多い場合（大面積・長時間）には一般的な注意が必要です。

### ユーカリ油

**リスクレベル：低**

通常の使用量では溶血リスクは低いとされています。ただし医薬品的な大量使用は避けてください。

## 3. G6PD欠損症の方が薬用オイルを使う際のガイドライン

### 使用前の確認事項

1. **診断の確認** — G6PD欠損症の診断を受けているか、または家族歴がある場合は使用前に医師に相談する
2. **成分表示の確認** — カンファー含有製品は可能な限り避ける
3. **少量パッチテスト** — 初めて使用する場合は少量を小さな面積に塗布し、30分様子を見る

### 比較的安全とされる代替品

G6PD欠損症の方に向けて、カンファーを含まない、またはリスクの低い代替品を示します。

| 製品 | カンファー含有 | G6PD相対リスク |
|------|-------------|--------------|
| タイガーバームWhite（白缶） | 含有（11%） | 注意必要 |
| タイガーバームRed（赤缶） | 含有（11%） | 注意必要 |
| 関平油（Kwan Loong） | 含有（低量） | 低リスクだが注意 |
| メントール単剤外用薬 | 非含有 | 低リスク |
| ユーカリ油単剤 | 非含有 | 低リスク |
| 冷湿布（サロンパス等） | 非含有 | 低リスク |

**最も安全な選択肢は、カンファーを含まない製品を選ぶことです。**

### 使用量・方法の注意

- **使用面積を小さく保つ** — 経皮吸収量を最小化する
- **長時間の密閉塗布は避ける** — 吸収量が増加する
- **換気の良い場所で使用する** — 揮発成分の吸入を減らす
- **乳幼児への使用は禁止** — G6PD欠損の有無に関わらず

## 4. G6PD欠損症の診断について

G6PD欠損症は血液検査で診断できます。東アジア系の方で以下に当てはまる場合は検査を検討してください。

- **蚕豆病（蚕豆を食べた後に貧血症状が出る）** の家族歴がある
- 原因不明の溶血性貧血のエピソードがある
- 特定の薬剤（プリマキン等）服用後に異常が起きた経験がある

日本では健康診断の標準項目には含まれていませんが、かかりつけ医に相談すれば保険適用で検査可能です。

---

## 5. 緊急時の対処法

薬用オイル使用後に以下の症状が出た場合は、**直ちに使用を中止し、医療機関を受診**してください。

- 尿の色が急に濃くなった（茶色・赤褐色）
- 急激な疲労感・息切れ
- 皮膚や白目が黄色くなった
- 腹痛・腰痛を伴う場合

医師には使用した製品名と成分（カンファー・メントール含有）を伝えてください。

---

## まとめ

G6PD欠損症は東アジア系の集団では無視できない有病率（男性の3〜8%）を持ちます。薬用オイルに含まれるカンファーや高濃度メントールは溶血リスクを高める可能性があります。診断がある方、または不確かな方は、使用前に必ず医師・薬剤師に相談し、成分表示を確認する習慣をつけることが重要です。
